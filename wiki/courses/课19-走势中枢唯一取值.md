# 课19：二阶段讲四 — 本质走势中枢之唯一取值

## 一句话摘要

本质走势中枢的唯一取值方法：以**反向线段的转折K为时间起点**，向左找到**有价差的34均线最值**作为波峰/波谷，再向左右找对应最值，**取最窄重叠区域**得到上下轨（下轨=取两个波谷中较高者，上轨=取两个波峰中较低者）。

| 核心目标 | 学习本质走势中枢的唯一取值方法——**精确量化中枢上轨和下轨** |
| 后续课程 | 课20：走势中枢唯一（中枢重组，一段与三段重叠处理） |

### 两类中枢的区分

| 类型 | 定义 | 应用场景 |
|------|------|---------|
| **本质走势中枢** | 日线级别线段内部的中枢 | 本课重点 |
| **转折中枢** | 两条日线级别线段之间的中枢 | 后续课程 |

> 一句话：我们学的是**日线级别线段的中枢**，服务于"看多区域"和"做多中枢"。

### 中小资金的核心逻辑

```
看多区域 = 日线级别线段向上的方向
做多中枢 = 在该区域内形成的本质走势中枢（买点在此产生）
```

## 二、作业讲评（纠正常见错误）

以"浙中股份"为案例，指出同学们的三大主要错误：

### 错误1：线段底取错

- **错误画法**：把底画到八块八三（更低点）
- **正确画法**：根据线段的五均线生长法则和转折K，底应在**九块零五附近**
- **错误原因**：主观使用非同级别分解，强行将不是线段的价格低点作为线段底，破坏了线段的唯一性
- **结论**：不会用非同就不要用；三期已把非同综合进线段唯一画法，只认本质线段

### 错误2：日线级别线段的识别问题

- 部分同学把**不是日线级别线段**的走势误判为日线级别线段（例如某段下跌未形成反向 → 无下则无上）
- 正确判断依赖**日线级别线段的四个必要条件**（课17内容）和连续性法则

### 错误3：被线段内部价格干扰

- 看日线级别线段时，**只看线段的顶和底**，不看线段内部的K线上涨下跌
- 案例中某向上线段的最低点是起始点，中间的回调不改变线段方向

> 作业的核心目的：让同学明白本质线段（30分钟级别）与日线级别线段的关联关系，纠正"乱用非同"的毛病。

## 三、本质走势中枢的唯一取值法则（核心）

### 3.1 基本定义回顾

| 走势类型 | 前三段构成 | 取值特征 |
|----------|-----------|---------|
| 下跌走势中枢 | 上-下-上（反向开始） | **最先确定下轨**（时间优先，反向线段先出现波谷） |
| 上涨走势中枢 | 下-上-下 | **最先确定上轨** |

> 本质走势中枢 = 由本质线段的前三段带来的**34均线重叠区域**。

### 3.2 取值的核心思路（以下跌走势中枢为例）

```
下轨 = 两个波谷值中较高者（取最高的低点）
上轨 = 两个波峰值中较低者（取最低的高点）
→ 得到最窄的重叠区域 = 本质走势中枢区间
```

下轨最先被确定，因为**时间优先**——反向线段（第二段上涨）先出现波谷。

### 3.3 波峰的精确量化：时间区域 + 空间区域

#### （1）时间区域：转折K

| 项目 | 说明 |
|------|------|
| 定义 | 五均线不再持续新低（或新高）所形成的转折K，**不能用特殊顶点法则的转折K** |
| 时间起点 | 以**第二条线段**（反向后的第一条线段）的转折K为时间起点 |
| 时间区域 | 从该转折K**向左看**（包含该K线本身），向右的不看 |

> 为什么用转折K？因为它代表下跌线段结束，34均线的滞后不会超出这个时间点。

#### （2）空间区域：34均线价差

找到转折K后，向左寻找34均线的最高值（波峰）。该波峰必须**同时满足**：
- **左边有34均线的价差**（比它低）
- **右边有34均线的价差**（比它低）

> 如果没有价差（如同价或连续平台），则该点不是波峰，说明**没有本质中枢**。

#### （3）由波峰确定波谷

确定波峰后：
- 向左找34均线的**最低值** → 波谷1
- 向右找34均线的**最低值** → 波谷2

这两个波谷值就是中枢下轨的候选。

### 3.4 中枢上下轨的最终取值

```
下轨 = max(波谷1, 波谷2)   ← 取两波谷中较高者
上轨 = min(波峰1, 波峰2)   ← 取两波峰中较低者
```

得到最窄的重叠区域，即本质走势中枢的区间。

### 3.5 上涨走势中枢的对称处理

- 反向为第一段（下跌），以第二段（上涨）的转折K为时间起点
- 向左寻找**波谷**（而非波峰）
- 波谷确定后，左右找波峰，取较高的低点为上轨
- 上涨中枢最先确定的是**上轨**

### 3.6 量化法则总结

```
转折K定时间 → 向左找最值（需价差） → 左右找对应最值 → 取最窄重叠
```

## 四、特殊情况的处理

### 4.1 一段重叠 vs 三段重叠

| 重叠类型 | 特征 | 优先级 |
|----------|------|--------|
| **三段重叠** | 三条线段全部进入重叠区域 | 标准本质中枢，核心 |
| **一段重叠** | 只有一条线段进入重叠区域，另两条未完全进入 | 仍是本质中枢（非肉眼中枢），优先级低于三段重叠 |
| **零段重叠** | 反向线段没有带来34均线价差 | **没有本质中枢**，该日线级别线段只是标准定义线段 |

### 4.2 微小价差（如0.01元）

- 只要符合法则（有价差），就算0.01元也是本质中枢
- 但在实战中，微小价差往往代表**资金犹豫不决**，该中枢可能**不会产生可靠的买点**

### 4.3 多个波峰/波谷的选择

时间区域内可能有多个34均线最值，选择**同时满足左右价差**且**离转折K最近**的那个。

## 五、作业与案例练习

### 课堂练习要点

- 给出转折K位置和34均线走势，让同学选择正确的波峰
- 正确答案：选同时满足左右价差且为最值的那个
- **核心原则**：万变不离其宗，复杂图形只要按**时间+空间法则**一步步分析，就能得到唯一答案

## 八、核心总结

> 本质走势中枢的取值 = 以反向线段的转折K为时间起点，向左找到有价差的34均线最值作为波峰/波谷，再左右找对应最值，取最窄重叠区域得到上下轨。

### 关键规则速查表

| 步骤 | 操作 | 要点 |
|------|------|------|
| ① 定时间 | 找到反向线段（第二段）的转折K | 不用特殊顶点法则的转折K，向左看 |
| ② 找最值 | 在时间区域内找34均线最值 | 必须左右都有价差，否则无中枢 |
| ③ 定波谷/波峰 | 从最值向左右找对应极值 | 下跌中枢：最值是波峰→左右找波谷；上涨中枢：最值是波谷→左右找波峰 |
| ④ 取重叠 | 下轨=max(波谷1,波谷2)，上轨=min(波峰1,波峰2) | 取最窄重叠区域 |
