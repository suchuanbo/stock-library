# 课18：二阶段讲三 — 本质走势中枢

## 一句话摘要

本质走势中枢 = **本质线段的前三段带来的34均线重叠**。34均线的唯一作用就是充当"中枢探测器"——通过波峰波谷找到重叠区域，从而确定中枢上下轨。三段重叠是核心（一段重叠让步），零段重叠不是中枢。掌握本质中枢即跨越从"工具"到"本质"的门槛。

---

## 一、课程定位与核心概念

| 维度 | 说明 |
|------|------|
| 所属阶段 | 第二阶段：走势中枢（课16~），本讲为中枢入门 |
| 前置知识 | 课17：日线级别线段 — 日线级别线段的两种类型 |
| 核心地位 | 本质走势中枢是"入门"的关键，掌握它意味着正式进入缠论实战门槛 |
| 核心隐喻 | **线段是手指，中枢是月亮** — 工具最终可丢，但必须通过工具找到本质 |

### 从工具到本质的跨越

```
第一阶段（课1~15）：本质线段 — 基础工具
第二阶段（课16~）：本质走势中枢 — 核心本质，服务于走势分类和买卖点
本课目标：将本质线段与本质走势中枢综合，实现跨越
```

## 二、日线级别线段的两种类型（复习）

| 类型 | 说明 | 操作价值 |
|------|------|----------|
| 最小日线级别线段 | 类似30分钟级别线段中的最小单位 | 基础构件 |
| 具备中枢的日线级别线段 | 真正有操作价值的走势载体 | ⭐ 核心 |

> 关键更新（三期）：日线级别线段不一定是多条线段组合，也可以是**单条升级线段**。取消了二期中的"最值"要求，直接以线段顶底为日线级别线段的顶底。

---

## 三、本质走势中枢的定义

### 3.1 传统走势中枢（缠论原文定义）

| 要素 | 说明 |
|------|------|
| 定义 | 由至少三个**连续次级别走势类型**的重叠部分构成 |
| 上涨中枢 | 由"下-上-下"三段线段的重叠区域得到 |
| 作用 | 给出中枢的上下轨，但**不是本质中枢** |

### 3.2 本质走势中枢（本课程核心）⭐

> **定义：由本质线段的前三段及以上带来的34均线重叠构成。**

| # | 必须同时满足的条件 | 
|---|-------------------|
| 1 | 存在三条本质线段 |
| 2 | 这三条线段对应带来**34均线的重叠** |

```
本质中枢 = 本质线段 + 34均线重叠
           ⚠️ 缺一不可
```

| 规则 | 说明 |
|------|------|
| 只看前三段 | 三段以后的线段**不参与中枢构造** |
| 最少要求 | 至少三条本质线段，不够则无中枢 |
| 关键条件 | 34均线必须形成重叠区域（波峰与波谷重叠） |

---

## 四、34均线的唯一作用

> ⚠️ **34均线既不服务于涨跌，也不服务于生长法则。**

| 维度 | 说明 |
|------|------|
| 唯一作用 | 确定本质走势中枢的**重叠区域** |
| 运作方式 | 通过34均线的**波峰、波谷**找到重叠区间 |
| 输出结果 | 得到中枢的**上下轨** |
| 重要提醒 | 34均线只是"中枢探测器"，**没有方向含义** |

---

## 五、34均线重叠的三种情况

| 类型 | 定义 | 处理规则 |
|------|------|----------|
| **三段重叠** | 三段本质线段均带来34均线重叠 | ⭐ **核心**：以此为准 |
| **一段重叠** | 仅一段本质线段带来34均线重叠 | 本质上是中枢（类似"非肉"），但**让步于三段重叠** |
| **零段重叠** | 走势中无34均线重叠 | **不是中枢**，只能作为一条线段处理 |

> **让步规则**：当同时存在一段重叠和三段重叠时，以三段重叠为核心，一段重叠不做中枢。

## 六、本质线段与日线级别线段的综合关系

本课将前置知识完全打通：

| 知识点 | 结论 |
|--------|------|
| 日线级别线段构成 | 不一定是多条线段组合，单条升级线段亦可 |
| 二期"最值"要求 | 已取消，直接以线段顶底为日线级别线段顶底 |
| 本质线段的作用 | 是构造本质中枢的基础材料 |
| 综合关系 | 本质线段 → 34均线重叠 → 本质中枢 → 服务于走势分类和买卖点 |

|------|----------|
| 中枢上轨 | 重叠区域中**波峰的低点**（取低） |
| 中枢下轨 | 重叠区域中**波谷的高点**（取高） |

## 八、实战应用要点

### 8.1 看多 vs 做多

| 概念 | 定义 | 关系 |
|------|------|------|
| **看多** | 突破转折中枢上轨后，进入"可做多区域" | 前提条件 |
| **做多** | 在看多区域内出现买点才可操作 | 实际操作 |

> ⚠️ **无看多区域，任何买点都是假的**（即使个股涨停也可能随后下跌）。

### 8.2 错误案例：彩虹股份的"三买"陷阱

| 错误判断 | 实际真相 |
|----------|----------|
| 某位置是"三买" | 该位置没有向上的日线级别线段（无下则无上） |
| 可以买入 | 走势仍属下跌状态，不具备看多区域 |
| — | 不是买点，买入会被套 |

### 8.3 仓位与风控（重申）

```
20% 尝试建仓 → 浮盈后加仓（40%） → 不动本金（40%）永远安全
```

| 风控规则 | 说明 |
|----------|------|
| 大盘下行风险 | 即使个股无卖点，也要**减仓**（不是清仓） |
| 本金安全 | 40%本金始终不动，只用浮盈加仓 |

## 十一、本课核心总结

> **本质走势中枢 = 本质线段的前三段带来的34均线重叠。**
>
> 34均线唯一作用就是找中枢；三段重叠是核心，一段重叠是让步，零段不是中枢。

| 口诀 | 内容 |
|------|------|
| 中枢公式 | 本质线段 + 34均线重叠 |
| 34均线 | 中枢探测器，无方向含义 |
| 重叠优先级 | 三段 > 一段 > 零段（非中枢） |
| 看多做多 | 先有看多区域（突破中枢上轨），再有买点才操作 |
