# 课36：二阶段（下）讲二 — 职业交易之进攻买二（重点）

## 一句话摘要

**进攻买二（进二）是从下跌转向上涨的初始起涨点，是一种抄底买点，适用于日线级别趋势的起始建仓和选股。** 其核心结构三要素：独立转折（看多前提）→ 做多中枢（安全边际）→ 涨停驱动（核心驱动力量）。进二不是唯一的抄底买点，也不是纯均线形态——均线只是描绘工具，本质是日线级别起涨的结构。

---

- **面向三期学员**：一期、二期学员常把买点问题简化为均线系统，忽略本质结构
- **教学理念**：「手指明月指」——均线只是工具，结构才是本质
- **从结构打磨到买卖点应用**：本课是二阶段从纯结构理论过渡到实战买点的关键一课

## 二、进攻买二的核心定义

### 2.1 进二是什么

| 维度 | 说明 |
|------|------|
| 本质 | 日线级别上涨的起涨点（从下跌转向上涨的初始建仓位置） |
| 类型 | **抄底买点**——在日线级别趋势起始建仓和选股 |
| 安全边际 | 与170均线有紧密关系，安全边际在170均线附近的某中枢上轨 |
| 适用场景 | 初始建仓（中小资金首仓）、选股买点 |

### 2.2 重要澄清

> **进二不是唯一的抄底买点。** 防二、进一、进三等也可以抄底，但结构不同。

| 买点 | 区别 |
|------|------|
| 进二 | 最典型的「底部启动买点」，日线级别起涨 |
| 防二 | 防守型抄底，安全边际要求更严 |
| 进一/进三 | 不同阶段的进攻买点，结构前提不同 |

---

## 三、进二买点的结构三要素

```
看多（独立转折成立）
  → 等待回撤
    → 在做多区域内确认线段终结
      → 配合涨停/大阳驱动
        → 形成进二买点
```

**核心逻辑**：看多 → 等待回撤 → 在安全边际内确认线段终结 → 配合核心驱动 → 买点触发。

---

## 四、进二标准结构的精确量化条件

触发进二买点时，**必须同时满足以下三条**（均线作为工具）：

| 序号 | 条件 | 含义 |
|------|------|------|
| ① | **五均线值 > 中枢上轨** | 股价已突破做多中枢 |
| ② | **五均线值 > 34均线** | 已进入进攻区域（非防守区） |
| ③ | **五均线值 > 170均线** | 日线级别起涨确认 |

> **注意**：170均线与中枢之间没有必然的「位置关系」，只要买点形成时170均线在中枢上方即可。不需要纠结34与170的关系、170与中枢的其他复杂关系。

---

## 五、独立转折与走势转折的区别（前提判断）

这是进二买点**最关键的前提条件**——只有独立转折才可能产生进二。

| 类型 | 定义 | 是否产生进二 |
|------|------|:--:|
| **独立转折** | 由独立的日线级别线段形成的转折（如三买独立、三卖独立终结前一段走势） | ✅ 可以 |
| **走势转折** | 由三段线段构成的中枢形成的转折 | ❌ 不可以 |

> **走势转折的个股只能产生防守、进一、进三等买点，不可能有进二。**

**判断示例**：
- 尚美股份：独立转折 → 有进二 ✅
- 海王生物：虽然独立，但170均线在中枢下方 → 无进二 ❌

---

## 六、做多中枢（安全边际）的确定

### 6.1 做多中枢可以是

| 类型 | 说明 |
|------|------|
| 独立转折的中枢 | 最常见的情况 |
| 走势分类后得到的中枢 | 如向上分类中枢 |
| 其他有效的中枢上轨 | — |

### 6.2 安全边际的核心

> **进二的安全边际一定是170均线附近的某中枢上轨。** 止损位常设为34均线或中枢下轨。

### 6.3 常见错误

> 认为「做多中枢 = 独立转折中枢」。实际上，做多中枢可以不同，只要埋点形成时股价与170有安全边际即可。

---

## 七、核心驱动力量（涨停/大阳）的位置

### 7.1 涨停的位置要求

> 涨停**必须**出现在：**独立转折后的170附近，且在买点形成之前。**

### 7.2 涨停的意义

| 维度 | 说明 |
|------|------|
| 吸引市场关注 | 展示资金力度 |
| 标志扭转 | 由跌转涨的关键信号 |
| 优选条件 | 同期板块内多只个股有结构、有共同驱动逻辑的涨停 |

### 7.3 应放弃的涨停

- 无结构的涨停（老鼠仓）
- 无板块一致性的涨停（突发消息）
- 涨停位置不在170附近的

---

## 八、进二的买入信号与方式

> **课程建议：进二以潜伏买入为主。** 因为其结构确定性强，回撤可控。

| 方式 | 特点 |
|------|------|
| 潜伏买入 | 结构确定，回撤可控，适合建底仓 |
| 追涨买入 | 不推荐（容易追高，盈亏比变差） |

---

## 九、进二的功能与择时匹配

### 9.1 功能

| 功能 | 说明 |
|------|------|
| 初始建仓 | 日线级别趋势的底仓买点 |
| 选股买点 | 筛选符合条件的个股 |
| 「彩票型」买点 | 可能出短期暴涨个股（如中青宝、朗姿股份等案例） |

### 9.2 择时匹配

| 条件 | 操作 |
|------|------|
| 30分钟上涨末端（大盘恐慌时） | **优选进二**，容易跑赢市场 |
| 前提 | 日线级别趋势未被破坏 |

### 9.3 与其他买点的关系

- 防二、进一、进三也可以抄底，但结构不同
- 进二是最典型的「底部启动买点」，适合中小资金首仓
- 进二建底仓 → 后续通过进三加仓 → 滚动操作

## 十、常见错误与纠偏

| 序号 | 错误 | 纠正 |
|:--:|------|------|
| 1 | 只记均线形态（如「170上方+中枢上轨」就认为是进二） | **必须先确认独立转折前提** |
| 2 | 做多中枢锁定错误（强行等同于独立转折中枢） | 做多中枢可以不同，关键是与170有安全边际 |
| 3 | 忽略核心驱动（没有涨停或大阳线） | 即使结构符合，上涨力度也有限，缺乏资金推动 |
| 4 | 混淆走势转折与独立转折 | 走势转折的个股**不可能有进二** |
| 5 | 止损缺失 | 进二虽然成功率较高，但仍需带好安全边际（34均线或中枢下轨） |

---

## 十一、交易心态与资金管理要点

| 要点 | 说明 |
|------|------|
| 进二不是唯一 | 不要迷恋进二，其他买点同样能赚钱 |
| 选股靠结构+驱动 | 每天复盘涨停板，筛选有板块一致性、有结构、有独立转折的个股 |
| 交易模式三要素 | 准确率（结构）× 盈亏比（安全边际）× 仓位（资金管理） |
| 趋势性操作 | 进二建底仓 → 进三加仓 → 滚动操作，不做无谓的短线折腾 |
| 主动卖点 | 当个股短期涨幅超过50%时，可全仓卖出，不必滚动 |
