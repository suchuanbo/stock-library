# 课10：线段顶底职业交易四法则

## 一句话摘要

线段顶和底各为一根K线，但必须同时满足四个量化法则——3k组合、独立两k、34均线两侧、结构完美与转折k最近——缺一不可。

---

## 一、课程定位

| 维度 | 说明 |
|------|------|
| 所属阶段 | 第二阶段：买点体系（课10~45） |
| 前置知识 | 课1~9：线段中枢全部内容（中枢线、起手三式、黑K、唯一选取） |
| 在体系中的位置 | 线段 = 线段中枢 + 顶 + 底。本课讲顶底的**静态四法则**，后续课讲顶底唯一选取 |
| 核心作用 | 顶底是新生线段转折的起点，是买卖点判断的基础 |

---

## 二、顶底基础概念

### 2.1 顶和底都是一根K线

> 一条线段的顶和底，各自**只指一根K线**，不是三根，不是区域。

但该K线不能是随意的高点或低点——必须同时满足四个法则。

### 2.2 只看单边价格

| | 看的价 | 不看的 |
|------|---------|--------|
| **顶** | 只看K线的**最高价** | 不看实体、不看最低价、不看开盘/收盘 |
| **底** | 只看K线的**最低价** | 不看实体、不看最高价、不看开盘/收盘 |

---

## 三、顶底四法则（静态）

### 法则一：3k组合

**一句话**：顶或底必须是三根连续K线中，价格最极端的中间那根。

**量化规则**：

| 顶 | 底 |
|----|----|
| 中间K线最高价 ≥ 左边K线最高价，且 ≥ 右边K线最高价 | 中间K线最低价 ≤ 左边K线最低价，且 ≤ 右边K线最低价 |
| **同价也算满足** | **同价也算满足** |

**意义**：最小级别的转折确认——至少在三K范围内形成了局部高/低点。

> ⚠️ 仅满足法则一远远不够，必须四个法则全部通过。

---

### 法则二：独立两k条件（至少其一）

**一句话**：顶和底中，至少有一个存在连续两根K线**完全在线段中枢区域之外**。

**量化规则**：

- 顶要满足独立两k → 顶K线及相邻某根K线（3k组合中的左二或右二）的**全部价格**都在中枢区域**上方**（同价也算外）
- 底要满足独立两k → 相关K线的**全部价格**都在中枢区域**下方**（同价也算外）

**逻辑约束**：

```
顶底都满足独立两k？ → ✅ 有效
只有顶满足？       → ✅ 有效
只有底满足？       → ✅ 有效
两者都不满足？     → ❌ 该顶底无效
```

**前置条件**：顶或底首先要本身就在中枢区域外（顶在上方、底在下方），否则不可能满足独立两k。

---

### 法则三：34均线两侧（时间 + 空间）

法则三包含两个独立维度，必须同时满足。

#### 3.1 时间条件：初始交叉临界点

线段发生转折（上涨↔下跌），必须出现**五均线与34均线的交叉**：

| 转折方向 | 交叉类型 | 判断 |
|----------|----------|------|
| 上涨 → 下跌 | 五均线向下**死叉**34均线 | 五均线值 < 34均线值 |
| 下跌 → 上涨 | 五均线向上**金叉**34均线 | 五均线值 > 34均线值 |

- 交叉点所在K线称为**临界点**
- **顶或底必须出现在临界点之后**；临界点本身**不可**作为顶/底
- 临界点之前出现的任何高/低点都不能作为线段的顶或底

**特殊情形 — 粘合同价**：

> 若五均线与34均线刚好同价（粘合），则该点**不作为临界点**，也不可作为顶/底。

**多次缠绕**：若均线反复交叉，**取第一次交叉**为有效临界点。

#### 3.2 空间条件：实体在均线两侧

| | 量化条件 | 注意 |
|------|----------|------|
| **顶** | 顶K线的**实体上沿** > 34均线值 | 同价**不算**上 |
| **底** | 底K线的**实体下沿** < 34均线值 | 同价**不算**下 |

> ⚠️ 这里是看**实体**，不是看价格（最高/最低价）。

#### 3.3 波峰波谷对应关系（简要）

- 顶的实体上沿在34均线上方 → 对应**波峰**
- 底的实体下沿在34均线下方 → 对应**波谷**

（波峰波谷与34均线两侧的衔接，需通过作业练习巩固。）

---

### 法则四：结构完美与转折k最近

#### 4.1 结构完美的两个临界条件

**条件一：顶/底是中枢的最后一根K线（结束k）**

- 顶或底必须是所在线段中枢区域的**最后一根K线**
- 允许该K线同时是起手三式的构造K（如5k重叠的第5根）
- **但不能是黑K**（黑K是线段级别的保障K线，不可作为顶或底）

**条件二：顶和底分居中枢区域两侧**

```
顶在中枢区域上方 + 底在中枢区域下方 → ✅ 结构完美
顶/底与中枢区域同侧               → ❌ 临界状态，不构成线段
→ 只有价格移动使顶明确在上、底明确在下时，才满足
```

> 换句话说：顶和底**不能在中枢的同一侧**。

#### 4.2 转折k最近

**规则**：一条线段的顶（或底），必须是**距离转折k最近**的那个满足前三个法则的K线。

| 关系 | 说明 |
|------|------|
| 顶/底 ≠ 转折k本身 | 顶和底必须在转折k**之前**（属于前一线段的结束点） |
| 转折k是新生线段的起点 | 相当于k0 |

---

## 四、转折k的量化定义

转折k是**线段结束与新线段开始**的标志——没有转折k就没有顶底。

### 4.1 普通转折k

同时满足两个条件：

1. **五均线不再持续新高/新低**：出现反向（同价也算反向）
   - 上涨转下跌：五均线值不再持续新高
   - 下跌转上涨：五均线值不再持续新低
2. **实体反突破五均线**：该K线的实体反方向突破五均线（与中枢线确定k的法则一致）

满足以上两条的K线即为转折k。

### 4.2 特殊转折k（双跳空）

若K线满足**双跳空**条件——即：

- K线价格对前一根K线跳空
- **且**实体对五均线也跳空

则即使五均线未出现反向，该K线也可作为**特殊转折k**。

> 与中枢线确定k的特殊情况规则相同。详见 [[concepts/中枢线与k0-k体系]]。

### 4.3 转折k与顶底的关系总览

```
转折k 出现之前 → 存在顶/底（前一线段的结束点）
转折k 出现之时 → 新线段开始
转折k 未出现   → 不存在顶/底
```

---

## 五、重要补充与禁忌

| 项目 | 说明 |
|------|------|
| **黑K不能做顶/底** | 黑K是线段级别的保障K线，绝不可作为顶或底的K线 |
| **"笔"的体系定义** | 本体系中的"笔" = 比线段级别更小的一切走势（一根K线也是笔），**不是**传统缠论意义上的顶分+底分+5K |
| **起手三式的"三笔"** | 仅用于服务线段中枢的形成和小级别买卖点，无其他作用 |

## 六、四条法则速查表

| 法则 | 核心问题 | 一句话口诀 | 量化要点 |
|------|----------|------------|----------|
| 法则一：3k组合 | 是否形成了局部转折？ | 中间K线是极端 | 三K连续，中间最高（顶）/最低（底）为最值，同价算满足 |
| 法则二：独立两k | 是否脱离了中枢？ | 至少一边有两K在外 | 顶/底至少一个，存在连续两K完全在中枢区域之外 |
| 法则三：34均线两侧 | 是否在均线层面确认转折？ | 交叉之后，实体跨线 | 时间：交叉之后；空间：实体上沿>34（顶）/ 实体下沿<34（底） |
| 法则四：结构完美+转折k最近 | 是否结构完整且位置正确？ | 结束k、分两侧、最近转折k | 是中枢最后一K；顶底分居中枢两侧；距转折k最近 |
