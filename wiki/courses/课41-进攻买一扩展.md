# 课41：二阶段（下）讲七 — 职业买卖之进攻买一（扩展）

## 一句话摘要

**进攻买一扩展是进一买点的补充类型，与标准进一的核心区别在于回撤跌破了34均线。** 当34均线不再能作为安全边际时，改以下跌线段最后一个中枢的上轨作为安全边际。本课同时提前引入三阶段核心内容：买点本质认知、资金管理、仓位配置与交易模式。

---

## 一、课程定位与核心目标

| 维度 | 说明 |
|------|------|
| 所属阶段 | 第二阶段（下）第七课，进攻买一的扩展类型 |
| 前置知识 | 课40：进攻买一（标准结构）；课38-39：线段终结（小级别回撤判断）；课35：买点本质（三要素） |
| 后续课程 | 课42：防守买一；课43-44：进三（滚动买点） |
| 核心目标 | 理解进一扩展与进一标准的区别（回撤是否跌破34均线）；建立正确的买点认知（买点是"活的"）；提前了解三阶段资金管理与仓位配置 |

### 课程背景

- **承上启下**：在标准进一基础上引入扩展类型，同时为三阶段（资金管理、交易模式、赚钱思维）做铺垫
- **核心观念转变**：引导学员从「寻找必胜买点」转变为「利用工具建底仓、通过资金运作降低成本」
- **适用人群**：职业交易者需掌握；非职业者可了解，重点是建立正确的买点认知

## 二、进一买点的本质（复习）

### 2.1 进一的本质

| 维度 | 说明 |
|------|------|
| 本质定义 | 一段30分钟级别线段下跌结束，带来新的30分钟级别上涨。买点就是这个上涨的起始或加速点 |
| 前提条件 | 必须存在**日线级别的看多结构**（独立转折看多 或 走势转折看多）。没有看多，就没有进一 |
| 级别定位 | 30分钟级别起涨/加速点 |

### 2.2 进一 vs 进二

| 买点 | 级别 | 看多前提 | 定位 |
|------|------|----------|------|
| 进二 | 日线级别起涨点（相对最低位） | 仅限独立转折看多 | 抄底建仓 |
| 进一 | 30分钟级别起涨/加速点 | 独立转折 或 走势转折看多均可 | 趋势中继/加速 |

> **关键**：进一的适用范围比进二更广（走势转折看多也可产生进一），但级别比进二小一级。

---

## 三、进一标准与扩展的核心区别

### 3.1 区别一览

| 维度 | 标准进一 | 扩展进一 |
|------|----------|----------|
| 回撤幅度 | 未跌破34均线 | **跌破了34均线** |
| 安全边际 | 34均线与中枢上轨的关联（相切或上方） | **下跌线段最后一个30分钟级别中枢的上轨** |
| 买入方式 | 潜伏买入 / 突破买入均可 | 潜伏买入更常见（回撤较深） |
| 本质 | 34均线仍可作安全边际 | 34均线已破，改用中枢上轨作安全边际 |

### 3.2 扩展的本质

> **当回撤跌破了34均线，无法用34均线作为安全边际时，改用下跌线段最后一个中枢的上轨作为安全边际。**

这个切换逻辑是进一扩展与标准进一的唯一本质区别。

---

## 四、进一扩展的结构三要素

```
看多结构（必须）
  → 做多中枢（安全边际 = 最后一个中枢上轨）
    → 核心驱动力量（涨停/大阳）
      → 进一扩展买点成立
```

### 4.1 看多结构（必须）

- **日线级别看多**：独立转折看多 或 走势转折看多
- **铁律**：没有看多，任何回撤都不构成买点

### 4.2 做多中枢（安全边际）

| 类型 | 安全边际 |
|------|----------|
| 标准进一 | 34均线与中枢上轨的关联（相切或在上方） |
| **扩展进一** | **下跌线段最后一个30分钟级别线段中枢的上轨**（因34均线已破） |

> **重要**：安全边际不是随便选一个中枢，必须与整体结构融合（如中枢托着均线，或均线托着中枢）。

### 4.3 核心驱动力量（大阳）

| 条件 | 说明 |
|------|------|
| 形态 | 涨停或7%以上大阳线，出现在构造买点的过程中 |
| 资金一致性 | 板块内多只个股同步、驱动因素一致 |
| 铁律 | **没有大阳，进一不成立** |

---

## 五、进一扩展的买点信号（小级别终结）

进一扩展的回撤属于30分钟级别线段的回撤，判断方法与课38-39小级别终结相同：

| 步骤 | 信号 |
|------|------|
| ① 回撤确认K线 | 最高价K线为回撤起点 |
| ② 方向延续 | 连续下跌创新低 |
| ③ 破五均线 | 价格跌破（非实体跌破） |
| ④ 终结确认 | 出现背驰 → 潜伏买入；或反突破 → 突破买入 |

> **注意**：扩展进一因跌破34均线、回撤较深，**潜伏买入比突破买入更常见**。

## 六、买点的本质（本节课重点强调）

### 6.1 买点是"活的"

| 认知 | 说明 |
|------|------|
| 无绝对性 | 市场是资金博弈的结果，买点不是"买了就一定涨" |
| 大资金影响 | 你的买入本身可能影响市场（大资金尤甚），中小资金影响小但仍有不确定性 |
| 相对较优 | 买点只是相对较优的介入位置，不是"必胜点" |

### 6.2 买点的真正目的

> **不是为了"一买就涨"，而是为了建立底仓、降低成本、滚动操作。**

真正的盈利来自于：
- **选股**（大概率上涨）
- **赔率**（盈亏比）
- **仓位管理**（资金分配）
- **资金运作**（滚动降成本）

学习买点是为了获得工具落脚点，最终服务于**资金成长**。

### 6.3 职业交易的正确观念

| 原则 | 说明 |
|------|------|
| 不追求绝对买卖点 | 不存在，不要试图寻找 |
| 不因下跌否定买点 | 市场是活的，可能因临时因素波动 |
| 结构不破不出局 | "狗皮膏药式"持仓，让主力洗不掉你 |
| 止损后重新来 | 止损是为了风控，不是否定选股；若结构仍在，出现新买点可重新介入 |

---

## 七、资金管理与仓位配置（三阶段提前讲解）

### 7.1 个股仓位管理（以中小资金为例）

| 规则 | 说明 |
|------|------|
| 单只个股上限 | 不超过总资金的 **20%**（分五只股票） |
| 建仓分配 | 4% + 8% + 8%（或类似比例），留机动资金做滚动 |
| 核心目的 | 将个股成本降至零，然后**增加股票数量**，而非不断加仓抬高成本 |

### 7.2 两种错误的加仓方式

| 错误方式 | 问题 |
|----------|------|
| 越涨越加 | 成本不断抬高，一旦回调利润回吐巨大 |
| 越跌越加 | 摊平成本，但若结构破坏会巨亏 |

**正确方式**：固定初始仓位 → 通过滚动操作降低成本 → 盈利后**增加股票数量**，而非增加单一仓位比例。

### 7.3 整体资金配置

| 原则 | 说明 |
|------|------|
| 大盘择时 | 根据大盘调整总仓位（如30分钟盘整：职业者≤6成，非职业≤3成） |
| 行业分散 | 不同行业配置，东方不亮西方亮 |
| 小资金目标 | 快速成长，年化收益率应远高于传统理财 |

## 八、买点应用中的常见误区

| 误区 | 纠正 |
|------|------|
| 追求"一买就涨"的必胜点 | 买点是建仓工具，不是必胜公式 |
| 买入后下跌就否定买点 | 市场是活的，结构不破不出局 |
| 忽略安全边际盲目买入 | 扩展进一的安全边际在中枢上轨，不是34均线 |
| 忽略核心驱动（没有大阳） | 没有大阳的买点不成立 |
| 不看大盘择时满仓干 | 根据大盘状态调整仓位 |

## 九、当前行情与选股建议（文稿时间点参考）

| 维度 | 内容 |
|------|------|
| 大盘状态 | 30分钟盘整，无系统性风险，不宜重仓 |
| 热点板块 | 教育（短线新热点，有核心驱动）；工业母机/电力/煤炭（前期板块的进三短线机会）；医美（有资金一致性但无外部驱动，只适合短线） |
| 选股方法 | 每日复盘涨停股 → 筛选有资金一致性+外部驱动+正在构造看多结构的个股 → 放入鱼池 → 跟踪买点 |
| 操作节奏 | 短线做进三；中长线从进一/进二底仓开始，跑出趋势则持有，否则换股 |
