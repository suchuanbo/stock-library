# 课53：三阶段讲八 — 资本市场的思维源头

## 一句话摘要

> **资本市场的思维源头是——市场无形，唯我有形；以结构为载体，以核心驱动为选股依据，以操作日志为成长载体，每日持续关联性复盘，严控股数不变，用好主动/被动卖点，接受卖飞与震荡，方能实现稳健持续盈利。**

## 二、操作表格（操作日志）的核心细则

### 2.1 为什么需要操作表格？

操作表格解决操作中的两大核心问题：

| 问题 | 表现 | 解决 |
|------|------|------|
| **空白** | 盘中不知道下一步该做什么，脑海无信号、无计划 | 表格让你每时每刻都知道触发哪些信号 |
| **期待** | 只看到自己期望的走势，忽略市场实际信号，容易死扛或错失 | 基于当下信号决策，不测而测（不对未来期许） |

### 2.2 操作表格与复盘表格的关系

| 表格类型 | 用途 | 关联 |
|----------|------|------|
| **复盘表格** | 选股、跟踪核心驱动、调整总账户仓位（大盘择时） | 选出股票 → 建仓 → 进入操作表格 |
| **操作表格** | 针对已持仓个股，每天记录跟踪、触发信号、交易动作 | 由复盘表格驱动，每日持续更新 |

### 2.3 建仓初始记录

| 记录项 | 说明 |
|--------|------|
| 日期、个股名称 | 建仓当日 |
| 建仓股数 | **股数**（不是金额）——风控的核心是控制股数不变 |
| 建仓成本 | 买入价 + 手续费 |
| 仓位上限 | 中小资金同时持仓建议 **2-3只**（最多5只），每只股数初始即定，后续不变 |

### 2.4 建仓后的跟踪流程（核心框架）

```
建仓
  │
  ├─→ 第一步：建仓成本是否被跌破？
  │     ├─ 跌破 → 立即进入「反踢」模式（先卖后买）
  │     │         记录「做多成本」（安全边际价格 = 结构破坏点）
  │     └─ 未跌破 → 继续持有，可做正踢（先买后卖），不会做可不做
  │
  ├─→ 第二步：跟踪「做多成本」是否被跌破？
  │     ├─ 做多成本 = 安全边际（中枢上轨、170线、34线等，根据级别定义）
  │     │   跌破 → 触发被动卖点：清仓（只留100股观察），等待下一次买点回补
  │     └─ 未跌破 → 持股待涨，同时跟踪主动卖点
  │
  ├─→ 第三步：主动卖点 vs 被动卖点（两者并行，谁先触发谁执行）
  │     ├─ 主动卖点：级别上涨出现背驰（30分钟/5分钟背驰），或按盈利比例（40-50%）
  │     │            触发后 → 卖半仓
  │     │            卖半仓后：
  │     │              ├─ 出现进三买点 → 补回仓位（股数不变，成本降低）
  │     │              └─ 出现被动卖点 → 清仓剩余半仓
  │     └─ 被动卖点：结构破坏（做多成本被击穿）
  │                   触发后 → 清仓（留100股）
  │
  └─→ 第四步：震荡处理
        股价在建仓成本与做多成本之间震荡，既未破成本也未破做多成本
        → 可做正踢/反踢（提高熟练度），不会做则耐心等待，不随意买卖
```

### 2.5 做T（正踢/反踢）的定位

| 要点 | 说明 |
|------|------|
| 定位 | **不是救命稻草，只是锦上添花**。前提是先会一买一卖 |
| 方向判断 | 5分钟级别趋势 + 1分钟背驰 |
| 正踢（先买后卖） | 5分钟向上 + 1分钟下跌背驰 |
| 反踢（先卖后买） | 5分钟向下 + 1分钟上涨背驰 |
| 心态 | 做T会卖飞或买不回，属正常。选择性地做，不追求100%成功 |

### 2.6 操作日志的每日更新

- 每天收盘后，将前一天的记录复制粘贴新一行，修改为当天的状态
- 记录：当前触发了哪个信号（建仓成本破？做多成本破？主动卖点？被动卖点？震荡？）
- **长期坚持**，形成连续的个人成长轨迹，避免「每天都是独立的一天」

## 三、资本市场的思维源头（核心认知）

### 3.1 市场无形，唯我有形

| 维度 | 内涵 |
|------|------|
| **市场是无形的** | 涨跌无常，资金流动没有固定轨迹，如同水一样 |
| **自我是有形的** | 你的结构分析、核心驱动判断、买卖点规则、操作表格——都是你的「有形工具」 |
| **核心载体** | 结构（体现涨跌的规律）+ 核心驱动（体现资金的合力） |

### 3.2 市场不存在唯一标准，唯一标准是你自己

- 没有「必涨的买点」或「必跌的卖点」，只有你自己的规则
- 买、洗、买、破、卖、破……是一条循环路径，任何时候都有多种可能
- 你要做的是知道**每个当下触发哪个信号**，而非预测未来

### 3.3 资金一致性与市场总龙头

| 概念 | 说明 |
|------|------|
| **资金一致性** | 固有资金（江湖老大）+ 游散资金共同做多某一板块 |
| **资金承接** | 一个旧的核心驱动板块结束（龙头个股出现卖点）→ 新核心驱动出现 → 资金流入新板块 |
| **注意** | 震荡市中可能只有一只核心驱动个股；不追求「市场总龙头」，同期多只上涨个股都可操作 |

### 3.4 职业成长的真相：每日关联性成长

> 股市的长期投资 ≠ 长持一只股票，而是个人每日按体系操作，形成连续的正向积累。

| 关键认知 | 说明 |
|----------|------|
| 今天的你与昨天的你、一个月前的你，**必须有关联**（通过复盘表格、操作日志） |
| 没有关联性 → 活了一年也只活了一天 |
| 有关联性 → 每一天都在成长 |

## 四、对学员的叮嘱与常见问题纠正

### 4.1 不要纠结「卖飞」

- 按自己的主动卖点卖出后，后面再涨 → 那是**新的买点**，不是卖点错误
- 如果每次都能「卖飞」（即每次都能盈利卖出），你已经比绝大多数人优秀

### 4.2 不要迷恋做T

- 做T无法扭转选股错误或结构破坏
- 先学会一买一卖（主动+被动卖点），再做T

### 4.3 不要无效止损，也不要死扛

- 无效的频繁止损 + 深套不止损，是亏损两大根源
- 严格按「做多成本」定义被动卖点，触发则清仓留100股

### 4.4 不要期待一只股票暴富

- 稳健持续盈利 = 赢多亏少，靠复利，不是靠单只股票翻倍
- 80%的时间账户可能震荡，20%的时间快速增长，**接受这种节奏**

### 4.5 关于财富观与人生

- 不要把钱财看得太重，格局打开，才能突破资金境界
- 股市没有「适合不适合」，只有「会与不会」。人性共通，学就会

## 结构速查清单

```
操作表格跟踪检查清单：
□ 今日触发信号：建仓成本破？ / 做多成本破？ / 主动卖点？ / 被动卖点？ / 震荡？
□ 股数是否保持不变？
□ 做多成本（安全边际）是否已定义清楚？
□ 主动卖点与被动卖点谁先触发？
□ 收盘后是否已将今日记录更新到操作日志？

思维源头每日检查清单：
□ 今天按自己的规则操作了吗？（非按市场情绪）
□ 今日复盘与昨日复盘是否关联（同一批股票跟踪）？
□ 核心驱动有无切换（新题材取代旧题材）？
□ 操作日志是否更新（每天一行，持续累积）？
```
