# 课43：二阶段（下）讲九 — 职业买卖之防二

## 一句话摘要

**防守买二（防二）是防守买点的第二种类型，核心在于通过背驰判断和动态线段重组来捕捉30分钟级别下跌的终结。** 防二与防一本质相同，区别仅在于下跌的呈现方式——防一是标准线段回撤，防二是需要通过动态重组来识别的非标准回撤。

---

## 一、课程定位与核心目标

| 维度 | 说明 |
|------|------|
| 所属阶段 | 第二阶段（下）第九课，防守买点的第二种类型 |
| 前置知识 | 课38-39：线段终结（小级别回撤判断）；课35：买点本质（三要素）；课42：防守买一 |
| 后续课程 | 课44：进三（一）——滚动买点 |
| 核心目标 | 正确理解背驰的本质（不是神技，而是工具）；掌握盘整背驰与趋势背驰的区别及实战应用；理解动态线段重组在30分钟级别回撤中的作用；明确防二与防一的区别 |

### 防二 vs 防一的核心区别

| 维度 | 防一 | 防二 |
|------|------|------|
| 回撤形态 | 标准的30分钟级别线段回撤（日线图上可清晰画出） | 非标准回撤（多条小线段、虚线转折混合） |
| 识别方式 | 直接看图即可识别 | 需要动态线段重组后才能识别背驰 |
| 本质 | 30分钟下跌终结 | **30分钟下跌终结**（本质相同） |

> **核心认知**：防一和防二的本质完全相同，都是30分钟级别下跌的终结。区别仅在于下跌的呈现方式不同——标准线段 vs. 需要动态重组的非标准线段。在实际交易中，可将两者合并理解。

## 二、背驰的本质与常见误区

### 2.1 背驰与背离的关系

| 概念 | 说明 |
|------|------|
| 传统背离 | 价格创新高/新低，但指标（如MACD）未创新高/新低 |
| 背驰 | 结合**级别与结构**的动能衰竭判断，比背离更精细 |

背驰与背离本质相似，都是上涨/下跌动能的衰竭信号。但背驰纳入了级别和中枢结构，判断更为严谨。

### 2.2 背驰的核心定义

```
所有背驰，最终都是某一级别的趋势背驰。
从最小级别看，任何下跌的结束，都对应某个级别的趋势背驰。
```

**两条铁律**：

| 铁律 | 说明 |
|------|------|
| **无趋势，不背驰** | 下跌必须形成至少一个中枢，才有背驰可言 |
| **不创新低，不背驰** | 背驰一定发生在创新低/新高之后 |

### 2.3 两种背驰（同一级别下）

| 类型 | 定义 | 实战意义 |
|------|------|----------|
| **盘整背驰** | 只有一个中枢的下跌结束（无三卖） | 可靠性较低，最好等34均线突破后再买入（突破买入） |
| **趋势背驰** | 至少两个中枢、有三卖终结后创新低的下跌结束 | 可靠性高，适合潜伏买入 |

> **注意**：同一段下跌从不同级别看可能呈现不同的背驰类型（如30分钟看是盘整背驰，5分钟看是趋势背驰）。判断时需明确操作级别。

## 三、背驰的实战应用原则

### 3.1 背驰一定会带来上涨——但幅度和级别不可控

| 原则 | 说明 |
|------|------|
| 必然上涨 | 背驰出现后，必然有技术性反弹 |
| 幅度不定 | 反弹可能只有一根K线，也可能是一段趋势 |
| 不可单独依赖 | 必须结合**结构和安全边际**，不能仅凭背驰做买卖决策 |

### 3.2 背驰的有效性取决于三卖

| 有无三卖 | 类型 | 可靠性 | 建仓方式 |
|----------|------|--------|----------|
| **有三卖** | 趋势背驰（中枢终结后出现三卖并创新低） | **高** | 潜伏买入 |
| **无三卖** | 盘整背驰（无中枢终结） | **低** | 突破买入（等34均线突破后再入） |

> **核心公式**：趋势背驰 = 中枢 + 三卖 + 创新低 + 力度衰减。四者缺一不可。

### 3.3 背驰级别与操作级别的匹配

| 操作级别 | 最小背驰级别要求 | 适用场景 |
|----------|------------------|----------|
| 做T（日内滚动） | 1分钟、5分钟趋势背驰 | 短线滚动 |
| 防守买点（30分钟级别） | **5分钟走势**（肉眼可见的5分钟中枢） | 波段建仓 |
| 日线级别防守 | 30分钟走势 | 中长线建仓 |

> **规则**：操作级别越大，要求的背驰级别越大。小级别背驰只适合做短线T，不适合做波段建仓。

## 四、动态线段重组

### 4.1 为什么需要动态重组？

某些30分钟级别的回撤，在日线图上**无法画成一条标准线段**：
- 可能由多条小线段组成
- 可能存在虚线转折
- 日线线段画法无法直接呈现

为了判断背驰，需要将这些非标准走势按照五均线的逻辑**重组成一条虚拟的30分钟级别线段**，然后分析其内部结构（中枢、三卖、背驰）。

### 4.2 重组的三条原则

| 原则 | 说明 |
|------|------|
| **只重组30分钟级别** | 不能重组为日线级别 |
| **必须有肉眼可见的中枢** | 重组后的线段至少有三笔五均线的重叠 |
| **先找三卖，再判背驰** | 重组后寻找三卖点，然后判断背驰 |

### 4.3 动态重组的简化操作

你**不需要纠结日线线段是否标准**。只需要看：

```
从高点到低点之间：
① 是否存在五均线的上下震荡形成中枢？
② 是否有三卖？
③ 是否创新低后出现背驰？
```

如果三者都满足，则视为一条有效的30分钟下跌线段，可以寻找防守买点。

### 4.4 案例

- 朗姿股份
- 电光网络

这些个股的防守买点，都是通过动态重组才能发现背驰的。

## 五、防守买二（防二）vs 防守买一（防一）

| 维度 | 防一 | 防二 |
|------|------|------|
| 下跌形态 | 标准30分钟线段（日线图可直接画线） | 非标准线段（需动态重组） |
| 识别难度 | 较低 | 较高（需重组后分析） |
| 背驰判断 | 直接在线段内判断 | 重组后在虚拟线段内判断 |
| 本质 | **30分钟下跌终结** | **30分钟下跌终结** |
| 安全边际 | 中枢上轨/170均线附近 | 中枢上轨/170均线附近 |

> **合并理解**：在实际交易中，防一和防二可统一理解为「30分钟下跌终结买点」。区别仅在于呈现方式，不影响操作逻辑。

## 六、交易理念与资金管理（复习与深化）

### 6.1 买点只有两类

```
下跌终结买点（防守买点）
  ├── 防一：标准线段回撤
  └── 防二：非标准回撤（动态重组）

上涨中继买点（进攻买点）
  ├── 进一：30分钟加速买点
  ├── 进二：日线级别起涨点
  └── 进三：滚动操作买点
```

学习多种买点是为了让你**看见**这些结构，最终形成自我的买点认知，而不是死记硬背模型。

### 6.2 市场是活的，买点也是活的

| 原则 | 说明 |
|------|------|
| 买点可能构造失败 | 如进一失败后砸下来，但可能在后面形成同级别的防守买点 |
| 不要因一次止损否定整只股票 | 只要结构未破坏，可留100股跟踪，等待新的买点 |
| 止损不是认输 | 止损是风控；止损后若结构仍在，可在新买点加倍接回 |

### 6.3 建仓与止损

| 规则 | 说明 |
|------|------|
| 任何下单都有风险 | 不要追求「买了必涨」 |
| 小资金持股上限 | 最多同时持有 **3只** 股票（50万以内资金） |
| 分仓建仓 | 每只股票分2次建仓（如第一次60%，第二次40%），留机动资金 |

### 6.4 什么是「满仓」？

> **满仓不是一开始计划好的，而是持仓全部盈利、资金不够用时的自然结果。**

满仓干的前提是：**所有持仓都在上涨趋势中，利润在奔跑。**
