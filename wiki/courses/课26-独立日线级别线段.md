# 课26：二阶讲十一 — 走势分类之独立日线级别线段

## 一句话摘要

**独立的日线级别线段是走势分类的基本单元，通过新低、一段、三买卖三种方式判断；其目的是分割走势、识别转折、组合成日线级别走势，最终服务于看多区域与做多中枢的定位。**

---

| 维度 | 说明 |
|------|------|
| 所属阶段 | 第二阶段（走势分类），走势分类系列第一课 |
| 前置知识 | 课16~25：日线级别线段、本质走势中枢、走势中枢唯一、级别唯一判定 |
| 课程性质 | **走势分类系列入门课**，从工具学习转向综合应用 |
| 核心目标 | 掌握独立日线级别线段的定义、三种判断方法，理解其在走势分类中的基本单元地位 |

```

```

## 二、走势分类的核心目的

### 2.1 市场本质与交易落脚点

```
市场的核心变化 = 大资金的驱动
唯一可量化的落脚点 = 结构
```

交易需要解决三个问题：

| 问题 | 核心 | 回答 |
|------|------|------|
| **因势而谋** | 个股属于上涨趋势吗？ | 看多区域 |
| **因势而动** | 从哪里启动？ | 做多中枢 + 买点 |
| **持多周期** | 涨多久？什么级别？ | 交易级别判定 |

### 2.2 走势分类的最终目标

```
市场的涨跌走势
    ├── 日线级别走势（上/下）
    └── 独立的日线级别线段
           │
           ▼ 分类组合
    ├── 看多区域（趋势方向）
    ├── 做多中枢（买点产生的位置）
    └── 交易级别（持仓周期 & 资金利用率）
```

> ⚠️ **核心观点**：唯一不变的是日线级别走势中枢，而不是独立的日线级别线段本身。

---

## 三、独立日线级别线段的定义与作用

### 3.1 定义

> 独立日线级别线段是从整体走势中**单独抠出来**的一段日线级别线段。

### 3.2 作用

| 作用 | 说明 |
|------|------|
| **分割走势** | 找到转折点（如下跌→上涨的转折） |
| **组合周线级别** | 独立线段是组合成周线级别走势的基本材料 |
| **分类基本单元** | 是分类组合法则的最小操作单元 |

### 3.3 级别的理解

```
独立日线级别线段的内部级别可以是：
  ├── 30分钟级别（如标准定义线段，无日线中枢）
  └── 日线级别（如标准走势线段，有日线中枢）

但作为"独立"工具，它一定是日线级别线段（符合日线级别线段法则）。
```

### 3.4 独立的前提：必须有反向 ⭐

```
独立线段一般是下跌方向（A股只做多，下跌独立用于捕捉转折）。

前提条件：
  独立之前必须有反向的日线级别线段（上涨）
  └── 否则无法独立
      └── 例：从头跌到尾的走势 → 只能是一条完整的日线级别下跌走势，不能分割
```

---

## 四、独立日线级别线段的三种判断方法 ⭐⭐⭐

### 4.1 方法一：新低独立（最简单）

| 条件 | 说明 |
|------|------|
| ① | 是一条日线级别下跌线段 |
| ② | 之前有反向的日线级别上涨线段 |
| ③ | 该下跌线段的**底**创了前一个下跌线段底的新低 |

> ⚠️ 同价不创新低。

```
示例：泰嘉股份等个股 — 新低下跌段即为独立。

前下跌段底 = 10元
当前下跌段底 = 9元  ← 新低 → 独立
```

### 4.2 方法二：一段独立（升级而来）

| 条件 | 说明 |
|------|------|
| ① | 是一条日线级别线段，由一条本质线段**升级**而来（内部有日线级别中枢） |
| ② | 之前有反向 |

> 在三期课程中更为常见，因为底层线段已综合唯一，无需非同处理。

```
示例：北部股份等 — 零段、一段的日线级别线段。

一条本质线段 → 内部震荡升级出日线级别中枢 → 一段独立
```

### 4.3 方法三：三买卖独立（走势分类中枢独立）⭐⭐⭐ — 重点难点

**核心判断原则**：

```
当走势分类中枢被三买或三卖确认结束时，
该中枢所属的日线级别线段就可以独立出来。
```

| 情景 | 判断 | 结论 |
|------|------|------|
| 上涨线段 + 中枢 + **三卖**（价格无法回到中枢上轨） | 上涨线段已结束 | ✅ 可独立分割 |
| 上涨线段 + 中枢 + 无三买也无三卖 | 无法证明结束 | ❌ 不能独立（视为震荡延续） |
| 下跌线段 + 中枢 + **三买**（价格无法回到中枢下轨） | 下跌线段已结束 | ✅ 可独立 |

**课堂案例辨析**：

```
图1（三卖独立）：
  下跌后反弹形成中枢 → 再次下跌不回到中枢 → 三卖 → 该下跌线段独立

图2（三买独立）：
  上涨中枢之后回踩不进入中枢 → 三买 → 该上涨线段独立

图3（无独立）：
  虽有中枢，但始终没有三买或三卖确认结束 → 不能独立 → 视为同一走势的震荡
```

**课后思考题**：

> 两幅图中，一个中枢成立，一个中枢不成立，但都出现了三卖，哪个能独立？
>
> **答案**：有中枢的才能通过三卖证明结束从而独立；无中枢的所谓"三卖"不成立。

**总结**：三买卖独立的本质 = **走势分类中枢的独立** — 中枢被破坏（三买/三卖）即证明该段走势结束，可独立分割。

## 五、分类组合法则简介（预告）

有了独立日线级别线段后，对整体走势进行分割与重组：

```
整体走势
  ├── 独立线段 → 单独拿出
  └── 非独立部分 → 组合成日线级别走势（上涨或下跌）
        │
        ▼ 唯一走势分类
  ├── 确定看多区域（以170均线为辅助）
  ├── 定位做多中枢（日线级别中枢 或 30分钟级别中枢）
  └── 匹配交易级别（周线 / 日线 / 30分钟）
```

## 九、核心总结

```
独立日线级别线段
  = 走势分类的基本单元

三种判断方式：
  ① 新低独立 — 下跌线段底创新低
  ② 一段独立 — 本质线段升级，内部有日线中枢
  ③ 三买卖独立 — 走势分类中枢被破坏，走势结束

目的：
  分割走势 → 识别转折 → 组合成日线级别走势
  → 服务于看多区域与做多中枢的定位
```
