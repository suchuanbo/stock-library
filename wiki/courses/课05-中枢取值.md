# 课5：本质线段中枢之中枢取值

## 一句话摘要

本课是**线段中枢静态知识点的收官之讲**。在起手三式 + 黑K确认中枢成立之后，核心解决两个问题：①中枢是「肉」还是「非肉」（正向一笔+五均线反向判断）；②肉/非肉分别如何取上下轨（波峰波谷 vs 连续两K重叠区）。上轨或下轨有一侧固定为中枢线，另一侧按规则取，所有规则均非黑即白。

## 二、肉/非肉判断深化

> 课4已初步引入：以中枢线确定K的五均线值为基准，看正向一笔之前五均线是否反向。本课将该法则**完整展开**，尤其攻克「正向一笔起始K」这一核心难点。

### 2.1 判断法则（总述）

```
判断窗口：从正向一笔的起始K → 正向一笔完成（含刚刚完成的那根K线）
参照物：中枢线确定K对应的五均线值（一条固定价位线）

在窗口内：
├─ 五均线出现过反向（跌破或等于参照价位） → 肉（肉眼可见）
└─ 始终未出现反向                           → 非肉（非肉眼可见）
```

> ⚠️ **判断对象是五均线的值，不是K线的价格。** 眼睛必须从K线转向均线。

### 2.2 正向一笔起始K的确定（难点攻克）

**方向法则**：
- 价格持续新低 → 方向向下
- 价格持续新高 → 方向向上
- **连续性铁律**：没有向上一笔的结束，就不会有向下一笔的开始；反之亦然。

**起始K的找法**：

| 情况 | 处理 |
|------|------|
| 中枢线确定K之前**有**反向一笔 | 必须等该反向一笔**结束**（不再创新低/新高）后，才能开始寻找正向一笔的起始K |
| 中枢线确定K之前**无**反向一笔 | 以中枢线确定K**本身**为正向一笔的起始K |

> 核心心法：必须站在**时间当下**的跟踪思维去判断，不能事后静态看图——事前的一笔方向是逐步展开的，不是画好后再判断。

### 2.3 五均线反向的准确定义

| 线段方向 | 正向 | 反向条件 |
|----------|------|----------|
| 上涨线段 | 向上 | 五均线值 **向下跌破或等于** 参照价位 |
| 下跌线段 | 向下 | 五均线值 **向上突破或等于** 参照价位 |

> 「等于」也算反向——临界值统一归入反向（非黑即白不留灰色地带）。

### 2.4 结论速查

| 正向一笔窗口内五均线表现 | 结论 |
|--------------------------|------|
| 出现过反向（含等于） | **肉** — 肉眼可见，后续必产生均线缠绕 |
| 始终未反向 | **非肉** — 非肉眼可见，单边但有中枢 |

## 三、下跌线段中枢取值

### 3.1 基础铁律

```
下跌线段：中枢线 = 上轨（固定不变，与肉/非肉无关）
          下轨 = 需取值
```

### 3.2 肉中枢 → 取前两波谷较高者

| 步骤 | 操作 |
|------|------|
| 1 | 确认该中枢为「肉」（五均线有波峰波谷缠绕） |
| 2 | 在均线缠绕中找出**前两个波谷**（量化定义见第五节） |
| 3 | 取两个波谷中**较高者**作为下轨 |
| 4 | 若第二个波谷无效（如未完成），则只取第一个 |

```
上轨 = 中枢线（固定）
下轨 = max(波谷1, 波谷2)
```

### 3.3 非肉中枢 → 找最左连续两K下沿

| 步骤 | 操作 |
|------|------|
| 1 | 以中枢线价位拉一条水平线，将中枢内所有K线分为**上下两个区域** |
| 2 | **只关注下区域**（中枢线下方） |
| 3 | 在下区域中，找到**最左边的连续两根K线** |
| 4 | 取这两根K线的价格重叠区：**高点取低者，低点取高者** |
| 5 | 该重叠区的**下沿**即为线段中枢的下轨 |

```
重叠区上沿 = min(K1高点, K2高点)
重叠区下沿 = max(K1低点, K2低点)
下轨 = 重叠区下沿
```

> 若下区域中**不存在连续两根K线**（如极端跳空），按跳空规则处理（见第六节）。

## 四、上涨线段中枢取值

### 4.1 基础铁律

```
上涨线段：中枢线 = 下轨（固定不变，与肉/非肉无关）
          上轨 = 需取值
```

### 4.2 肉中枢 → 取前两波峰较低者

| 步骤 | 操作 |
|------|------|
| 1 | 确认该中枢为「肉」（五均线有波峰波谷缠绕） |
| 2 | 找出**前两个波峰** |
| 3 | 取两个波峰中**较低者**作为上轨 |
| 4 | 若第二个波峰无效，则只取第一个 |

```
下轨 = 中枢线（固定）
上轨 = min(波峰1, 波峰2)
```

> 前两个波峰取较窄者——目的是取**最保守**的上轨边界，宁可中枢偏窄也不可偏宽。

### 4.3 非肉中枢 → 找最左连续两K上沿

| 步骤 | 操作 |
|------|------|
| 1 | 以中枢线价位拉水平线，分为上下两个区域 |
| 2 | **只关注上区域**（中枢线上方） |
| 3 | 找到**最左边的连续两根K线** |
| 4 | 取价格重叠区（高点取低者，低点取高者） |
| 5 | 该重叠区的**上沿**即为线段中枢的上轨 |

```
上轨 = 重叠区上沿 = min(K1高点, K2高点)
下轨 = 中枢线（固定）
```

## 五、波峰波谷的量化定义

> 这是肉中枢取值的底层算法，必须精确到每一天的母均线值。

### 5.1 波峰定义

```
某一天的母均线值：
├─ 比前一天高（或等同）
└─ 比后一天高（或等同）
→ 该天为一个波峰
```

### 5.2 波谷定义

```
某一天的母均线值：
├─ 比前一天低（或等同）
└─ 比后一天低（或等同）
→ 该天为一个波谷
```

### 5.3 关键规则

| 规则 | 说明 |
|------|------|
| 比较范围 | **只与前后相邻两天比较**，不与更远的比较 |
| 同价处理 | 同价也算波峰/波谷（停顿即为峰/谷） |
| 交替原则 | 相邻波峰与波谷必然交替出现（不会连续两个波峰） |
| 本质 | 肉眼可见中枢的均线缠绕 = 一天一天之间波峰波谷的**锯齿**组成 |

## 六、跳空处理

### 6.1 非肉中枢的跳空

在非肉中枢找连续两根K线时，若出现价格跳空：

| 情况 | 处理 |
|------|------|
| 两根K线之间有跳空缺口 | **默认中间插入一根虚拟K线**，视为连续 |
| 取值 | 取**跳空前那根K线**的价位（最低价或最高价）作为重叠边界 |

> 本质：跳空不打断「连续」认定，但用跳空前K线的边界来控制重叠范围。

### 6.2 肉中枢的跳空

跳空**不影响波峰判断**——仍按前后天的母均线值比较规则正常判定。均线是连续的，不会因为价格跳空而断裂。

## 七、案例

> 文稿中通过多个作业案例反复演示，关键点是**时间当下的跟踪思维**，不能事后静态看图。

### 7.1 典型案例汇总

| 案例 | 线段方向 | 肉/非肉 | 取值要点 |
|------|----------|---------|----------|
| 江苏吴中 | 下跌 | 肉 | 找前两波谷，取较高者为下轨 |
| 贵州茅台 | 上涨 | 非肉 | 上区域找最左连续两K，取重叠区上沿 |
| 金益科技 | 上涨 | 肉 | 前两波峰取较低者为上轨 |

### 7.2 核心方法启示

- **先判肉/非肉，再取值**——顺序不可颠倒。先确定正向一笔起始K → 看五均线是否反向 → 确定类型 → 按对应规则取值。
- **同价即临界**——波峰/波谷判断中「同价也算」，五均线反向中「等于也算反向」，非黑即白不留模糊空间。
- **取窄不取宽**——肉中枢取前两波峰/波谷中较窄者；非肉中枢取最左连续K线重叠，天然取最紧区间。

## 八、常见错误

| 错误 | 正确做法 |
|------|----------|
| **混淆判断与取值** | 肉/非肉判断回答「中枢形态是什么」，取值回答「上下轨具体价位」——二者先后有序，不可混为一谈 |
| **用K线价格代替五均线值判断反向** | 反向判断对象是**五均线的值**，眼睛从K线转向均线 |
| **事后静态看图判断正向一笔起始K** | 必须站在时间当下，模拟「不知道未来」的视角逐步跟踪 |
| **把正向跳空当成反向标志** | 只有五均线值本身的跌破/突破才算反向，K线价格跳空是另一个概念 |
| **肉中枢用连续K线法取值** | 肉中枢用波峰波谷法（均线缠绕本身就给出了边界），非肉中枢才用连续K线法 |
| **忽略「等于」临界值** | 五均线等于参照价位=反向，同价=波峰/波谷——临界值一律纳入，不留灰色 |
| **波峰波谷与更远的比较** | 只与**前后相邻两天**比较，不与更远的比较 |
| **跳空打断连续K认定** | 非肉中枢跳空插入虚拟K线，仍视为连续 |

## 九、大盘

课程当日（及附近）市场状态：

- **上证指数**：仍处于**30分钟级别中枢震荡**，5分钟级别向上，但**未形成30分钟级别进攻买点**
- **关键观察**：若后续突破中枢上方并形成进攻买点，市场可能加速上行；否则继续维持震荡格局
- **三类个股格局**：
  - ① 前期已布局、未出卖点的**相对高位股**（持仓赚钱效应）
  - ② 底部刚出现进攻买点的**启动个股**
  - ③ 仍在吸筹阶段的**底部股**
- **整体定性**：仍属**局部行情**，非普涨牛市

**操作建议**：
- 不会选股者 → 不强行操作，耐心学习掌握法则
- 会选股者 → 严格按买卖点规则操作，尤其注意高位股的卖点信号
- 仓位管理 → 震荡市以防守思维为主
