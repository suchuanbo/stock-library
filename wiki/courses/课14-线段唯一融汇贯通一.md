# 课14：线段唯一融汇贯通（一）

## 一句话摘要

线段的唯一 = 本质生长的两种方式（五均线极值或价格极值）+ 延伸起点的正确选取（五均线最值）+ 时间当下的动态跟踪 + 自我检查（能否分成三段）。学习的关键是"用心看"明白本质，而非死记图形表象。

## 二、线段中枢唯一的两大核心思维

### 2.1 核心判断流程

**唯一难点**：判断前方飞肉是否会提前变成肉（临界条件）。其余情况按部就班：

```
有中枢线刷新
  └→ 检查是否触发重组
        ├─ 触发重组 → 前方全部无效，以新中枢为准
        └─ 不触发   → 按 k0、k 动态刷新处理
```

### 2.2 常见错误

| 错误 | 纠正 |
|------|------|
| 把"延伸"与"重组"混为一谈 | 重组是延伸的一种特殊形式，但**并非所有延伸都是重组** |
| 认为黑k与中枢重组有关 | 黑k只影响线段中枢是否**最终成立**，不影响重组触发 |

## 三、线段唯一的本质

线段唯一由三个核心要素构成：**两种生长方式 + 选取法则 + 时间当下动态思维**。

### 3.1 本质线段的新生（两种方式）

| 方式 | 触发条件 | 性质 |
|------|----------|------|
| **方式一：正常生长** | 五均线值持续创新高（上涨）或新低（下跌） | 常规 |
| **方式二：特殊法则** | 五均线值无生长，但价格创整个线段最高/最低，且满足实体位置关系 | 特殊 |

> 只有符合两种方式之一，该顶/底才有效。

### 3.2 线段顶底的选取唯一

**何时需要"选"？** 当存在多个可能作为延伸起点的顶/底时。

| 选取维度 | 规则 |
|----------|------|
| **常规选取** | 以五均线值最极值（最高或最低）的点为延伸起点 |
| **特殊法则下** | 价格极值才起作用（五均线不生长但价格创新极值） |

### 3.3 时间当下动态思维

画线段不是静态看全图，而是**站在每个当下**，逐步跟踪：

```
顶/底的产生 → 转折K的刷新 → 延伸起点的选取
```

每一步都在时间当下做判断，而非事后看图说话。

---

## 四、延伸的正确性自我检查法

### 检查方法

> 画出你认定的整条延伸线段后，看它内部**能否分成三段或更多完整的线段**。

| 检查结果 | 结论 |
|----------|------|
| **能分成三段**（同级别分解可行） | ❌ 延伸错误，应直接画成多段 |
| **不能分成三段** | ✅ 延伸正确 |

### 案例

| 案例 | 结果 |
|------|------|
| 华银电力 | 只能延伸成一条，无法分成三段 → 延伸正确 |
| 某些可分成四段的走势 | 不应延伸，应直接画成多段 |

## 五、作业与答疑中的典型问题

### 5.1 任何软件（个股案例）

- **线段中枢**：存在肉中枢，触发重组 → 前方飞肉无效
- **线段唯一**：转折K确定底 → 底确定后检查中枢 → 符合则成线段

### 5.2 星河环保（延伸选取问题）

| 问题 | 答案 |
|------|------|
| 从A到D延伸，选B1还是其他？ | B**不存在**（不成底），只能从A延伸，经过特殊法则刷新后，新底产生 |

### 5.3 某图三（延伸起点问题）

有同学想了三天三夜不明白。

**解答**：该图**不存在选取问题**——底已经被刷新无效，只能从新底开始。

### 5.4 五均线值创新高但价格未创新高

| 问题 | 答案 |
|------|------|
| 为什么选二不选一？ | 触发了顶底唯一特殊法则：原转折K之后无五均线生长，但价格创整个线段最高 → 原顶无效，新顶产生 |

## 六、学习建议与心态提醒

### 6.1 学习方法

| ❌ 错误做法 | ✅ 正确做法 |
|------------|------------|
| 死记表象：各种图形千变万化 | 抓住本质：只有**两种生长方式** |
| "用眼看"：看图形表象 | **"用心看"**：看五均线生长、价格极值、转折K刷新 |
| 空想三天三夜 | 回看视频——老师表示"视频原话已经讲得非常清晰" |

### 6.2 心态成长

- 交易是**心境成长**的过程，知识只是工具
- 最终成就有多大，在于个人的悟性和坚持
- 不要停留在"自我安慰的舒适区"，要敢于突破、敢于出错（出错才能入心）
- 成年人学习要为自己负责，老师的角色只是引路人

### 6.3 交易本质

| 要素 | 说明 |
|------|------|
| 交易核心 | 买点、卖点、选股、滚动 |
| 大盘的作用 | 仅用于**防控**（被动卖点）和起涨时的**选股优先级**，不用每日预测 |
| 中小资金定位 | 不应幻想一只个股发家致富，而应追求持续稳健的**滚动盈利** |
