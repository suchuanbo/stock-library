# 课25：二阶段讲十 — 职业交易之级别的唯一判定

## 一句话摘要

**级别的本质是「符合法则之下的时间周期」，核心在于走势中枢。** 日线级别由连续30分钟级别线段带来的34均线重叠判定；周线级别由三条及以上日线级别线段组合而成。掌握日线级别判定，其他级别自然清晰 — "除了日线，就是周线（或30分钟）。"

```
级别 ≠ 时间周期
级别 ≠ 线段段数
级别 = 符合法则之下的走势中枢

判定口诀：除了日线，就是周线（或30分钟）。
能判定日线级别，就能反推其他级别。
```

## 二、传统级别观念的误区 ⭐

### 2.1 常见错误认知

| 错误认知 | 为什么错 | 反例 |
|----------|----------|------|
| **级别 = 时间周期** | 如果级别只是时间，直接用均线即可，无需复杂法则 | — |
| **级别 = 线段段数** | 九段不一定升级为周线，长时间震荡也不一定是大级别 | 上证指数、南方汇通 |
| **30分钟×3 = 日线，日线×3 = 周线** | 数学递归得出"30分钟×9 = 周线"，但实践中九段线段并不必然构成周线级别 | 南方汇通 vs 东华科技 |

### 2.2 问题的根源

```
递归需要精确的数理单位
但不同线段、不同周期图的"单位"并不统一
—— 停盘、跳空、连续涨停等导致时间断层

→ 纯粹按段数或时间跨度定级别 = 错误结论
```

**典型案例**：南方汇通与东华科技时间跨度相近，但南方汇通是**日线级别**，东华科技是**周线级别**。

## 三、本课程的简化级别体系 ⭐

### 3.1 只保留两种基础级别

| 级别 | 定义 | 地位 |
|------|------|------|
| **30分钟级别线段** | 本质线段，底层基本单位 | 最小分析单元 |
| **日线级别走势** | 由日线级别线段构成（本质走势中枢） | 核心交易级别 |

> 📌 5分钟级别不做独立判定，仅用于买卖点内部细节（5K重叠、三笔、反正两穿）。

### 3.2 级别的本质定义

```
级别 = 符合法则之下的走势中枢

走势中枢由次级别线段（30分钟级别）的三段及以上重叠构成

┌─────────────────────────────────────────────┐
│ 有日线级别走势中枢 → 日线级别              │
│ 无日线级别走势中枢，但有30分钟走势中枢       │
│   → 30分钟级别                             │
│ 周线级别 = 三条及以上日线级别线段组合而成   │
└─────────────────────────────────────────────┘
```

### 3.3 判定口诀

| # | 口诀 | 含义 |
|---|------|------|
| 1 | **除了日线，就是周线（或30分钟）** | 只有这三个级别，无中间灰色地带 |
| 2 | **能判定日线级别，就能反推其他级别** | 日线级别是判定核心 |

## 四、日线级别走势中枢的精准判定（核心）⭐⭐⭐

### 4.1 判定依据

| 条件 | 说明 |
|------|------|
| **本质线段带来** | 必须由本质线段（30分钟级别）带来的34均线重叠构成 |
| **连续性要求** | 重叠必须由**连续性的30分钟级别**带来，不能有级别断层 |
| **前方级别** | 形成日线中枢的反向，其前方必须有连续的30分钟级别走势 |

```
日线中枢构造的连续性前提：

前方走势          反向走势
 30分钟 ─────→ 产生34均线重叠 ──→ 日线级别中枢
 （可不完整成线段，但级别必须达到30分钟）

❌ 前方是5分钟级别 → 无法触发日线中枢构造
```

### 4.2 非同级别分解（非同）的应用

| 场景 | 传统处理 | 非同处理 |
|------|----------|----------|
| 走势无法形成完整的本质线段（有虚线、断层） | 无法构造中枢 | **仍可构造日线级别中枢** |
| 北部股份、安道麦A、新国都 | 无中枢或低级别中枢 | 通过"零段"或"一段"的级别处理，找到34均线重叠 |

> 🔑 **非同的本质**：处理「不成线段、但成级别」的走势 — 将它们视为30分钟级别盘整或趋势，用于构造更高级别中枢。

### 4.3 日线级别 vs 周线级别

| 级别 | 判定标准 | 案例 |
|------|----------|------|
| **日线级别** | 日线级别线段内部有**日线走势中枢**（由30分钟线段带来） | 南方汇通 |
| **周线级别** | 由**三条及以上**日线级别线段组合而成 | 东华科技（下-上-下三条日线线段构成周线下跌） |

```
日线级别：   [日线线段] ← 内部有日线中枢（30分钟线段→34均线重叠）

周线级别：   [日线线段1] + [日线线段2] + [日线线段3] + ...
                  └── 三条及以上组合 ──┘
```

## 五、实战应用：级别与交易的关系

### 5.1 看多区域与做多买点的级别

| 概念 | 有级别？ | 说明 |
|------|----------|------|
| 看多趋势 | ✅ 日线级别 / 周线级别 | 决定上涨空间与持续性 |
| 买点 | ✅ 日线级别 / 30分钟级别 | 不同级别决定持仓策略 |

### 5.2 中小资金最适合的级别

| 级别 | 评价 |
|------|------|
| **30分钟级别走势**（日线线段内部波段） | ✅ **最佳** — 灵活、机会多 |
| **日线级别线段** | ✅ 适合 — 波段复利 |
| 5分钟级别 | ❌ 过小 — 容易错失大机会，非职业不友好 |
| 周线级别 | ❌ 过大 — 资金利用率低，机会少 |

### 5.3 选股与择时匹配

```
任何大牛股的初始底仓建立，
都有属于它当时的市场择时。

大盘下跌不可怕，
每次下跌之后都会出现新的牛股机会。

职业投资者的态度：
大盘下跌时兴奋（机会将至），而不是恐慌。
```

## 六、案例讲解

| 案例 | 核心要点 | 判定结果 |
|------|----------|----------|
| **某走势图（多段线段）** | 无日线中枢 | 30分钟级别，非日线级别 |
| **东华科技 vs 南方汇通** | 时间跨度相近，级别不同 | 东华=周线（三条日线线段），南方=日线（内部日线中枢但未构成周线） |
| **北部股份（零段处理）** | 将不成线段的走势视为30分钟级别盘整 | 通过34均线重叠判定为日线级别中枢 |
| **新国都** | 通过级别连续性（30分钟盘整）构造中枢 | 日线级别，非简单按线段画法 |
| **彩虹股份（转折中枢）** | 虚线-实线-虚线构成30分钟级别转折 | 30分钟级别，非日线级别 |

> 📌 北部股份"零段处理"是非同级别分解的典型应用 — 不完整成线段，但级别达到30分钟即可参与构造日线中枢。

## 九、核心总结

### 一句话精炼

> **级别 = 符合法则之下的走势中枢，不是简单的时间周期或段数。日线级别由连续30分钟级别线段带来的34均线重叠判定，周线级别由三条及以上日线级别线段组合而成。掌握日线级别判定，其他级别自然清晰。**

### 知识链

```
本质线段（30分钟级别）
    │
    ├── 三段及以上重叠 + 34均线重叠
    │        │
    │        ▼
    │   日线级别走势中枢 → 日线级别
    │
    └── 三条及以上日线级别线段组合
             │
             ▼
         周线级别
```

### 三大核心判定

| # | 判定 | 关键条件 |
|---|------|----------|
| 1 | **日线级别中枢** | 连续性30分钟级别带来的34均线重叠 |
| 2 | **日线级别 vs 周线级别** | 内部日线中枢 vs 三条日线线段组合 |
| 3 | **非同级别分解** | 不成线段但成级别 → 仍可构造高级别中枢 |

## 总结

本课是二阶段走势分类的**核心枢轴课**。级别判定不是数学递归（段数×倍率），而是**中枢法则**下的唯一判定：日线级别由30分钟连续走势带来的34均线重叠决定，周线级别由日线级别线段组合决定。非同级别分解是处理「线段不完整但级别达标」的关键工具。最终服务于实战：**看多级别决定上涨空间，买卖点级别决定持仓策略，中小资金以30分钟/日线级别为最佳操作区间。**
