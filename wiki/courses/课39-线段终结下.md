# 课39：二阶段（下）讲五 — 职业交易之线段终结（下）

## 一句话摘要

**小级别回撤终结的两种结构（趋势 vs 盘整）、背驰的辅助地位、两种建仓方式（潜伏 vs 突破）。** 无中枢时看趋势下跌+背驰→潜伏买入；有中枢时需三卖新低后背驰，或无三卖则用突破建仓法（反中枢线→回测不破前低→突破五均线）。背驰只是辅助信号，结构+安全边际+核心驱动力量才是根本。

## 二、小级别回撤的核心概念复习

> **「没有回撤，就没有买点。」** 任何买点都是某个级别的回撤带来的。

回撤的完整判断链：
```
回撤起始K线 → 方向延续（趋势 or 盘整）→ 安全边际 → 背驰/突破信号 → 买入
```

---

## 三、方向的判断（两根K线的趋势确认）

### 3.1 方向延续的定义

| 条件 | 结论 |
|------|------|
| 昨天下跌 + 今天下跌 + 今天收盘价 < 昨天收盘价 | **趋势性下跌成立** |
| 昨天下跌 + 今天下跌 + 收盘价≥昨天收盘价 | **趋势不成立**（无新低，盘整） |
| 下跌 + 上涨（反向） | 无趋势，可能形成中枢 |

> **最快的小级别回撤趋势可在两根K线内完成。**

### 3.2 六组K线案例分析

| 图序 | 走势组合 | 判定 |
|------|----------|------|
| 图1 | 下跌+下跌且创新低 | ✅ 趋势成立 |
| 图2 | 下跌+下跌但收盘价相同 | ❌ 趋势不成立（无新低） |
| 图3 | 下跌+上涨 | ❌ 无趋势，可能形成中枢 |
| 图4 | 上涨+下跌（先上后下） | ❌ 无法形成趋势性下跌 |
| 图5 | 上下上 / 下上下 | 盘整结构 |
| 图6 | 高开低走且收盘价创新低 | ✅ 趋势成立（两天完成回撤趋势） |

## 四、小级别终结的两种结构：有中枢 vs 无中枢

### 4.1 结构一：无中枢的趋势性下跌

| 步骤 | 动作 | 说明 |
|------|------|------|
| ① | 确认回撤开始K线 | 起始=最高价、跌破前日收盘价、跌破五均线 |
| ② | 方向延续 | 连续下跌创新低，形成趋势 |
| ③ | 价格回撤到安全边际 | 中枢上轨 / 170均线附近 |
| ④ | 出现背驰信号 | 新低后下跌力度衰减（MACD辅助） |
| ⑤ | **潜伏买入** | 提前介入，等待反转 |

> 案例：泰达股份、三美股份

### 4.2 结构二：有中枢的盘整下跌

| 特征 | 说明 |
|------|------|
| 下跌过程 | 形成折线中枢（上-下-上结构） |
| 直接背驰？ | ❌ 无法直接背驰 |
| 背驰前提 | 需要**三卖新低后**才可能出现背驰 |

#### 若无三卖新低 → 突破建仓法（三步）

```
第一步：出现反中枢线（由下向上突破中枢上轨）
第二步：回测不破前低
第三步：再次向上突破五均线 → 买入
```

> 案例：海王生物

### 4.3 两种结构对比

| 维度 | 趋势下跌（无中枢） | 盘整下跌（有中枢） |
|------|--------------------|--------------------|
| 结构特征 | 连续下跌创新低 | 下跌中有折线中枢 |
| 背驰条件 | 新低后力度衰减即可 | 需三卖新低后方可 |
| 无背驰时 | 等方向反转确认 | 用突破建仓法（三步） |
| 买入方式 | 潜伏买入 | 潜伏（有背驰）或 突破（无背驰） |

---

## 五、两种建仓方式对比

| 维度 | 潜伏买入 | 突破买入 |
|------|----------|----------|
| 触发条件 | 背驰信号出现 | 反中枢线 + 回测不破前低 + 突破五均线 |
| 优点 | 成本低，盈亏比高 | 确定性高，买入即涨 |
| 缺点 | 可能继续下跌，需承受浮亏 | 成本较高，可能追高 |
| 适用结构 | 趋势下跌（有背驰） | 盘整中枢（无背驰或不等背驰） |
| 实战要点 | 必须结合安全边际 | 三步缺一不可 |

> **课程建议**：职业交易者两者都要掌握；非职业者只用「回撤+安全边际+突破五均线」简单建仓，带止损即可。

---

## 六、安全边际（做多区域）在小级别中的应用

### 6.1 定义

```
安全边际 = 中枢上轨 或 170均线附近 或 二者交叉区域
```

### 6.2 铁律

> ⚠️ **买点必须发生在安全边际附近**，否则即使结构正确、有背驰，也不应建仓（风险大，盈亏比差）。

### 6.3 案例对比

| 个股 | 回撤情况 | 判定 |
|------|----------|------|
| 宏达股份 | 回撤到170均线附近 | ✅ 安全边际成立 → 可做潜伏 |
| 润和软件 | 回撤未到安全边际 | ❌ 无买点 |

---

## 七、背驰在小级别中的地位

### 7.1 核心观点

> **背驰是辅助信号，不是决定性因素。** 有背驰可能涨，无背驰也可能涨。

### 7.2 背驰的作用

| 条件 | 作用 |
|------|------|
| 结构正确 + 安全边际成立 | 背驰可**提高潜伏买入的成功概率** |
| 结构不对 / 无安全边际 | 背驰无用，不可单独依赖 |

### 7.3 背驰的条件

1. **必须有新低**（趋势性下跌）
2. **下跌力度衰减**（新低时MACD绿柱缩小 / 下跌斜率减缓）
3. 有中枢的趋势下跌：**需三卖新低后**才可能背驰

### 7.4 关键提醒

> 不要迷信背驰。**结构 + 安全边际 + 核心驱动力量**才是根本。

## 八、什么是真正的「买点」（中小资金视角）

### 8.1 买点本质

> **买点 = 主力不得不涨的关键启动点。**

| 步骤 | 主力动作 | 中小资金策略 |
|------|----------|-------------|
| ① 吸筹 | 低位建仓，降低成本 | 等待，不参与 |
| ② 洗盘 | 震荡出清浮筹 | 等待结构完成 |
| ③ 点火 | 放量突破，启动上涨 | **这就是买点** |
| ④ 拉升 | 趋势运行，需要抬轿 | 持有+滚动 |

### 8.2 实战含义

| 现象 | 说明 |
|------|------|
| 买入后不涨或震荡 | 该位置还不是真正的启动点（主力尚未完成准备） |
| 买入后立即上涨 | 主力已完成吸筹洗盘，选择了点火启动 |

> **中小资金的优势在于「赌」主力不得不涨，而不是与大资金对抗。**

## 九、行情与操作建议（国庆前）

| 维度 | 判断 |
|------|------|
| 大盘状态 | 30分钟级别震荡，无系统性风险 |
| 下跌性质 | 前期涨幅过大个股的利润兑换 |
| 仓位建议 | **50%以下**，有明确买点可建仓 |
| 选股方向 | 电力、环保、特高压、造纸等有核心驱动力量的板块 |
| 个股类型 | 进二、进三为主 |
| 节后展望 | 行情逐渐明朗，利用假期复盘、复习基础结构 |

## 十一、总结金句

> **「买点不是追涨，而是回撤后的必然。主力不得不涨的位置，才是中小资金的战场。背驰只是辅助，结构才是根基。」**

## 核心精炼

```
小级别回撤终结 = 方向确认 + 结构判断 + 安全边际 + 建仓方式

两种结构：
  无中枢趋势下跌 → 背驰出现 → 潜伏买入
  有中枢盘整下跌 → 三卖新低后背驰 → 潜伏买入
                   → 无三卖则用突破建仓法（反中枢线+回测不破+突破五均线）

两大铁律：
  ① 买点必须在安全边际附近
  ② 背驰是辅助，结构+安全边际+核心驱动才是根本

主力思维：
  买点 = 主力不得不涨的关键启动点
  中小资金优势 = 等主力点火后跟进，而非对抗
```
