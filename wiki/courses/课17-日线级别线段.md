# 课17：二阶段讲二 — 日线级别线段

## 一句话摘要

**日线级别线段是走势分类的起点。** 通过四个必要条件（三段、顺时关系、34/170交叉、连续性）精确划分日线级别线段，从而知道何时"看多"；看多之后，等待中枢上轨附近的买点再"做多"——这是职业交易结构的第一步。**看多 ≠ 做多**，分类为"还原过去、明确当下"，服务未来的买卖点。

三期不再单独讲解"同级别分解"与"非同级别分解"，而是**综合进线段的唯一画法和动态线段逻辑**中，避免理论干扰，直奔实战本质。

## 三、日线级别线段的四个必要条件

> 四个条件**缺一不可**，是判断日线级别线段的量化标准。

### 条件1：至少30分钟级别三段（线段三段）

日线级别线段必须由**至少3条30分钟级别线段**组合而成。

| 段数类型 | 具体形式 | 说明 |
|----------|----------|------|
| 常规段 | 3段、4.5段、5段、6.5段、7段、8.5段、9段等 | 最低**不少于3段** |
| 零段 | 尚未形成完整三段 | 通过震荡升级为日线级别线段（见下文） |

```
三段示例（最小单位）：
   ___     ___
  /   \   /   \    ← 第2段（30分钟级别线段）
 /     \_/     \_  ← 第1段、第3段
      3条30分钟级别线段组合 → 1条日线级别线段
```

### 条件2：顺时关系

> 口诀：**合n为一，只看端点，同价顺势。**

- **定义**：以第一条线段的底为起点，随时间推进，时间末端（线段终点）必须是所有组成线段中的**最高价**（或最值）。
- **只看线段的顶和底**，不看线段内部K线价格。
- **同价**也算顺时关系（临界状态）。
- **违背顺时关系**（时间末端价格不是最值）则**不能构成**日线级别线段。

```
顺时关系示例：
  ___B___
 /       \      B = 时间末端 = 最高价 ✓
A         \
           C
A = 起点（底），B = 终点（顶），B价格 > A、C → 顺时关系成立
```

### 条件3：34均线与170均线的交叉关联关系

日线级别线段必须由三段及后续段**带来34线与170线的交叉**（金叉/死叉）。

| 关键点 | 说明 |
|--------|------|
| 具体表现 | 日线级别线段的**顶和底位于170均线的两侧** |
| 粘合 | 同价**不算**交叉 |
| 本质 | 这是30分钟级别线段**升级为日线级别线段**的关键标志 |
| 滞后允许 | 允许滞后的交叉（通过"还原过去"来明确） |

```
34/170交叉示意：

   顶 ─────────── · · · · · · ───── 34均线
                  \   交叉   /
170均线 ───────────\───X───/──────── 170均线
                     \   /
   底 ─────────────── · · · · · · ── 34均线

顶在170线一侧，底在170线另一侧 → 交叉关系成立
```

### 条件4：日线级别线段的连续性

> **"无上则无下，无下则无上"**

没有反向线段，就不会有新方向线段的新生。如果一直没有出现反向的日线级别线段，则走势只能是**原方向的延续**，不会产生新的独立线段。

**日线级别线段的破坏有两种情况：**

| 破坏类型 | 说明 |
|----------|------|
| 独立 | 出现**反向独立**的日线级别线段 |
| 违背趋势 | 价格**创出新低/新高**，打破原有趋势 |

```
连续性示意：

上涨线段 —→ 无反向线段出现 —→ 原上涨延续（不产生新线段）
上涨线段 —→ 反向独立线段出现 —→ 新生下跌线段 ✓
上涨线段 —→ 价格创新低（违背趋势）—→ 原有上涨线段破坏 ✓
```

## 四、关键概念补充

### 4.1 只看线段顶底，不看线段内部价格

线段画出的**唯一目的**：过滤掉内部无关震荡，只保留大局观所需的端点。

- 一条线段内部的K线最高/最低价，如果超出线段端点，**不纳入考虑**。
- 这是避免被短期波动干扰判断的关键。

### 4.2 零段日线级别线段

| 属性 | 说明 |
|------|------|
| 来源 | 由30分钟级别线段**震荡升级**而来 |
| 特征 | 尚未形成三段，但已具备日线级别中枢 |
| 用途 | 仍可用于走势分类，属于"看多"的起点 |
| 案例 | 北玻股份 |

> 核心思维：不应强行把非三段走势处理为三段，而应通过**线段唯一画法直接判断**。

### 4.3 转折中枢与分类中枢

| 类型 | 出现位置 | 作用 |
|------|----------|------|
| **转折中枢** | 下跌 → 上涨的转换位置 | 突破其上轨即**看多** |
| **分类中枢** | 上涨过程中的顺势中枢 | 用于判断**买点位置** |

> 所有买点都出现在**中枢上轨附近**，而不是下轨。

## 五、实战案例要点

### 5.1 任何软件（个股案例）

- 通过日线级别线段划分，明确**何时看多、何时看空**。
- 股价突破转折中枢上轨 → **看多**；但需等待买点才做多。
- 若股价创出新低（跌破前低）→ 趋势破坏，转向**看空**。

### 5.2 彩虹股份 / 北玻股份

- 用于说明"非同"思维误区：不应强行把非三段走势处理为三段。
- 北玻股份 = **零段日线级别线段**案例。

### 5.3 重心菌业（众星菌业）

- 日线级别线段标准，上涨力度强，是**教科书式走势**。

## 六、资金管理与心态

| 要点 | 内容 |
|------|------|
| 核心原则 | 炒股不是赌大小，是有策略、有仓位管理、有风控的**系统** |
| 仓位示例 | 20% 尝试建仓 / 40% 浮盈加仓 / 40% 不动本金（永远保证安全） |
| 滚动操作本质 | 上涨过程**卖**，下跌过程**买**，而不是追涨杀跌 |
| 学习心态 | 不要追求"满分通关"，允许自己逐步理解；回归本质——看多/做多/中枢上轨 |

## 七、本课总结

```
学会用四个条件精确划分日线级别线段
         ↓
    知道何时「看多」
         ↓
    等待中枢上轨附近的买点
         ↓
    再「做多」
         ↓
  职业交易结构的第一步
```

> 走势分类 → 还原过去，明确当下 → 服务未来的买卖点
