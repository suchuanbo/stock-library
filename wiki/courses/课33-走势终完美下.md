# 课33：二阶段讲十八 — 职业交易看多（走势终完美下）

## 一句话摘要

**看多 ≠ 做多。** 看多是"不跌了"——下跌结构走完、具备上涨条件；做多是"开始涨了"——出现具体买点信号。走势中完美看多（转折看多）的核心量化标准：先有34均线反穿170均线（停顿），再有回踩后四条件满足（价格、34线、5线均在转折中枢上方 + 5线在170上方），此时才具备"不下跌"的充分条件。结构在前，买点在后，没有看多结构就没有真正的买点。

---

## 一、课程定位与目标

| 维度 | 说明 |
|------|------|
| 所属阶段 | 第二阶段（走势分类），走势终完美系列下篇 |
| 前置知识 | 课32（走势终完美上）、课31（走势分类）、课16~30（日线级别线段、中枢、级别、独立、周线中枢） |
| 课程性质 | **看多结构的量化讲解**——判断下跌结束、具备上涨条件的结构 |
| 核心目标 | 帮助学员在买卖点出现之前先建立"看多"结构概念，避免抄底在半山腰，提高交易的概率和稳健性 |

### 核心思想

```
结构在前，买点在后。
没有看多结构 → 就不会有真正的买点。

看多 = "不跌了"（可以开始跟踪）
做多 = "开始涨了"（出现买入信号）

先有看多结构，后有买点。
```

> ⚠️ **关键提醒**：千万不要看到看多就立即买入。看多只是下跌结构完美、不跌了，可能只是震荡；做多需要出现具体的买点信号（如进攻买点、防守买点）。

## 二、看多的两种主要类型

| 类型 | 本质 | 适用场景 | 本课重点 |
|------|------|----------|----------|
| **走势中完美看多（转折看多）** | 下跌结束，趋势转折 | 底部抄底、选股布局 | ✅ 核心重点 |
| **走势分类看多（趋势看多）** | 上涨中继，趋势延续 | 短线滚动、三买、进攻买点 | 要点梳理 |

> 走势中完美看多（转折看多）是本课重点，因为它最容易让人抄底在半山腰，也最需要精确量化。

**额外补充：独立转折看多**——由独立的日线级别线段带来的转折，只需五均线向上穿越170均线即可看多，不需要二次确认。

## 三、走势中完美看多（转折看多）的量化

### 3.1 前提条件：34均线的一次向上反穿

```
下跌过程中，必须有一条本质线段（30分钟级别）向上穿越170均线，
形成第一次停顿。

如果没有这一步 → 说明下跌连停都没有停顿过，不可能转折。
```

| 条件 | 说明 |
|------|------|
| 动作 | 30分钟级别本质线段向上穿越170均线 |
| 意义 | 下跌过程中的第一次"停顿" |
| 必要性 | **必须满足**，否则无从谈转折 |

### 3.2 核心逻辑：170均线"抬头"与"二次确认"

```
第一次穿越170均线（停顿）
    ↓ 价格回落
第二次回升时，若满足四条件 → 构成看多结构
    
    = 下跌大概率结束
    = 可以开始跟踪买点
```

### 3.3 四条件量化标准

| # | 条件 | 含义 | 
|---|------|------|
| ① | **价格**在转折中枢的上方 | 价格已突破关键阻力区 |
| ② | **34均线**在转折中枢的上方 | 次级别（30分钟线段）已向上 |
| ③ | **五均线**在34均线的上方 | 本质线段向上（走势方向确认） |
| ④ | **五均线**在170均线的上方 | 170均线"抬头"确认 |

> **这四个条件共同构成"不下跌"的充分条件，是看多结构的核心量化标准。**

### 3.4 直观理解

```
第一次向上（穿170均线）  = 停顿  →  跌势暂停
第二次向上（回踩后再次向上） = 抬头确认 → 跌势终结

抬头确认的那个时间点 = 结构上的看多点
                         → 此时可以开始跟踪买点
```

### 3.5 案例解析

| 案例 | 特征 | 结果 |
|------|------|------|
| **彩虹股份** | 反穿170均线后没有二次确认 | ❌ 看多不成立，继续下跌 |
| **任何软件** | 第一次反穿后回踩，第二次向上时满足四条件 | ✅ 看多成立，后续出现买点 |
| **美丽云** | 中枢上轨与170均线相切，满足四条件 | ✅ 看多成立 |

> 彩虹股份的错误：看到第一次穿越170均线就认为见底了 → 抄在半山腰 → 没有二次确认不能算看多。

## 四、走势分类看多（趋势看多）

### 本质与条件

| 维度 | 说明 |
|------|------|
| 本质 | 上涨过程中，价格在分类中枢上方，且满足类似条件 |
| 条件 | 价格、34均线、五均线均在**分类中枢**上方 |
| 特点 | 不需要复杂的转折判断，只要价格在中枢上方就处于看多状态 |

### 适用场景

```
走势分类看多更适用于：
    ├── 上涨趋势中的回踩买点（如三买、进攻买点）
    ├── 短线交易
    └── 超级滚动交易
```

### 案例：东华科技

> 分类中枢形成后，价格一直在中枢上方 → 看多延续。

## 五、独立转折看多（简化版）

| 维度 | 说明 |
|------|------|
| 本质 | 由独立的日线级别线段带来的转折 |
| 判断 | 独立线段形成后，只要**五均线向上穿越170均线**，即可看多 |
| 简化原因 | 独立本身已具备背驰或结构终结的条件，转折更直接 |
| 区别 | **不需要二次确认**，比走势中完美看多更简洁 |

### 案例

- **北部股份**：独立转折看多
- **东华科技**：独立转折看多

## 六、重要辨析

### 6.1 看多 vs 做多

| | 看多 | 做多 |
|------|------|------|
| 定义 | 下跌结构完美，不跌了 | 出现具体买点信号 |
| 操作 | 开始跟踪、养股 | 建仓买入 |
| 信号 | 四条件满足 | 进攻买点 / 防守买点 |
| 关系 | 看多是做多的**前提** | 做多是看多的**后续** |

```
错误做法：看到看多就立即买入
    → 可能只是震荡，没有买点
    → 正确做法：看多成立后，等待买点信号再建仓
```

### 6.2 走势中完美看多 ≠ 传统缠论的"一买二买三买"

| 传统缠论 | 本课看多结构 |
|----------|-------------|
| 一买二买三买缺乏精确量化 | 通过"170均线抬头 + 四条件"精确量化 |
| 实战中容易失败 | 规避形态学上的盲目抄底 |
| 基于背驰/形态 | 基于均线关系 + 中枢上轨的量化 |

### 6.3 170均线是"针"，本质线段是"线"

```
结构 = 本质线段（针） + 170均线（线） + 中枢、级别、走势分类（穿针引线）

不能只看均线 → 必须结合中枢和级别
不能只看中枢 → 必须借助均线量化

两者缺一不可。
```

## 七、实战应用要点

### 7.1 选股中的看多

| 对象 | 推荐方法 |
|------|----------|
| 80%-90%的中小资金投资者（非职业、非短线） | 重点使用**走势中完美看多**和**独立转折看多**进行选股和底部布局 |
| 做法 | 看多结构出现后 → 开始养股 → 等待买点 |

### 7.2 短线中的看多

| 对象 | 推荐方法 |
|------|----------|
| 短线交易者 | 使用**走势分类看多**（上涨趋势中的回踩买点，如进攻买点、三买） |

### 7.3 抄底的核心原则

```
不要在"没有二次确认"的地方抄底！

只有满足四条件时，才具备不下跌的条件：
    ✅ 价格在中枢上方
    ✅ 34均线在中枢上方
    ✅ 五均线在34和170上方
```

### 操作流程图

```
步骤1：确认34均线曾向上穿越170均线（停顿）
    │
步骤2：等待回踩后的二次向上
    │
步骤3：检查四条件是否满足
    │ 价格、34线、5线均在转折中枢上方
    │ 5线在170上方
    │
步骤4：条件满足 → 看多成立 → 开始跟踪
    │
步骤5：等待买点信号 → 做多建仓
```

## 八、大盘与市场分析（讲课当时）

### 大盘状态

| 维度 | 判断 |
|------|------|
| 级别 | 30分钟级别下跌过程中的宽幅震荡 |
| 企稳确认 | 未确认下跌结束 |
| 护盘力量 | 券商护盘但无持续性 |

### 市场特征

| 板块/题材 | 状态 |
|-----------|------|
| 小盘股 | 活跃 |
| 环保工程（水务、固废） | 出现新的资金一致性，可能是短期主线 |
| 其他题材 | 多为老题材延续或叠加，无新的核心驱动 |

### 操作建议

| 对象 | 建议 |
|------|------|
| 新学员 | 不急于开仓，利用震荡过程选股，关注有看多结构的个股 |
| 老学员 | 可小仓位实践短线 |
| 所有人 | 不要被"全面反攻"的观点误导，目前不具备全面上涨条件 |
