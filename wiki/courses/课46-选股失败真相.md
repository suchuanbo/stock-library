# 课46：三阶段讲一 — 选股失败的真相

## 一句话摘要

**选股失败的根本原因在于关注了80%无关紧要的因素，而忽略了20%的核心——资金体系认知与核心驱动力量识别。** 本课作为三阶段第一讲，系统讲解市场资金体系的"江湖效应"、核心驱动的判断标准，并建立一套可执行的每日复盘选股流程：鱼池（涨停海选）→ 龙门（看多结构筛选）→ 化龙（买点跟踪建仓）。

### 1.1 三阶段学习关键

- **三层楼比喻**：一、二阶段（1和2）是底层基础（看涨结构与买点本质），没有底层就无法支撑第三层（应用与认知）
- **100步理论**：前99步是积累，最后一步水到渠成。三阶段是"做减法"——从繁杂因素中提取20%的核心因素（影响90%以上结果），摒弃80%无关紧要的因素
- **独立思考**：学而不会的原因有二——①基础没打牢；②缺乏实战认知和"血肉灵魂感"

## 二、买点本质回顾（承上课45）

> 买点 = 大级别看涨 + 小级别下跌结束 + 安全边际 + 核心驱动

| 要素 | 说明 |
|------|------|
| 结构优先 | 先有看多结构（走势转折），然后才有买点（进一/进二/进三/防守类） |
| 安全边际 | 不同买点服务于实战的有效性和安全边际 |
| 战场比喻 | 选股如打仗——攻击最有效部位（结构转折），前提是"能接触到敌人"（有工具/武器） |

## 三、选股失败的根本原因

### 3.1 选股的目的

> 找到上涨的个股，且是**最短时间、最快速、最有力**的上涨。

### 3.2 多数人失败的原因

| 错误做法 | 正确做法 |
|----------|----------|
| 关注80%无关紧要的因素（杂音、消息、指标） | 锁定20%的核心因素 |
| 依赖单一资金指标（主力净流入） | 看资金的**合力** |
| 寄望从头吃到尾（如片仔癀/茅台长牛） | 做30分钟级别牛熊轮回，波段复利 |

### 3.3 股市上涨的核心因素

> 最终落脚点是**资金**——价格背后是资金流入。但单一资金不可靠，需要看**资金的合力**。

## 四、市场资金体系与"江湖效应"（核心认知）

### 4.1 资金分类

| 资金类型 | 特征 |
|----------|------|
| 公募 | 趋势型、稳健 |
| 私募 | 灵活、追求绝对收益 |
| 险资（含社保） | 长期、低风险偏好 |
| 券商自营 | 灵活、信息优势 |
| 境外资金 | 价值导向、大波段 |
| 游资 | 短平快、连板模式 |
| 散户 | 分散、跟随 |
| 国家队（汇金等） | 政策引导、维稳 |

> **超大资金**：不是某一类资金，而是**管理者**（政策引导）带动多类资金**协同作战**的体系。但超大资金也无法逆势（如2015年杠杆泡沫破灭）。

### 4.2 资金的"天然江湖效应"

| 效应 | 说明 | 实战启示 |
|------|------|----------|
| **固有派系** | 每个板块/行业存在固有资金派系，有资金龙头（趋势龙头，非基本面龙头） | 选股前先识别板块资金龙头 |
| **等级地位** | 派系之间有地位等级，上层不希望下层快速壮大 | 板块轮动是正常现象 |
| **绞杀效应** | 尤其针对游资——连板模式在"价值投资/趋势维稳"环境下越来越难生存 | 不追游资连板股 |
| **地盘维护** | 每个派系要维护自身地盘、保持造血功能 | 板块轮动 = 正常生态 |

### 4.3 对散户的启示

- 散户无法主导市场，只能**跟随大资金喝汤**（人家吃肉，你喝汤）
- 不要幻想从头拿到尾——那是极小概率的"中奖"事件
- **必须学会空仓**：熊市/大级别卖点出现时不要逆势
- **30分钟级别的牛熊轮回**才是散户主要操作周期
- 宁愿卖飞，也不要不舍得卖——**机会永远存在**

## 五、核心驱动力量（选股的关键）

### 5.1 定义

> **核心驱动 = 资金的趋同（协同）**——即不同派系的大资金同时做多某一板块/个股。

### 5.2 外部驱动 vs 内部驱动

| 类型 | 定义 | 实战判断 |
|------|------|----------|
| **外部驱动** | 政策、行业消息等导火索 | 反映在涨停分析、热点事件中——**主流驱动** |
| **内部驱动** | 板块内部资金自己的动作（如下跌后的回抽） | 没有外部协同，难成主流——**辅助参考** |

### 5.3 资金趋同（协同）的识别条件（核心细则）

| 条件 | 量化参考 | 说明 |
|------|----------|------|
| ① 板块内多个涨停 | 板块成分股越多，涨停数越多越好，至少**2只以上** | 资金一致性核心信号 |
| ② 资金趋势龙头涨停/大涨 | 龙头也涨停，或至少大涨 | **最强信号**——老大表态 |
| ③ 特大单/大单主导 | 涨停由特大单/大单主导（非小单、中单） | 说明大资金抢筹，怕拿不到筹码 |
| ④ 底部构造结构个股上涨 | 存在底部构造结构的个股也在涨 | "老大涨，小弟也涨"的协同确认 |

### 5.4 什么是"资金龙头"

> 在每个行业板块中，一眼能看出涨幅最大、趋势最强的个股，就是**资金趋势龙头**（不是基本面龙头）。

- 用 **20日、60日涨幅排序** 快速识别
- 资金龙头是板块的"江湖老大"——它表态，板块才有方向

## 六、选股的具体操作流程（每日复盘工作）

### 6.1 三个递进股票池："鱼池 → 龙门 → 化龙"

```
涨停海选（鱼池）
    ↓ 筛选有核心驱动的个股
核心驱动池
    ↓ 筛选已形成看多结构（转折结构）
看多池（龙门）
    ↓ 跟踪安全边际，等待买点
买点跟踪池 → 自选股重点跟踪（化龙）
```

### 6.2 每日复盘涨停股的五条件筛选法

| 步骤 | 条件 | 操作 |
|------|------|------|
| ① | **涨停大阳线** | 涨停（或10%以上）大阳线，且正在构造转折结构（即使尚未完成转折也要纳入跟踪） |
| ② | **板块资金一致性** | 检查该涨停股所属行业板块，是否满足多个涨停？ |
| ③ | **龙头表态** | 该板块的资金趋势龙头是否也涨停或大涨？ |
| ④ | **大单验证** | 涨停是否由特大单/大单主导（软件可看）？ |
| ⑤ | **外部驱动** | 是否有外部事件驱动（涨停分析/热点题材）？不需要自己深入挖掘，软件已反映 |

### 6.3 选股不是为明天立即买入

| 阶段 | 时间周期 | 说明 |
|------|----------|------|
| 核心驱动到买点出现 | 通常 **3~5天**（运气好）到数周甚至1~2个月 | 要**养股**，提前跟踪 |
| 散户优势 | **提前发现**（通过涨停复盘抢在大多数人之前） | 不是追高 |

### 6.4 关于"进三"买点

| 场景 | 条件 |
|------|------|
| 进三定义 | 34均线上方的买点，属于**强者恒强** |
| 适用环境 | 震荡市或牛市 |
| 不适用环境 | 下跌市中进三失败概率高（大盘环境差） |
| 下跌后做进三 | 需30分钟转折结构 + 本身已有核心驱动 |

## 七、辅助因素：公司基本面

| 维度 | 态度 | 具体做法 |
|------|------|----------|
| 公司经营/老板/调研报告 | **不需要深入研究** | 散户无信息优势 |
| 净利润 | **必看** | 同比正增长，数字越漂亮越好 |
| 现金流、成长能力 | **简单参考** | 辅助优选用 |
| 基本面定位 | **辅助优选，不是核心驱动力** | 核心驱动在资金不在基本面 |

## 八、大盘与仓位控制

| 维度 | 判断 |
|------|------|
| 当前环境（课程录制时） | 30分钟级别震荡，防止出现30分钟卖点（三卖） |
| 风险情景 | 若出现30分钟三卖，可能跌到3300点构造周线/月线级别的进三买点 |
| 仓位控制 | 震荡市不超过**6成**，技术不好则**4成**，以防守为主 |

---

## 九、职业交易者的工作心态

| 原则 | 说明 |
|------|------|
| **选股是每天必做的工作** | 就像生意需要持续经营，不能等卖完股票再重新选股 |
| **积累是核心** | 每天复盘100多只涨停股（实际经常不到100只），是很轻松的工作量 |
| **行动派非空想家** | 职业玩家是行动派 |
| **认知 > 技术** | 改变认知比学习技术更重要——认知决定执行（该卖时果断卖，不纠结后面是否涨） |

## 十、常见亏损模式与解决方向

| 亏损模式 | 根本原因 | 解决方向 |
|----------|----------|----------|
| 追涨停板被套 | 无核心驱动判断，追的是游资连板 | 按五条件筛选，只做有资金一致性的涨停 |
| 选股后不涨 | 无看多结构，无核心驱动 | 先有结构后有买点，无核心驱动不跟踪 |
| 买到就跌 | 无安全边际，追高建仓 | 等回撤 + 安全边际附近建仓 |
| 频繁换股 | 无养股过程，期望买入即涨 | 建立股票池，跟踪3天~2个月 |

## 金句提炼

> - 选股如打仗，你不知道敌人在哪、敌人是谁，你怎么打？
> - 股市上涨最终看资金，但单一资金不可靠，要看资金的合力。
> - 每个板块都有江湖老大——找到它，等它表态。
> - 核心驱动 = 资金的趋同：多个涨停 + 龙头涨停 + 大单主导。
> - 散户的优势是提前发现，不是追高。
> - 选股不是为明天买入，而是养股——从涨停海选到买点建仓，是一个跟踪的过程。
> - 改变认知比学习技术更重要：认知决定你该卖时能不能果断卖。
