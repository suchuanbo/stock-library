# 课44：二阶段（下）讲十 — 职业买卖进三（一）

## 一句话摘要

**进三买点是30分钟级别上涨的加速买点（三买），它不是独立存在的正儿八经买点，而是偏操作类、滚动类的买点。** 核心应用有两种：初始选股（看多结构与买点同时出现，需170背驰+核心驱动）和卖点后的滚动补仓（必须前期有初始买点）。本课为进三第一部分，重点讲授进三的结构定义、量化条件和交易理念。

## 二、进三买点的核心定义

### 2.1 本质

> **进三 = 30分钟级别上涨过程的加速上涨点 = 30分钟级别的三买点。**

它是上涨中继的加速起爆点，依托于30分钟级别中枢的突破回踩确认。

### 2.2 结构要求（以日线图判断）

| 条件 | 说明 |
|------|------|
| ① 30分钟中枢 | 必须有肉眼可见的30分钟级别中枢（由五均线震荡形成） |
| ② 中枢位置 | 该中枢的上轨必须位于日线级别中枢上轨的上方（或与170均线有明确关联） |
| ③ 回撤方式 | **回撤是五均线回撤**（不是K线回撤），五均线回撤后在中枢上轨附近形成买点 |
| ④ 均线排列 | 五均线必须在34均线上方，34均线通常在170均线上方（趋势向上） |

### 2.3 选股前提（初始选股场景）

进三作为初始选股买点时，必须同时满足：

| 前提条件 | 说明 |
|----------|------|
| 170均线背驰 | 必须有盘整背驰或趋势背驰，否则放弃 |
| 核心驱动力量 | 必须有涨停/大阳线，体现资金一致性 |
| 结构同步 | 看多结构与买点同步出现 |

---

## 三、进三的两种应用场景

### 3.1 场景对比

| 场景 | 初始选股 | 滚动补仓 |
|------|----------|----------|
| 使用时机 | 第一次建仓 | 卖点出现后回补 |
| 前提条件 | 170背驰 + 核心驱动 + 看多结构同步 | **必须有前期初始买点**（没有初始买点的个股，不会出现滚动补仓的进三） |
| 独立程度 | 可独立作为首次介入点 | **不可独立**，是对已有持仓的补充 |

> **核心铁律**：没有初始买点的个股，后面不会出现滚动补仓的进三。进三不是独立存在的买点，而是对已有持仓的补充。

---

## 四、进三与其他买点的区别

| 买点类型 | 级别 | 回撤方式 | 安全边际依托 | 独立性 |
|----------|------|----------|-------------|--------|
| **进三** | 30分钟级别三买（加速上涨点） | **五均线回撤**（非K线回撤） | 30分钟中枢上轨 | 非独立，偏操作类 |
| 进一 | 30分钟级别起涨/加速点 | 34均线附近回撤 | 中枢上轨或34均线 | 可独立（标准）/ 半独立（扩展） |
| 进二 | 日线级别起涨点 | 日线回撤 | 做多中枢（170附近） | 可独立 |
| 防守类 | 大级别底部 | 日线及以上回撤 | 170托中枢 / 中枢托170 | 可独立 |

**进三的独特之处**：必须依托**肉眼可见的30分钟中枢**，回撤是**五均线而非K线**，这是进三区别于所有其他买点的核心标志。

---

## 五、进三的结构量化条件（初始选股完整清单）

用于初始选股时，进三必须满足以下全部条件：

| 序号 | 条件 | 性质 |
|------|------|------|
| 1 | 日线级别中枢（转折或分类）在170均线上方 | 必须 |
| 2 | 30分钟级别中枢位于日线中枢上轨上方（且为肉眼可见） | 必须 |
| 3 | 五均线回撤至30分钟中枢上轨附近 | 必须 |
| 4 | 五均线在34均线上方，34均线通常在170均线上方 | 必须 |
| 5 | **170均线有背驰**（盘整背驰或趋势背驰）— 否则放弃 | 必须 |
| 6 | 具备核心驱动力量（涨停/大阳线 + 资金一致性） | 必须 |

> **⚠️ 第5条是硬性过滤条件**：若170均线无背驰，即使前四条形态符合，也不建议做进三（宁可错过）。

---

## 六、背驰在进三中的应用

### 6.1 背驰的重要性

> **170均线的背驰是进三初始选股的重要前提，是硬性门槛。**

### 6.2 背驰类型

| 类型 | 定义 | 说明 |
|------|------|------|
| 盘整背驰 | 一段下跌中只有一个中枢 | 一中枢下跌后背驰 |
| 趋势背驰 | 一段下跌中有两个及以上中枢 | 两中枢以上下跌后背驰 |

### 6.3 操作原则

- **有背驰** → 形态符合即可考虑进三初始选股
- **无背驰** → 即使形态符合，也必须放弃（宁可错过，不做错）

## 七、交易理念与心态（本课重点引导）

### 7.1 重结构，轻信号

| 理念 | 说明 |
|------|------|
| 信号是死的 | 金叉、突破等信号是固定规则，无法应对市场多变 |
| 市场是活的 | 市场走势是动态博弈，需要活的判断 |
| 结构的破坏 | **结构的破坏才是真正的离场信号**，不要依赖单一信号决策 |

### 7.2 持仓线（控盘线）的重要性

| 要点 | 说明 |
|------|------|
| 自问三问 | 你的持仓周期是什么级别的？什么情况下结构破坏？ |
| 看长做短 | 以170均线判断日线趋势（大方向），以34均线操作30分钟级别（实际买卖） |
| 结构破坏即离场 | 一旦上涨结构被破坏（如跌破关键均线或中枢），应立即减仓或离场，**不要死扛** |

### 7.3 止损 ≠ 出局

| 概念 | 说明 |
|------|------|
| 止损 | 是风控手段，保护资金安全 |
| 出局 | 是彻底放弃，清仓且不再跟踪 |
| 区别 | 若整体结构仍在（如日线上涨趋势未破），止损后可**留100股跟踪**，等待新的买点（如进三）接回 |
| 完全出局 | 清仓且不再跟踪，才是真正的放弃 |

### 7.4 卖点之后的心态

| 心态 | 说明 |
|------|------|
| 卖飞是常态 | 不要纠结于卖飞，这是交易的一部分 |
| 重新接回 | 卖点出现后，若后续出现进三，可重新接回；若没有，则换下一只个股 |
| 循环滚动 | 中小资金的节奏是**一买一卖，循环滚动**，不是死守一只股 |

### 7.5 什么是「活的市场」？

| 维度 | 说明 |
|------|------|
| 能力要求 | 你能肉眼排除无效波动，看到真正的趋势启动点（如170均线拐头 + 背驰 + 结构） |
| 工具与理解 | 工具（均线、线段）是死的，但你对结构的理解必须是活的 |
| 核心能力 | 能够区分「真抬头」和「假反弹」— 这是职业与非职业的分水岭 |

## 总结金句

> **「进三不是独立买点，它要么是初始选股的加速点（需170背驰），要么是卖点后的回补点（需初始买点）。重结构、轻信号，知道你的持仓线在哪里，你会卖才是师傅。」**

## 结构速查清单

```
进三初始选股检查清单：
□ 日线中枢在170上方？
□ 30分钟中枢在日线中枢上轨上方（肉眼可见）？
□ 五均线回撤到30分钟中枢上轨附近？
□ 五均线 > 34均线 > 170均线（趋势向上）？
□ 170均线有背驰（盘整/趋势）？
□ 有核心驱动力量（涨停/大阳 + 资金一致性）？
→ 全部打勾，进三初始选股成立
→ 任何一项不满足，放弃或等待
```
