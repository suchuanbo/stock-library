# 课3：线段中枢之起手三式

## 一句话摘要

线段中枢只能由**反正两穿、三笔、5k重叠**三种方式之一构成（无第四种）。本课详解每种方式的定义、条件与核心法则，重点掌握「先反后正」的反正两穿逻辑、方向与穿的概念、跳空处理规则，以及黑K在后续课程中的必要性。

---

## 课程定位

本课属于**一阶段（基础知识点）**第三讲，承接课2的「中枢线与k0/k体系」，在已明确中枢线定义后，讲解如何判断线段中枢是否成立——即「起手三式」三种构成方式。本课是一阶段后续内容（线段中枢形成确认、上下轨取值、唯一选取等）的直接前置。

**课程体系位置**：

```
课1（资本入门礼）
  → 课2（中枢线与k0/k体系）
    → ★课3（起手三式）← 当前

        → 后续（线段顶底四法则 ... 走势分类 ... 买卖点）
```

## 起手三式逐一详解

### 前提：中枢线（复习）

- 中枢线是线段中枢的基础，由 **k0 与 k 配对**形成（实体K线与五均线的关系）
- 跳空情况下的中枢线确认：需满足**双跳空**（价格跳空 + 实体跳空）
- 所有起手三式的判断必须围绕中枢线展开

---

### 一、反正两穿（最复杂）

#### 定义

「反正两穿」是通过实体价格对中枢线的两次穿越（先反方向、后正方向）来形成线段中枢。它是最复杂的一种方式，也是在实际走势中最常见的方式。

#### 前提概念：方向

| 概念 | 定义 |
|------|------|
| **方向（上/下）** | 实体价格持续新低为「下」，持续新高为「上」 |
| **正方向** | 与线段方向相同的方向。向上线段的正方向 = 向上；向下线段的正方向 = 向下 |
| **反方向** | 与线段方向相反的方向。向上线段的反方向 = 向下 |

> ⚠️ **方向只看实体最值**，不看K线的最高价/最低价（不看影线）。

#### 核心法则

```
先反后正：必须先有反穿，才能有正穿。
          不下才有上，不上才有下。

无反便无正：如果连反穿都没有，说明级别未升级，
            不可能有正穿，线段中枢也不成立。
```

**执行顺序**：
1. 确定线段方向（向上线段 / 向下线段）
2. 沿「正方向」运行，直到方向终结
3. 出现「反穿」（实体向反方向突破中枢线）
4. 反方向运行终结后，出现「正穿」（实体向正方向突破中枢线）
5. 正穿完成后，线段中枢成立

#### 「穿」的定义

| 类型 | 向上线段中 | 向下线段中 |
|------|-----------|-----------|
| **反穿** | 实体**跌破**中枢线 | 实体**上穿**中枢线 |
| **正穿** | 实体**上穿**中枢线 | 实体**跌破**中枢线 |

> 「穿」的动作必须在**方向确定之后**才有意义。即必须先有方向，再看穿。

#### 方向终结

- **向下方向终结**：不再出现实体持续新低
- **向上方向终结**：不再出现实体持续新高

方向终结是「穿」的前置条件——方向在延续过程中不存在「穿」的问题，只有方向终结后才能开始看反方向。

#### 关键要点

- k0 与 k 必须**连续**（无小三K线插入），否则中枢线刷新
- 方向只看**实体最值**，不看最高最低价（影线不算）
- 反穿和正穿之间可能间隔多根K线，但只要反方向未终结、正穿未出现，仍可等待

#### 图示（文字描述）

```
【向上线段示例】

    ↗️ 正方向持续新高 → 方向延续
    ↗️ (k0) 正方向终结 → 不再新高
    ↘️ 实体跌破中枢线 → ★反穿完成
    ↘️↘️ 反方向运行 → 等待...
    ↘️ 不再新低 → 反方向终结
    ↗️ 实体上穿中枢线 → ★正穿完成 → 中枢成立
```

---

### 二、三笔（最简单）

#### 定义

三笔成线段中枢：**3根或以上K线**，价格（最高价/最低价）持续新高或持续新低，形成一笔回撤。

#### 特点

- **与方向无关**：只看价格持续创新高/新低，不关心正方向还是反方向
- 最标准、最简单，但实际交易中出现**较少**
- 三笔的形成必须围绕**中枢线**确定 k

#### 条件

1. k0 已确认（中枢线已确定）
2. 至少有 3 根 K 线，其价格沿同一方向持续运动
   - 持续新高：每根K线最高价 > 前一根K线最高价
   - 持续新低：每根K线最低价 < 前一根K线最低价
3. 这三根K线形成的「一笔」构成中枢的一次回撤

#### 跳空情况

价格跳空可视为**中间插入K线**，仍可构成三笔（跳空不破坏三笔的结构连续性）。

```
示例：K1(10元) → 跳空高开 → K2(11元)
     跳空本身被视为「价格持续新高」的延续
     三笔计数中，跳空 = 一根虚拟K线
```

#### ⚠️ 重要限制

仅有三笔而无「**黑K**」则线段中枢**不成立**。黑K是线段中枢成立的另一个必要条件（将在课4详细讲解）。

---

### 三、5k重叠（中等难度）

#### 基本概念

| 概念 | 定义 |
|------|------|
| **首k** | 第一根出现价格破坏（不再持续新高/新低）的K线，且必须与中枢线相关 |
| **破坏k** | 首k之后第一根反向突破的K线 |
| **重叠区域** | 首k与破坏k之间的价格区间（高点取低者，低点取高者） |

#### 成立条件

在重叠区域内，有 **5根或以上K线**（含首k和破坏k）→ 符合5k重叠。

- 若不足 5 根，则**不成立**
- 同价也算重叠（即K线价格触及重叠区域边界即算在内）

#### 与基础课程的区别

| 维度 | 基础课程 | 高阶课程（本课） |
|------|---------|----------------|
| 级别保障 | 依赖「13k」保障级别 | 用五均线工具替代，无需13k |
| 5k重叠判断 | 硬刻板（严格机械） | 结合起手三式更灵活 |

#### 跳空处理（5k重叠特有）

| 跳空类型 | 处理方式 |
|----------|---------|
| 重叠区域**内部**的跳空 | 无需额外处理，归类于反正两穿或三笔 |
| **离开**重叠区域的跳空（第5根K线跳空） | 可理解为添加一根K线，仍算5k重叠 |

---

## 三种方式对比

| 维度 | 反正两穿 | 三笔 | 5k重叠 |
|------|---------|------|--------|
| **难度** | 最复杂 | 最简单 | 中等 |
| **核心逻辑** | 先反后正，两次穿越中枢线 | 3根K线持续新高/新低 | 5根K线在重叠区域内 |
| **与方向关系** | 严格依赖正/反方向 | 与方向无关 | 间接相关（通过首k/破坏k） |
| **出现频率** | 最常见 | 较少 | 常见 |
| **依赖中枢线** | 是（穿的定义基于中枢线） | 是（围绕中枢线确定k） | 是（首k必须与中枢线相关） |

### 谁最先完成

在实际走势中，三种方式可能同时存在或交叉出现。此时需判断**谁最先满足条件**——最先完成的方式即为本次中枢的构成方式。

> 意义：动态分析中需实时判断当下线段中枢是否已成立，利于及时做出交易决策。

---

## 方向与穿的概念

### 方向定义（深度学习）

缠论中「方向」不是常识意义上的涨跌，而是严格按**实体价格序列**定义的：

- **向上方向**：当前K线实体最高价 > 前一根K线实体最高价（持续状态）
- **向下方向**：当前K线实体最低价 < 前一根K线实体最低价（持续状态）
- **方向延续**：只要方向条件持续满足，方向即在延续中
- **方向终结**：方向条件不再满足的那一刻，该方向终结

### 穿的前提

「穿」必须同时满足：
1. **方向已终结**——前一方向不再延续
2. **实体价格跨越中枢线**——实体（非影线）突破中枢线价位
3. **穿的方向明确**——反穿是向反方向跨越，正穿是向正方向跨越

### 正方向 vs 线段方向

| 线段方向 | 正方向 | 反方向 |
|---------|--------|--------|
| 向上线段 | 向上（持续新高） | 向下（持续新低） |
| 向下线段 | 向下（持续新低） | 向上（持续新高） |

> 正方向永远与线段方向一致。

---

## 跳空处理（综合总结）

跳空是本课程体系中的重要特殊情况，在起手三式中需统一处理：

| 场景 | 规则 |
|------|------|
| **中枢线确认**（课前复习） | 双跳空（价格跳空 + 实体跳空）才能确认中枢线 |
| **三笔中的跳空** | 价格跳空可视为中间插入K线，仍可构成三笔 |
| **5k重叠内部的跳空** | 无需额外处理，归类于反正两穿或三笔 |
| **5k重叠外部的跳空** | 第5根K线跳空离开重叠区域 → 可理解为添加一根K线 |
| **反正两穿中的跳空** | 跳空本身不改变方向判断（只看实体最值），但可能影响k0/k的连续性 |

---

## 常见错误

| 错误 | 正确认知 |
|------|---------|
| ❌ 把影线突破当作「穿」 | ✅ 只看**实体**（开盘价↔收盘价），影线不算 |
| ❌ 看到价格反向就认为反穿 | ✅ 必须先有**方向终结**，才能谈「穿」 |
| ❌ 正穿可以在反穿之前 | ✅ **先反后正**是铁律，无反便无正 |
| ❌ 三笔不需要中枢线 | ✅ 三笔也必须围绕中枢线确定k |
| ❌ 4根K线在重叠区就算5k重叠 | ✅ 必须**≥5根**（含首k和破坏k） |
| ❌ 满足起手三式=线段中枢成立 | ✅ 还必须满足**黑K**条件（课4讲解） |
| ❌ k0和k可以间隔小三K线 | ✅ k0和k必须**连续**，否则中枢线刷新 |

## 总结

### 本课核心收获

1. **起手三式 = 线段中枢的唯一三种构成方式**：反正两穿、三笔、5k重叠，无第四种
2. **反正两穿的核心**：「先反后正，无反便无正」，方向只看实体最值
3. **三笔的核心**：3根K线持续新高/新低，最简单但出现最少
4. **5k重叠的核心**：首k与破坏k之间 ≥ 5根K线
5. **黑K是必要条件**：即使满足起手三式，无黑K则线段中枢不成立（课4详解）
6. **方向是前提**：先有方向终结，才能有「穿」；先反后正不可颠倒
7. **技术只是工具**：底层「非黑即白」的法则和正确的市场认知（大户对立视角）才是持续盈利的关键
