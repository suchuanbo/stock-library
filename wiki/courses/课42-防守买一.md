# 课42：二阶段（下）讲八 — 职业交易之防守买一

## 一句话摘要

**防守买一（防一）是30分钟级别下跌终结类买点（类三买），属于买点体系中有效性最高的买点类型。** 核心创新是「防守区域」的量化定义——通过中枢上轨与前一构造线段的五均线峰值构成的区间，配合170均线和34均线的位置关系，精确判断回撤有效性。防守买一可遇不可求，但一旦成立持仓周期长、盈亏比高。

---

## 一、课程定位与核心目标

| 维度 | 说明 |
|------|------|
| 所属阶段 | 第二阶段（下）第八课，防守买点体系第一课 |
| 前置知识 | 课40-41：进攻买一（标准+扩展）；课38-39：线段终结（小级别回撤判断）；课35：买点本质（三要素） |
| 后续课程 | 课43：防守买二；课44：进三（滚动买点）；三阶段课程 |
| 核心目标 | 理解防守买一的本质（30分钟下跌终结 → 30分钟上涨）；掌握防守区域的量化定义与两个量化条件；建立「重结构、轻信号」的交易认知 |

### 课程背景

- **买点体系完善**：继进二（日线起涨）、进一（30分钟中继加速）后，引入第三种买点——防守买一（30分钟下跌终结）
- **核心观念**：90%亏损来源于不正确的止损、频繁换股、不会处理持仓；防守买点是承接止损后重新介入的关键工具
- **适用人群**：职业交易者需掌握；非职业者可了解，重点是理解「买点只有两类」——下跌终结和上涨中继

## 二、买点的本质（课程反复强调）

### 2.1 买点只有两类

| 类型 | 代表买点 | 本质 |
|------|----------|------|
| **下跌终结买点** | 防守买点（防一、防二）、类一买/类三买 | 一段下跌结束，新的上涨开始 |
| **上涨中继买点** | 进一、进二、进三 | 上涨趋势中的加速或加仓点 |

> **核心认知**：任何买点研究的目的都是无限接近下跌的终结或上涨的中继，提升成功概率，而非寻找必胜点。

### 2.2 中小资金的固定节奏

| 维度 | 说明 |
|------|------|
| 操作级别 | 所有买点都是**30分钟级别**（30分钟下跌结束 → 30分钟上涨） |
| 结构背景 | 日线级别结构提供趋势背景（看多/看空），但操作级别固定为30分钟 |
| 节奏 | 做一段30分钟上涨，结束后全仓卖出，等待下一个买点 |

---

## 三、买点之间的关联关系

防守买一不是孤立存在的，它在买点体系中承担「承接」角色：

| 买点 | 级别 | 特点 | 相互承接关系 |
|------|------|------|-------------|
| **进二** | 日线级别起涨点（相对最低位） | 一旦被打破很难承接，需严格风控 | 被进一/防守买点承接 |
| **进一** | 30分钟级别上涨中继（加速点） | 出现频率最高，适合大多数情况 | 可承接防守买点，也可被防守买点承接 |
| **防守买一** | 30分钟级别下跌终结（类三买） | 可遇不可求，有效性高 | 承接进一止损后的重新介入 |
| **进三** | 滚动买点 | 加仓或清仓后回补 | 承接所有买点后的加仓 |

> **关键理念**：买点之间在交易模式中相互承接。止损不等于出局——若结构仍在，可通过更大级别买点接回。

---

## 四、防守买一的核心定义

### 4.1 什么是防守买一

| 维度 | 说明 |
|------|------|
| **本质定义** | 30分钟级别（线段或走势）下跌的终结，带来30分钟级别上涨 |
| **类型归属** | 下跌终结类买点，属于「真三买」（有效三买） |
| **核心特征** | 可遇不可求，但有效性高、持仓周期长、盈亏比高 |
| **操作方式** | 只做背驰潜伏，不做突破买入 |

### 4.2 前提条件

| 条件 | 说明 |
|------|------|
| **日线级别看多结构** | 必须有独立转折看多 或 走势转折看多。**没有看多，就没有防守买点** |
| **做多中枢** | 通常是日线级别或周线级别的**转折中枢**（分类中枢较少使用） |
| **安全边际** | 通过「防守区域」量化（见第五节） |

---

## 五、防守区域（核心新概念，本课独创）

### 5.1 防守区域的定义

| 构成要素 | 说明 |
|----------|------|
| **上边界** | 构造中枢的**前一线段的五均线峰值** |
| **下边界** | **中枢上轨**（由线段转折形成） |
| **区间** | 上边界与下边界之间的价格区域 |

> **作用**：判断回撤是否有效。防守买点的回撤必须进入防守区域内部，否则不是有效防守买点。

### 5.2 防守买一的量化关系（两个条件同时满足）

| 条件 | 要求 | 含义 |
|------|------|------|
| **条件一** | 形成买点时，**170均线**必须在防守区域**内部** | 中枢托着170均线 |
| **条件二** | 回撤线段的**34均线**必须在防守区域**上方**（或相切），不能进入防守区域内部 | 170均线托着34均线 |

> **结构理解**：中枢托着170均线 → 170均线托着34均线 → 形成三层支撑结构。

### 5.3 防守区域的意义

| 要点 | 说明 |
|------|------|
| **回撤未进入防守区域** | 说明下跌不彻底，后续容易再跌或震荡，不是有效三买 |
| **防守区域过大** | 如周线级别防守区域过大时，量化意义减弱，需谨慎使用 |
| **核心价值** | 将抽象的「安全边际」转化为精确可量化的价格区间 |

## 六、防守买一的回撤要求

### 6.1 回撤的级别要求

| 要求 | 说明 |
|------|------|
| **级别** | 必须是「肉眼可见」的30分钟级别（线段或走势） |
| **无效回撤** | 非肉眼可见的小级别回撤不构成防守买一 |

### 6.2 回撤的形态要求

| 要求 | 说明 |
|------|------|
| **K线价格位置** | 回撤的所有K线价格（或五均线值）必须在中枢上轨**上方**（形态要求，非最核心） |
| **底3K位置** | 回撤的最低K线（底3k）必须进入防守区域**内部** |
| **背驰必须** | 必须有下跌力度衰减（背驰），否则不成买点。防守买点不做突破买入 |

### 6.3 防守买点买入方式

> **只做背驰潜伏，不做突破买入。** 防守买一的本质是下跌终结，必须在背驰后潜伏，而非追涨突破。

## 七、防守买一与其他买点的对比

| 维度 | 防守买一 | 进一（标准/扩展） | 进二 |
|------|----------|-------------------|------|
| **本质** | 下跌终结（30分钟级别） | 上涨中继（30分钟级别内部） | 日线级别起涨（最低位） |
| **出现频率** | 可遇不可求，频率低 | 频率高，适合大多数情况 | 频率中等 |
| **持仓周期** | 较长（30分钟级别一段上涨） | 中等（30分钟内部波段） | 最长（日线级别趋势） |
| **安全边际** | 防守区域（中枢上轨+五均线峰值） | 34均线（或中枢上轨） | 170均线与中枢上轨 |
| **看多前提** | 独立转折 或 走势转折 | 独立转折 或 走势转折 | 仅限独立转折 |
| **建仓方式** | 仅背驰潜伏 | 潜伏 或 突破均可 | 潜伏为主 |
| **适合择时** | 日线下跌末端或上涨中30分钟下跌末端 | 任何择时（温和） | 30分钟上涨末端（大盘恐慌时） |

---

## 八、防守买一的择时匹配

| 择时环境 | 适用买点 | 说明 |
|----------|----------|------|
| **日线级别上涨过程中的30分钟下跌末端** | 防守买一（最佳） | 顺势回调，安全性最高 |
| **日线级别下跌末端** | 防守买一 | 抄底型防守买点，需严格确认看多结构 |
| **30分钟盘整（无系统性风险）** | 进一、进三为主 | 防守买点出现概率较低 |
| **趋势明确上涨** | 进一为主，防守为辅 | 防守买点可遇不可求 |

> **当前大盘状态（文稿时间点）**：30分钟盘整，无系统性风险，防守买点出现概率较低，更应关注进一和进三。

## 九、选股与核心驱动力量（五步法）

满足以下五步的个股具备核心驱动力量，放入跟踪池：

| 步骤 | 内容 | 判断要点 |
|------|------|----------|
| **第一步** | 看涨停 | 涨停板是资金关注的信号 |
| **第二步** | 看是否构造买点结构 | 是否有看多结构 + 做多中枢 + 安全边际 |
| **第三步** | 看资金一致性 | 板块内多只个股同时出现结构和驱动逻辑 |
| **第四步** | 看外部驱动 | 政策利好、行业消息、题材催化 |
| **第五步** | 看大单资金 | 主力资金是否真正介入 |

## 十、核心理念总结（三阶段提前讲解）

### 10.1 重结构，轻信号

| 理念 | 说明 |
|------|------|
| **买点是结构，不是信号** | 结构不破，不轻易出局；信号是死的，市场是活的 |
| **过度依赖信号的危害** | 错过机会、频繁止损 |
| **正确做法** | 以结构为主线，信号为辅助确认 |

### 10.2 止损不等于出局

| 理念 | 说明 |
|------|------|
| **止损的本质** | 风控手段，不是完全放弃 |
| **结构仍在的处理** | 止损后若上涨趋势未破，可通过更大级别买点接回或做短差降低成本 |
| **实操建议** | 止损后留下100股跟踪，防止被洗后踏空 |

### 10.3 中小资金的节奏

| 理念 | 说明 |
|------|------|
| **操作级别** | 固定为30分钟，做一段30分钟上涨 |
| **结束操作** | 一段上涨结束后全仓卖出，等待下一个买点 |
| **升级方式** | 若想做更大级别（日线），通过进三等买点逐步承接，而非死扛 |

### 10.4 两类短差（滚动）

| 类型 | 操作 | 适用场景 |
|------|------|----------|
| **下跌短差** | 先卖后买 | 下跌过程中降低成本 |
| **上涨短差** | 先买后卖 | 中枢震荡中降低成本 |

> **重要性**：不会做短差，很难将成本降至零，也难以获得高利润率。

### 10.5 交易模式三要素

| 要素 | 含义 |
|------|------|
| **选股** | 概率（提高胜率） |
| **赔率** | 盈亏比（每笔交易的期望收益） |
| **仓位管理** | 资金配置（控制风险敞口） |

## 十二、常见误区与纠偏

| 误区 | 纠偏 |
|------|------|
| 把防守买一当作突破买入 | 防守买点只做背驰潜伏，**不做突破买入** |
| 回撤未进入防守区域就买入 | 回撤必须进入防守区域内部，否则下跌不彻底，不是有效买点 |
| 忽略日线看多前提 | 没有日线级别看多结构，就没有防守买点 |
| 防守区域过大仍然依赖量化 | 周线级别防守区域过大时量化意义减弱，需结合结构判断 |
| 止损后就完全放弃跟踪 | 止损≠出局，结构仍在应通过更大级别买点接回 |
| 把防守买一当作常规工具频繁使用 | 防守买一可遇不可求，频率低；日常以进一、进三为主 |
