# 课49：三阶段讲四 — 卖飞的真相

## 一句话摘要

**卖飞的真相是你混淆了卖点、做T和滚动的边界；做T的前提是结构还在，且上涨过程中先买后卖更稳健；不要刻意追求做T，好股票不需要做T；核心是建立自我标准，接受取舍，追求持续稳健的资金增长。**

## 二、什么是「卖飞」？——大多数人的理解

### 2.1 定义

| 维度 | 内容 |
|------|------|
| **定义** | 卖出后 **1-2天内** 股价快速大涨，且未能及时买回，感到后悔。**不是**指长期（数月）后的大涨。 |
| **本质** | 混淆了卖点、做T、滚动的边界，没有清晰的自我标准 |

### 2.2 常见场景

| 场景 | 典型表现 |
|------|----------|
| 日内卖飞 | 日内卖出后，当天即涨8%以上，次日连续涨停 → 「拍大腿」 |
| 犹豫卖飞 | 卖出后下跌几个点，犹豫未买回，随后突然拉涨停 → 确认卖飞 |

---

## 三、做T vs 滚动 vs 买卖点（核心区分）

> **关键结论：做T ≠ 滚动。做T是日内仓位不变的流水操作；滚动是清仓后重新介入。做T绝不能替代卖点。**

| 概念 | 定义 | 仓位变化 | 操作周期 | 目的 |
|------|------|----------|----------|------|
| **做T** | 日内 T+0，当天买卖，利用底仓完成低买高卖 | **仓位不变** | 日内（当天） | 降低成本、赚流水 |
| **滚动** | 多日 / 新买点出现后的再次介入（如进三买点） | 仓位变化（加仓/回补） | 多日 | 重新参与上涨 |
| **卖点** | 主动背驰卖点 或 被动结构破坏卖点 | 仓位减少（减仓/清仓） | 按级别 | 锁定利润 / 控制风险 |

### 三者的关系

```
卖点是「结束」→ 主动卖点（背驰）或被动卖点（结构破坏）
做T是「延续」→ 结构还在时，日内流水降成本
滚动是「重新开始」→ 卖点之后，新买点出现时再次介入
```

---

## 四、做T的前提条件：结构还在

| 条件 | 规则 |
|------|------|
| **可以做T** | 交易级别结构未被破坏（持仓线、成本线未破） |
| **不能做T** | 结构破坏（跌破持仓线 / 成本线）→ 只能被动卖出（清仓），**不能做T** |

### 举例

| 场景 | 做法 |
|------|------|
| 30分钟级别上涨结构完好，即使日内下跌 | 可以做T降低成本 |
| 30分钟结构已破 | 必须清仓，任何T都会导致更大亏损 |
| 单边下跌中强行做T | ❌ 错误做法——以为能无限降低成本，结果深套 |

---

## 五、真正的「好股票」不需要做T

| 维度 | 内容 |
|------|------|
| **好股票特征** | 连续涨停、单边上涨，根本不给你做T的机会（抢筹阶段） |
| **应对** | 买到这种股票，躺着持有即可 |
| **做T的意义** | 仅适用于**非总龙头**的普通上涨个股——主力通过洗盘、震荡来甩人，此时通过做T来降低成本、粘住主力 |
| **性质** | 做T是**锦上添花**，不是雪中送炭 |
| **关键认知** | 如果你选股能力强，可以完全不学做T，照样赚钱。做T只是让优秀者更优秀。不要刻意追求做T |

---

## 六、做T的方向策略（上涨 vs 下跌）

### 6.1 上涨过程中做T —— 推荐方式

| 维度 | 内容 |
|------|------|
| **方式** | **先买后卖**（早盘低开或下探时买入，盘中冲高时卖出） |
| **优势** | 不会丢失筹码（卖飞风险低）；即使做错，也只是成本略高，结构仍在 |
| **级别参考** | 看 **5分钟周期图** 判断方向（上涨趋势中），看 **1分钟周期图** 的背驰作为买卖点 |
| **止盈参考** | ① 一般赚 **3个点以上** 即可卖出 ② 出现1分钟背驰立即卖 ③ 若未出现背驰但冲高回落跌破3%，也卖 |

### 6.2 下跌过程中做T —— 高风险

| 维度 | 内容 |
|------|------|
| **方式** | **先卖后买**（早盘卖出，尾盘或盘中低点买回） |
| **风险** | 下跌过程做T难度大，容易「T飞」（卖后买不回）或「T无效」（连续阴跌无波动） |
| **适用场景** | 结构未破，但处于建仓初期的洗盘阶段——此时做T是为了降低持仓成本，承受该承受的回撤 |
| **注意** | 下跌过程中若出现 **5分钟级别背驰**，可做先卖后买；若没有背驰，不要强行做T |

### 6.3 绝对禁止的情况

| 禁止场景 | 原因 |
|----------|------|
| 主动卖点之后 | 结构已结束 |
| 被动卖点之后 | 结构已破坏 |
| 单边暴跌 | 任何T都无法挽救，只能止损 |

---

## 七、关于「卖飞」的深层认知

| # | 认知 |
|---|------|
| 1 | **卖飞是常态**：因为你不可能卖在最高点。只要按自己的标准（主动背驰卖点、被动破坏卖点）执行，卖飞也是正确的交易 |
| 2 | **不要试图优化到「不卖飞」**：市场会变，资金会变，追求不卖飞会陷入「自我怀疑 → 修改标准 → 再错」的恶性循环 |
| 3 | **卖出后股价继续涨**：那是**新的买点（滚动）机会**，不是卖点错误。你应该研究是否要滚动介入，而不是后悔卖早了 |

---

## 八、仓位建议：主动卖点后保留50%

### 8.1 操作模式

```
出现主动卖点（如30分钟背驰）→ 先卖出50%仓位，锁定利润
剩余50%仓位继续持有，用于滚动或做T

随后走势应对：
├─ 下跌但结构未破 → 做T降低成本
├─ 结构破坏 → 清仓剩余50%
└─ 出现新的买点（如进三）→ 可加回仓位
```

### 8.2 取舍逻辑

| 选项 | 优势 | 劣势 |
|------|------|------|
| 全仓卖出 | 锁定全部利润 | 可能错失后续上涨 |
| 全仓持有 | 享受全部涨幅 | 可能回吐利润 |
| **50%卖出 + 50%持有** | 防止卖飞（还有一半享受利润）+ 防止回撤（损失可控） | 非最优、非最差 —— 是平衡选择 |

> **没有完美方案**。50% 是平衡选择——进可攻退可守。

## 九、常见错误与纠正

| 错误 | 正确做法 |
|------|----------|
| 把做T当作主要盈利手段 | 做T是锦上添花，不是交易核心 |
| 结构破了还做T | 结构破坏 → 清仓，不是做T |
| 下跌中强行做T | 没有背驰不做T，单边下跌只止损 |
| 卖飞后追高买回 | 卖飞后等新买点（滚动），不要追高 |
| 追求「从不卖飞」 | 接受卖飞是交易常态，按标准执行即可 |
| 混淆做T和滚动 | 做T=日内仓位不变；滚动=清仓后重新介入 |
| 好股票也做T | 连续涨停的好股票躺着持有，不要做T |

## 十、心态与认知总结

| # | 核心认知 |
|---|----------|
| 1 | **交易的本质**：是应对活的市场，不是寻找一成不变的标准。你的自我标准（级别、持仓线、成本线、主动/被动卖点）是唯一的锚 |
| 2 | **持续稳健盈利 ≠ 单次暴利**：不要试图靠一只股票发家致富，要靠一整套交易模式（选股→买点→做T→主动卖点→滚动→被动卖点）长期积累 |
| 3 | **接受不完美**：卖飞、T错、止损都是交易的一部分。只要按标准执行，长期下来资金曲线正向，就是成功 |
| 4 | **孤独的修行**：市场会不断引导你偏离正确方向，你需要通过回顾笔记、视频，反复纠正自己的认知和行为 |

## 结构速查清单

```
做T检查清单：
□ 交易结构是否还在？（持仓线/成本线未破）
□ 做T方向：上涨中 → 先买后卖 | 下跌中 → 先卖后买
□ 级别参考：5分钟判断方向，1分钟背驰找买卖点
□ 止盈：3个点以上或1分钟背驰即走
□ 禁止：主动/被动卖点后不做T，单边暴跌不做T

卖飞应对清单：
□ 是否按自我标准（主动背驰/被动破坏）执行？
□ 卖出后继续涨 → 那是滚动机会，不是卖点错误
□ 仓位：主动卖点后保留50%防止卖飞
□ 接受卖飞是常态，不追求卖在最高点
```
