# 课37：二阶（下） — 职业交易之进攻买二扩展

## 一句话摘要

**进二扩展的本质是把日线中枢换成30分钟中枢，其余条件（170上方、五均线金叉、核心驱动）不变。** 当下跌过程没有日线级别独立转折、但存在30分钟级别「中完美」结构（小级别虚线反向带来中枢）且满足看多条件时，即形成进二扩展买点。关键是看懂「虚线」——那是小级别带来的反向，是30分钟中完美的灵魂。

## 二、进二扩展的核心定义

### 2.1 什么是进二扩展

> 当下跌过程**没有日线级别的独立转折**（无法画出日线级别上下上三段），但存在**30分钟级别的中完美中枢**，且该中枢满足看多条件，形成的买点即为进二扩展。

### 2.2 与标准进二的区别

| 维度 | 标准进二（课36） | 进二扩展（课37） |
|------|-----------------|-----------------|
| 转折基础 | 日线级别独立转折（有日线中枢） | 30分钟级别中完美（无日线级别独立转折） |
| 中枢级别 | 日线级别中枢 | 30分钟级别中枢 |
| 看多条件 | 170均线在日线中枢上方 + 五均线金叉170 | 170均线在30分钟中枢上方 + 五均线金叉170 |
| 核心驱动 | 需要涨停/大阳线 | 需要涨停/大阳线（同标准进二） |
| 本质 | 完全一致，仅中枢级别降维 | — |

---

## 三、30分钟级别中完美的两种类型

| 类型 | 说明 |
|------|------|
| **虚线本质** | 比30分钟更小的级别（5分钟/1分钟）带来的反向，无法形成30分钟本质线段，但能构成30分钟中枢 |
| **中完美判断** | 该中枢由虚线转折构成，即存在小级别反向。**虚线是30分钟中完美的灵魂** |

> 中小资金做30分钟级别中完美的进二扩展，往往对应快速上涨行情。

---

## 四、进二扩展的看多条件（量化）

**以下条件同时满足时，产生看多结构：**

| 条件 | 说明 |
|------|------|
| ① 五均线金叉170均线 | 时间当下的金叉信号 |
| ② 170均线在30分钟中枢上轨上方 | 可等于或稍高于上轨 |
| ③ 30分钟中枢由虚线转折构成 | 存在小级别反向（否则不是中完美） |

> ⚠️ 此看多条件与标准进二**完全一致**，只是把「日线中枢」替换为「30分钟中枢」。

---

## 五、做多区域（安全边际）的确定

### 5.1 做多中枢

| 项目 | 说明 |
|------|------|
| 定义 | 任何走势分类后得到的中枢（下跌中枢、上涨中枢、周线转折中枢等），只要其**上轨与170均线形成安全边际** |
| 选择时机 | 时间当下，股价回撤到该中枢上轨附近，且满足**线段终结信号**时，该中枢才成为有效做多区域 |

### 5.2 常见错误

> ❌ 误认为做多中枢必须等于独立转折中枢。
> ✅ 实际上可以是前方下跌中枢、周线转折中枢等任何分类中枢。

---

## 六、分组多（分、组、多）详解

> 本节课重点复习——分、组、多是「看多→做多」的核心框架。

### 6.1 分（分独立）

| 维度 | 说明 |
|------|------|
| 目的 | 得到**周线级别的转折中枢**（通过独立线的识别） |
| 方法 | 将走势按独立日线级别线段分开 |
| 独立四种 | 三买新高独立、三卖新低独立、周线升级独立、本身独立（次新股） |
| 关键 | 分不是为了分而分，是为了**找出周线中枢上轨**（用于后续做多可能） |

### 6.2 组（组走势）

| 维度 | 说明 |
|------|------|
| 目的 | 得到**日线级别的走势中枢**（用于判断趋势方向） |
| 方法 | 站在时间当下，将独立的日线线段组合成日线级别走势（上涨或下跌） |
| 要求 | 组合后的走势必须包含至少一个日线中枢 |

**组的方向规则：**

| 情况 | 操作 |
|------|------|
| 有向下的日线下跌走势（含中枢）+ 无向上看多信号 | → 组为**下跌未完成** |
| 有向上的日线走势中枢 | → 组为**上涨趋势** |
| 同时有向上和向下的日线中枢 | → 优先组为与**当前方向一致**的走势 |

### 6.3 多（看多）

**看多只有两种结构：**

| 类型 | 说明 |
|------|------|
| **走势转折看多** | 由日线级别下跌走势中枢形成的转折（三段式） |
| **独立转折看多** | 由独立日线级别线段形成的转折（需170在中枢上方） |

> 💡 30分钟级别的中完美也可以提供看多——即**进二扩展**的看多来源。

### 6.4 分组多综合示例

- **案例**：任何软件（润和软件）
  1. **分**：找出所有独立线段（如三卖新低独立）
  2. **组**：站在时间当下，发现有日线下跌中枢且无向上看多信号 → 组为**下跌未完成**
  3. **多**：当五均线金叉170且170在30分钟中枢上方 → 产生**看多结构**
  4. **做多**：回撤到该30分钟中枢上轨附近，配合涨停驱动 → 形成**进二扩展买点**

---

## 七、核心驱动力量（涨停/大阳线）

| 维度 | 要求 |
|------|------|
| **位置要求** | 涨停必须出现在看多结构形成之后、买点形成之前，且位于170均线附近的30分钟中枢区域 |
| **作用** | 涨停是**资金一致性的体现**，是「点火」信号 |
| **重要性** | 没有涨停的进二扩展，不建议参与 |
| **优选项** | 同期板块内多只个股有结构 + 有涨停 + 题材有真实驱动（涨价、政策等） |

---

## 八、进二扩展的买入信号

| 买入方式 | 操作 |
|----------|------|
| **潜伏买入** | 回撤到30分钟中枢上轨附近，小级别出现背驰（三卖背驰），第二天不破位买入 |
| **突破买入** | 跳空或放量突破中枢上轨，且开盘方向向上 |

> ⚠️ 扩展买点往往出现在快速拉升前（如润和软件不给你回撤直接上），此时只能等**第一次回撤**作为买点。

---

## 九、常见误区与纠偏

| 误区 | 纠偏 |
|------|------|
| 认为必须有日线独立转折才有买点 | 进二扩展不需要日线独立转折，只需30分钟中完美 |
| 做多中枢必须等于独立转折中枢 | 做多中枢可以是任何分类中枢，只要上轨与170形成安全边际 |
| 忽略虚线（小级别反向） | 虚线是30分钟中完美的灵魂，没有虚线就没有中完美 |
| 忽视核心驱动 | 没有涨停/板块效应的进二扩展不建议参与 |
| 回撤过浅不敢追 | 扩展买点常出现在快速拉升前，需敢于在第一次回撤处潜伏 |

## 十、交易心态与资金管理

### 10.1 进二扩展的优劣势

| 优势 | 劣势 |
|------|------|
| 往往对应快速上涨（如润和软件、君正集团），短期收益高 | 回撤可能较浅，容易错过上车机会，需要敢于潜伏 |

### 10.2 选股策略

- 每天复盘涨停板，筛选有：**板块一致性 + 结构（30分钟中完美）+ 独立转折**的个股
- 造纸、环保、特高压等题材需判断是中线还是短线（看是否有机构抱团趋势股）

### 10.3 止损

- 以**34均线**或**中枢下轨**为止损线

### 10.4 卖点提示（非本课重点）

- 主动卖点参考前方**筹码密集区**
- 个股短期涨幅超**50%**可全仓卖出

## 十一、大盘简评（2021年9月）

| 项目 | 点评 |
|------|------|
| 大盘风险 | 无系统性风险，仍为30分钟级别震荡 |
| 热点轮动 | 造纸（涨价驱动+资金一致性）、环保、特高压、老基建 |
| 高位板块 | 锂电、煤炭为机构被动配置，不适合开新仓 |
| 建议 | 优选底部有结构、有涨停、有板块一致性的个股，做中线/短中线布局 |

## 核心精炼

```
进二扩展 = 标准进二框架 + 30分钟中枢替代日线中枢

看多条件：170在30分钟中枢上方 + 五均线金叉170 + 中枢由虚线构成

分组多：
  分 → 找出周线中枢上轨
  组 → 判断日线级别趋势方向

买点 = 做多区域（30分钟中枢上轨） + 线段终结 + 核心驱动（涨停）
```
