# 课34：二阶段讲十九 — 职业交易走势思维认知

## 一句话摘要

**走势认知 = 日线级别为基准，170均线为量化落脚，通过「分+组」判断时间当下的看多结构。** 将复杂工具简化为「分组结构看多/看空」的纯粹状态——先有趋势（看多信号），后有买卖点（趋势内部线段的终结）。走势认知的三要素：级别、涨跌、量化落脚。

---

## 一、课程定位与目标

| 维度 | 说明 |
|------|------|
| 所属阶段 | 第二阶段（走势），承上启下认知课 |
| 前置知识 | 课16~31：日线级别线段、本质走势中枢、独立日线级别线段、走势分类 |
| 后续课程 | 课35：买卖点本质（线段终结的精确判断） |
| 核心目标 | 从复杂的工具学习中做减法，建立简洁本质的走势思维 |

### 课程属性

- **性质**：认知课，介于结构与买卖点之间
- **目的**：让学员跳出来思考，学会做减法
- **核心三要素**：级别、涨跌、量化落脚

> 思考的目的是让事情变简单。复杂是因为无法简单。股市交易最后应回归到「分组结构看多/看空」的纯粹状态。

## 二、独立思考与做减法

### 2.1 独立思考

| 要点 | 说明 |
|------|------|
| 课程设计的「空白期」 | 特意安排，让学员独立思考，避免只依赖圈子提问 |
| 学习的本质 | 不仅是掌握投资技巧，更是自我成长的过程 |
| 个人成就 | 股市的成就是个人的成就，工具和方法只是引路 |

### 2.2 简单 vs 复杂

```
复杂 ≠ 精细

工具学习需要精细（本质线段、中枢的精确量化）
    ↕
认知和决策必须简单（分组结构看多/看空）

类比：复杂机械表——对外行复杂，对内行简单
股市同理：一旦理解本质，涨跌判断就极为简单
```

> **核心原则**：思考的目的是让走势判断变得简单纯粹。所有障碍在于无法简单。

## 三、走势的本质与量化（核心）

### 3.1 走势的定义

走势 = **上涨、下跌、盘整** 三种基本形态。

但仅此不够——必须结合**级别**和**量化落脚点**才能用于实战。

### 3.2 中小资金的最优级别：日线级别

| 维度 | 为什么是日线级别 |
|------|------------------|
| 适用范围 | 中小资金（建议5000万以下） |
| 收益 | 日线级别上涨/下跌足够带来可观收益 |
| 回撤 | 能承受合理回撤；超日线级别的回撤（如50%）对小资金无必要 |
| 策略 | 以日线级别为基准判断涨跌/盘整，在趋势内部做小级别买卖点（看长做短） |

### 3.3 量化落脚：170均线 + 结构

| 概念 | 正确理解 | 错误理解 |
|------|----------|----------|
| 170均线 | 通过结构（中枢、级别、走势分类）串联起来的**时间工具** | 单纯的支撑/压力线 |
| 均线上穿 ≠ 上涨 | 必须通过走势结构（转折中枢、独立转折、分类中枢）判断真正的涨跌 | 肉眼看到均线上穿就是上涨 |

> 170均线只是结构的辅助表达，决策依据始终是走势结构本身。

### 3.4 走势认知的三个维度

```
走势认知三要素 ── 缺一不可

1. 级别      → 日线级别（中小资金最优基准）
2. 涨跌      → 看多/看空（由结构判断，非均线判断）
3. 量化落脚  → 170均线 + 走势结构（分+组）

三者合一 → 实战有效的走势判断
```

## 四、分组结构：看多/看空的核心

### 4.1 分组结构的含义

```
分：站在任何时间当下，将历史走势分割成独立的日线级别线段
    ├── 使用独立判断方法：
    │   ├── 新低
    │   ├── 三买卖
    │   ├── 一段
    │   └── 升级
    │
组：站在当前时间当下，组合成日线级别向上的走势
    └── 构造日线级别中枢（或走势类型）

看多 = 下跌结束的信号
    ├── 走势中完美转折
    ├── 独立转折
    └── 分类中枢看多

看空 = 上涨结束的信号（用于大盘或衍生品；个股做多市场只需掌握看多）
```

### 4.2 时间当下的重要性

| 原则 | 说明 |
|------|------|
| 站在当前时间点 | 交易永远以当下为基准，依据当下的分组结果判断 |
| 没有看多结构 → 不交易 | 没有看多结构就没有向上的趋势，不应交易 |
| 趋势确立后 → 等待线段终结 | 看多结构一旦出现，趋势确立，之后只需等待买卖点 |

### 4.3 案例：朗姿股份

```
底部震荡 → 通过分组得到独立日线级别线段
         → 最终出现独立转折
         → 看多确立
```

## 五、趋势与买卖点的关系

### 5.1 先有趋势，后有买卖点

| 要素 | 角色 | 说明 |
|------|------|------|
| 趋势（看多/看空） | **前提** | 先确定方向 |
| 买卖点 | **执行** | 趋势内部线段的终结 |

> 没有趋势，线段的终结只是震荡，没有交易价值。不同趋势方向（上涨/下跌）下，线段终结的判断方法不同。

### 5.2 买卖点的本质

```
买卖点 = 线段的终结

上涨趋势中 → 下跌线段的终结 = 买点
下跌趋势中 → 上涨线段的终结 = 卖点（或做空点）
滚动交易   → 基于不同级别的线段终结
```

### 5.3 中小资金的策略

| 原则 | 说明 |
|------|------|
| **不掐头去尾** | 不抄最底部的底（大资金才需要底部建仓），等趋势确立后做趋势内部的余生部分 |
| **买点的本质** | 所有买点本质上都是「三买」或类三买，但必须结合结构和级别，不能只看形态 |

```mermaid
graph LR
    A[等待看多结构] --> B[趋势确立]
    B --> C[等待线段终结（买点）]
    C --> D[持仓/滚动]
    D --> E[等待线段终结（卖点）]
    E --> A
```

## 六、常见误区纠正

### 6.1 不要依赖均线做决策

| 错误做法 | 正确理解 | 后果 |
|----------|----------|------|
| 看均线支撑/压力 | 170均线是辅助判断**结构**的工具 | 只看均线→「回踩就买、破位就卖」恶性循环 |

### 6.2 不要迷信题材和板块

| 错误认知 | 正确认知 |
|----------|----------|
| 跟随题材/板块选股 | 个股涨跌是**因**，板块/题材是**果**（计算机软件自动归类） |
| 找主线 = 找题材名称 | 真正的主线 = 一致性资金驱动的**个股结构** |

### 6.3 不要学而不会

| 现象 | 根本原因 |
|------|----------|
| 学了工具不会应用 | 没有把走势认知简化为「分组结构看多/看空」 |
| 买卖点无法精炼 | 把结构和买卖点混淆了 |

> **走势的认知 = 日线级别为基准，170均线为量化落脚，通过「分+组」判断时间当下的看多结构（下跌结束信号）；趋势确立后，买卖点即趋势内部线段的终结；所有复杂最终归于简单——分组结构看多/看空。**
