# 课47：三阶段讲二 — 股市亏钱的真相

## 一句话摘要

**「不会选股，赚不了大钱，但也亏不了大钱；不会操作，才会亏大钱。」** 本课从「市场认知」转向「自我认知」，揭示亏损的真正根源不是选股差，而是没有建立基于级别的自我买卖标准（持仓线、成本线），并被情绪左右。核心解决之道：认清「高低点」的相对性，以30分钟级别为节奏基准，严格执行持仓线与成本线。

## 二、股市亏钱的真正原因

### 2.1 课堂互动归纳：最亏钱的交易类型

- 重仓退市股、踩雷
- 买在下跌中继
- 追高（如欧菲光）
- 涨停到跌停
- 死扛、不止损/止盈
- 总是赚小钱跑掉

### 2.2 核心结论

> **「不会选股，赚不了大钱，但也亏不了大钱；不会操作，才会亏大钱。」**

| 问题类型 | 表现 | 根源 |
|----------|------|------|
| 选股问题 | 买到退市股、持续下跌股 | 底层认知缺失 |
| 操作问题 | 不会止盈/止损、死扛、恐慌卖出、反复被洗 | **亏损主因** |

### 2.3 典型亏钱模式

| 模式 | 过程 | 结果 |
|------|------|------|
| 模式一 | 买后微涨 → 不止盈 → 跌回 | 亏 |
| 模式二 | 买后上涨 → 赚一点就跑 → 后面大涨 | 卖飞 |
| 模式三 | 买后下跌 → 死扛 | 深套 |
| 恶性循环 | 反复「赚小亏大」 | 积累成大亏 |

### 2.4 为什么散户总是「赚一点就跑，亏了死扛」？

| 原因 | 说明 |
|------|------|
| 害怕 | 过去多次「坐过山车」（涨了没卖又跌回去），产生心理阴影 |
| 缺乏体验 | 没有尝试过赚大钱的感觉 → 缺乏持仓信心 |
| 根本根源 | **没有建立自我的买卖标准**，被市场情绪左右 |

## 三、核心认知转变：打破「低买高卖」的虚妄

### 3.1 「低点买入，高点卖出」是正确的废话

| 误区 | 正确认知 |
|------|----------|
| 脱离时间周期谈高低点 | 毫无意义 |
| 追求绝对低点/高点 | 不存在，只有某个级别下的相对高低 |
| 追求「卖在最高点」 | 打破时间维度的虚妄，反而陷入不断自我怀疑 |

### 3.2 市场游戏规则

- 市场不会针对某个人，但规则天然让人贪婪、恐惧、自我怀疑
- 试图找到「完美卖点」会让人入魔，最终亏损
- **正确做法**：建立**自我的**低点与高点标准（基于级别和结构），而非追求市场绝对高低

### 3.3 时间与空间的一体性

> 没有时间周期，就没有空间高低。

| 原则 | 说明 |
|------|------|
| 选择适合自我的级别 | 散户推荐30分钟级别 |
| 接受不完美 | 「卖不到最高点，买不到最低点」是常态 |
| 放弃虚妄 | 放弃追求绝对高低，建立自我相对标准 |

---

## 四、解决亏损的核心工具：持仓线 & 成本线

### 4.1 持仓线（决定何时离场、何为「高点」）

| 维度 | 说明 |
|------|------|
| 本质 | **不是一个均线（如34线），而是一个级别——30分钟级别** |
| 含义 | 你参与的是30分钟级别的一段上涨，当该级别完成（出现背驰/卖点），即为你的「高点」 |
| 视觉落脚 | 五均线的背离/背驰 + 结构完成 |
| 关键原则 | 30分钟级别走完就应卖出（**主动卖点**） |
| 常见错误 | 等跌破34线再卖（**被动卖点**，会回吐很多利润） |

**案例**：闽东电力、中青宝、清水源的卖点均为30分钟级别完成处。

### 4.2 成本线（决定何处是「低点」、安全边际）

| 维度 | 说明 |
|------|------|
| 定义 | **结构的破坏线 + 做多成本的参考**（如中枢、170线、34线等） |
| 作用一 | 告诉你何时该止损（结构破坏，只留100股） |
| 作用二 | 告诉你何处可做日T降低成本（结构未坏，下跌时低吸高抛） |
| 安全边际参考 | 170线、中枢、34线等 |

> ⚠️ **做T的前提是结构未破坏**，否则T得再好也亏市值。
> 散户应通过滚动操作（不同个股、不同波段）积累资金，而非死守一只。

---

## 五、操作节奏：30分钟级别是最适合散户的级别

| 维度 | 说明 |
|------|------|
| 中小资金优势 | 灵活，可快速切换，不参与不确定的下跌 |
| 最佳节奏 | 做30分钟级别的一段一段（上涨段做完就出，下跌段空仓或滚动） |
| 避免误区 | 不要试图做日线级别的大长线（那是大资金或极少数长牛股的事） |

> **一句话**：你的「高点」就是30分钟级别结束点，你的「低点」就是结构完成后的买点。

## 六、选股复盘的实战应用（以房地产开发为例）

### 6.1 每日复盘要区分「新旧核心驱动」

| 类型 | 特征 | 操作 |
|------|------|------|
| 旧的（如传媒/元宇宙/军工） | 已涨过一波 | 仅适合短线或关注卖点 |
| 新的（如房地产开发） | 第一次出现资金一致性 | **重点跟踪** |

### 6.2 房地产开发当天是否具备核心驱动？

| 条件 | 满足情况 |
|------|----------|
| 资金龙头（万科、保利）涨停或大涨 | ✅ |
| 底部个股（新华联等）也涨停 → 资金一致性 | ✅ |
| 有大单介入 | ✅ |
| 存在外部消息（高盛买入债券等） | ✅ |
| 具备买点（个股已形成转折结构） | ❌ **尚不具备** |

> **操作**：放入初始鱼池跟踪，等待转折结构完成后的买点（可能需要数周甚至两个月）。

### 6.3 市场环境

| 维度 | 状态 |
|------|------|
| 大盘 | 30分钟震荡，无系统性风险 |
| 旧热点（传媒） | 接近尾声 |
| 新热点（房地产） | 刚露头 |
| 散户策略 | 快人一步，在复盘中发现资金切换，而不是追涨 |

## 结构速查清单

```
持仓线检查（何时卖出）：
□ 30分钟级别上涨结构是否完成？
□ 是否出现五均线背离/背驰？
□ 是否达到主动卖点（结构完成处）？
→ 完成 → 卖出（不要等跌破34线被动卖出）
→ 未完成 → 继续持有

成本线检查（安全边际）：
□ 结构是否破坏（跌破关键成本线：中枢/170/34）？
□ 结构未坏 → 可做日T降低成本
□ 结构破坏 → 止损，只留100股跟踪
□ 结构全破 → 完全出局换股

每日复盘检查：
□ 区分新旧核心驱动板块
□ 新板块：资金龙头涨停？底部个股跟涨？资金一致性？
□ 有核心驱动但无买点 → 放入鱼池跟踪
□ 等待转折结构完成后的买点
```
