# 课31：二阶段讲十六 — 走势分类

## 一句话摘要

**走势分类 = 先「分」后「组」。** 分——用四种独立方法将历史走势分割成独立日线级别线段；组——站在当下组合成日线级别向上走势，得到三类中枢（周线转折、周线分类、中完美），进而判断看多区域和做多买点。独立是分的基础，组是为了得到中枢，中枢是交易的边界。

---

## 一、课程定位与目标

| 维度 | 说明 |
|------|------|
| 所属阶段 | 第二阶段（走势分类），走势分类核心综合课 |
| 前置知识 | 课16~28：日线级别线段、本质走势中枢、走势中枢唯一、独立日线级别线段及精炼、独立理解 |
| 课程性质 | **走势分类综合应用课**，整合独立判断、分类组合、三类中枢识别，为看多/做多奠定基础 |
| 核心目标 | 解决投资者两大痛点——抄底抄在半山腰与买在启动前却拿不住，即实现初始上涨的确认与关键启动点的确认 |

### 核心思想

```
走势分类 ≠ 独立日线级别线段本身
走势分类 = 分（分割历史） + 组（组合当下）

目的：得到当下的三类中枢
    周线转折中枢 → 判断下跌结束、上涨开始
    周线分类中枢 → 判断上涨过程中的中继
    中完美中枢   → 判断小级别买点

最终服务：看多区域 + 做多买点
```

> ⚠️ **关键提醒**：走势分类不是独立日线级别线段本身，而是基于独立线段进行分与组，得到当下的三类中枢，进而判断看多与做多。

## 二、走势分类的核心逻辑

### 2.1 为什么要学习走势分类？

| 痛点 | 原因 | 走势分类的解决 |
|------|------|---------------|
| 抄底抄在半山腰 | 买在下跌中继 | 明确个股的初始上涨（相对底部） |
| 买对底部但被洗出 | 资金利用率低 | 识别关键启动点，拿得住 |
| 不懂卖点也能赚钱 | 缺少参照 | 若能明确初始上涨和启动点，设定年化20%止盈即可 |

### 2.2 走势分类的本质

```
分：将整个走势图分割成一段段独立的日线级别线段
    ├── 使用四种独立判断方法
    │   ├── 新低独立
    │   ├── 三买卖独立
    │   ├── 一段独立
    │   └── 升级独立
    │
组：站在时间当下，将最近的独立线段组合成日线级别向上的走势
    └── 构造日线级别走势中枢（至少三段30分钟级别线段构成的日线中枢）
        └── 作为后续看多/做多的基准
```

### 2.3 关键概念辨析

| 误解 | 正确理解 |
|------|----------|
| 日线级别线段 = 看涨/看跌 | 日线级别线段只是方向的描述（向上或向下），不代表市场涨跌 |
| 独立 = 走势结束 | 独立只表示该段可以单独拿出，不影响走势的连续性 |
| 分完就结束了 | 真正的看涨/看跌需要通过组合后的中枢以及突破/回踩等法则来判断 |

> **核心原则**：独立是「分」的工具，但独立 ≠ 走势结束。分完后必须站在当下组合，构造出日线级别向上的走势。

---

## 三、走势分类的操作步骤

### 第一步：分（分割历史）

运用独立日线级别线段的四种判断方法，将历史走势分割成一段段独立的日线级别线段。

| 方法 | 触发条件 | 说明 |
|------|----------|------|
| 新低独立 | 下跌线段底创新低，且之前有反向 | 最基础的独立判定 |
| 三买卖独立 | 走势分类中枢出现三买（且新高）或三卖 | 中枢被终结后独立 |
| 一段独立 | 由单条本质线段升级而来的日线级别线段 | 内部有日线级别中枢 |
| 升级独立 | 通过非同处理具备日线级别走势中枢 | 虽非标准独立，但可作为独立使用 |

**分的原则**：有独立则分，无独立则不分。分是为了后续组合时能准确找到中枢。

### 第二步：组（组合当下）

站在当前时间点，从最近的低点（或转折点）开始，将分割出的独立线段组合成一条日线级别向上的走势。

**组合的目的**：构造出日线级别走势中枢（即由至少三段30分钟级别线段构成的日线中枢）。

> 如果无法组合出向上的日线走势（例如当前是下跌段），则等待新的独立线段出现。

### 第三步：识别三类中枢

经过分与组后，在当下走势中可能存在三类中枢（不一定全部存在）：

| 中枢类型 | 构成 | 作用 |
|----------|------|------|
| **周线级别转折中枢** | 由三条日线级别线段（上下上）构成 | 判断下跌结束、上涨开始 |
| **周线级别分类中枢** | 由三条日线级别线段构成 | 判断上涨过程中的中继 |
| **中完美中枢** | 由独立日线级别线段带来的特殊中枢（与170均线相关） | 判断小级别买点 |

> 这三类中枢的上下轨（边界）是后续看多区域和做多买点的参照。

### 第四步：看多与做多（后续课程展开）

```
看多区域：价格突破某中枢上轨（或170均线）后，进入可关注区域
    ↓
做多中枢：在看多区域内，围绕某中枢形成买点
    └── 如回踩不破上轨 → 产生做多机会
```

## 四、重要案例解析

### 4.1 穆高迪

| 阶段 | 特征 | 中枢情况 |
|------|------|----------|
| 初始 | 一条向上的日线级别线段（无独立） | 无任何中枢 |
| 走到16.79元 | 仍是一条向上的未完成走势 | 只有一个日线级别中枢 |
| 后续 | 出现周线级别转折中枢（非同升级得到） | + 日线分类中枢 + 中完美中枢 |
| 看多 | 突破170均线后 | 但真正做多买点后面才出现（小级别进攻买点） |

### 4.2 东华科技

| 特征 | 说明 |
|------|------|
| 中枢情况 | 无日线级别走势中枢（34均线无重叠），无独立 |
| 独立来源 | 通过三卖（形态上的三卖）得到一条独立线段 |
| 分类中枢 | 无周线分类，仅日线级别中枢 |
| 买点级别 | 30分钟级别进攻买点，非日线级别 |

### 4.3 任何软件

| 特征 | 说明 |
|------|------|
| 下跌段 | 无独立 |
| 上涨段 | 有中完美中枢（小级别） |
| 做多中枢 | 唯一的做多中枢即那个小级别中枢，无周线转折、无日线分类 |
| 看多区域 | 由170均线给出 |
| 买点位置 | 中枢上轨附近 |

### 4.4 浙中股份

| 特征 | 说明 |
|------|------|
| 中枢类型 | 有日线分类中枢（向上的）+ 周线转折中枢（由独立得到） |
| 买点出现 | 突破分类中枢上轨后的回踩 |

### 案例速查表

| 案例 | 周线转折 | 周线分类 | 中完美 | 买点特征 |
|------|----------|----------|--------|----------|
| 穆高迪 | ✅（非同升级） | ✅ | ✅ | 小级别进攻买点 |
| 东华科技 | ❌ | ❌ | ❌ | 30分钟级别进攻买点 |
| 任何软件 | ❌ | ❌ | ✅ | 中枢上轨附近 |
| 浙中股份 | ✅（独立得到） | ✅ | — | 突破分类中枢上轨回踩 |

## 五、独立日线级别线段的四种判断（复习）

| 方法 | 核心条件 | 要点 |
|------|----------|------|
| 新低独立 | 下跌线段底创新低 + 之前有反向 | 同价不创新低 |
| 三买卖独立 | 走势分类中枢 + 三买（且新高）/三卖 | 三买必须内部且创新高（有中枢时） |
| 一段独立 | 单条本质线段升级而来，内部有日线中枢 | 最直接的独立形式 |
| 升级独立 | 通过非同处理具备日线级别走势中枢 | **必须保证日线级别的连续性**，否则只能是一条日线级别走势 |

> ⚠️ **升级独立特别注意**：升级独立的前提是前后必须能组成同级别的走势（如三条日线级别走势类型构成周线），否则只是一条日线级别走势，不能作为独立使用。

## 六、常见错误与纠偏

### 错误1：只「分」不「组」

| 错误表现 | 纠正 |
|----------|------|
| 只把走势分成独立线段，却不知道如何组合成日线级别走势 | 分完后必须站在当下组合，构造出日线级别向上的走势（哪怕它只是未完成的） |

### 错误2：把日线级别线段当作看涨/看跌

| 错误表现 | 纠正 |
|----------|------|
| 向上线段代表未来会涨，向下线段代表未来会跌 | 日线级别线段只是同级别分解的基本单位，涨跌由中枢和突破决定 |

### 错误3：独立判断混乱（尤其升级独立）

| 错误表现 | 纠正 |
|----------|------|
| 误以为任何有日线中枢的片段都能独立，忽略连续性要求 | 升级独立的前提是前后必须能组成同级别的走势 |

### 错误4：忽略当下的「组」

| 错误表现 | 纠正 |
|----------|------|
| 总是从历史起点开始看，而不是从当下最近的低点开始组合 | 走势分类服务于当下交易，应站在当下时间点，从最近的可独立点开始组合 |

### 纠偏速查

```
常见错误路径：
    分完独立线段 → 直接判断涨跌 → ❌ 错误

正确路径：
    分完独立线段 → 站在当下组合 → 得到三类中枢 → 判断看多区域 → 等待做多买点 → ✅ 正确
```

## 七、大盘与操作建议（讲课当时）

### 大盘状态

| 维度 | 判断 |
|------|------|
| 级别 | 30分钟级别下跌过程中的第二个中枢震荡（5分钟反抽） |
| 企稳确认 | 未确认下跌结束 |
| 护盘力量 | 券商等大金融护盘，但无持续性 |
| 题材状态 | 分化严重，无新的核心资金驱动主线 |

### 操作建议

| 对象 | 建议 |
|------|------|
| 新学员 | 不急于开仓，利用下跌过程选股（找有独立线段、有中枢结构的个股），等待买点课程学完后再实践 |
| 老学员 | 可小仓位参与有结构的个股（如周线转折或日线分类中枢的买点），但仓位不宜高，注意分化风险 |
| 选股方向 | 关注有独立线段、有日线或周线中枢的个股，忽略纯题材炒作 |

## 八、总结与后续

### 走势分类全流程

```
原始走势图
    │
    ├─ ① 分：用四种独立方法分割历史
    │     ├── 新低独立
    │     ├── 三买卖独立
    │     ├── 一段独立
    │     └── 升级独立
    │
    ├─ ② 组：站在当下组合成日线级别向上走势
    │     └── 构造日线级别走势中枢
    │
    ├─ ③ 识：识别三类中枢
    │     ├── 周线级别转折中枢
    │     ├── 周线级别分类中枢
    │     └── 中完美中枢
    │
    └─ ④ 判：判断看多区域与做多买点
          ├── 突破中枢上轨/170均线 → 看多区域
          └── 回踩不破上轨 → 做多买点
```

### 核心公式

```
走势分类 = 分（独立日线级别线段） + 组（日线级别向上走势）
         → 三类中枢（周线转折、周线分类、中完美）
         → 看多区域 + 做多中枢
```

### 一句话精炼

> **走势分类 = 先「分」（将走势分割成独立的日线级别线段）+ 后「组」（站在当下组合成日线级别向上的走势），得到三类中枢（周线转折、周线分类、中完美），进而判断看多区域和做多买点。独立是分的基础，组是为了得到中枢，中枢是交易的边界。**
