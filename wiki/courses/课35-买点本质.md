# 课35：二阶段（下）讲一 — 职业交易买点的本质

## 一句话摘要

**买点 = 做多区域（安全边际） + 线段终结 + 核心驱动力量，三要素缺一不可。** 买点不是上涨点，也不是公式化的均线/形态信号；买点分为选股买点和建仓买点两类。精通一类买点、找到自我买点，是职业交易的核心能力。

## 二、买点的"两不是"——纠正常见误区

### 2.1 买点不是「上涨点」

| 区分 | 上涨点 | 职业买点 |
|------|--------|----------|
| 特征 | 任何股票都可能突然上涨（重组、事件驱动、一字涨停） | 有结构支撑、可复制的交易信号 |
| 可复制性 | 无法复制，无法持续盈利 | 可持续盈利 + 复利滚动 |
| 例子 | 广宇发展、中公教育因重组一字涨停——不是职业买点 | 趋势确认后，线段终结处的回撤结束信号 |

> **核心理念：「掐头去尾赚余生」**——不追求不切实际的暴涨，只做符合交易模式的买点。

### 2.2 买点不是「公式化」

| 误区 | 说明 |
|------|------|
| 纯代码选股 | 市场情绪、资金逻辑无法完全量化，单纯靠指标/代码选股长期会失败 |
| 量化适用范围 | 仅适用于趋势性交易执行（如进三买点），不适用于选股的有效性判断 |
| 建议 | 代码可作为辅助工具，不建议作为主要选股手段 |

---

## 三、买点三要素（核心框架）

**任何有效买点，必须同时满足以下三个要素：**

```
买点 = 做多区域（安全边际） + 线段终结 + 核心驱动力量
       ~~~~~~~~~~~~~~~~~   ~~~~~~~~~~   ~~~~~~~~~~~~~~
          结构前提            信号确认       资金验证
```

### 3.1 要素一：做多区域（安全边际）

- **定义**：通过「分+组+多」流程得出的中枢上轨或关键价位，是安全边界
- **作用**：控制盈亏比——回撤结束位置离做多区域越近，安全边际越好
- **获取方法**（见第五节详细展开）

### 3.2 要素二：线段终结

- **定义**：下跌线段的结束信号（背驰、动态线段结束），需要明确级别（30分钟/日线）
- **核心原则**：回撤结束位置必须在安全边际附近，否则盈亏比不划算
- **本质**：所有买点本质上都是一买（第一类买点），但出现在不同结构下（如回撤到中枢上轨的买点，对更大级别而言是类三买）

### 3.3 要素三：核心驱动力量（资金一致性）

- **定义**：资金的趋同行为——板块/题材内多只个股同时具备结构性买点
- **判断维度**：
  - 板块效应：板块内多只个股同时涨停或大涨
  - 涨停逻辑：有外部事件驱动（政策、行业消息等）
  - 个股结构：不能是底部无结构的突然涨停
- **举例**：磷化工、鸿蒙概念等——通过复盘涨停榜、板块效应、结构筛选

> ⚠️ 未来学习每个具体买点（防一、防二、进一、进二、进三）时，都要围绕这三要素展开。

## 四、买点的两分类：选股买点 vs 建仓买点

| 维度 | 选股买点 | 建仓买点 |
|------|----------|----------|
| 目的 | 发现具备核心驱动力量的潜在标的 | 在安全边际附近精确入场 |
| 侧重 | 三要素中的「核心驱动力量」 | 三要素中的「做多区域 + 线段终结」 |
| 时机 | 提前于建仓，属于「养股」阶段 | 回撤结束信号确认后，属于「做多」阶段 |

### 交易模式

- **底仓**（长期持有，基于核心驱动）
- **滚动**（短线操作，降低持仓成本）
- **加仓**（趋势确认后逐级加仓，如通过进三买点再次加仓）

> 一个底部启动的优秀个股（具备核心驱动力量），做完第一波后不要丢弃，后续可通过进三再次加仓。

---

## 五、安全边际（做多区域）的获取方法：「分组多」

### 三步流程

| 步骤 | 操作 | 说明 |
|------|------|------|
| **分** | 将历史行情分为日线级别线段 | 只有独立转折的线段才可分 |
| **组** | 站在时间当下，组成日线级别走势（或周线级别） | 按走势分类法则组合 |
| **多** | 判断看多信号 | 两种看多方式：独立转折看多 / 下跌走势中枢看多（走势中完美） |

### 结论

**做多区域 = 中枢上轨 / 关键价位**，是一切买点的安全边界。

---

## 六、线段终结与级别

- **所有买点本质上都是一买**（第一类买点），具体出现在哪取决于所在结构
  - 回撤到中枢上轨 = 对本级别是一买，对更大级别是类三买
- **线段终结需要明确两点**：
  1. **级别**：是30分钟级别的线段终结，还是日线级别的线段终结
  2. **终结信号**：背驰、动态线段结束（具体内容在后续买卖点课程展开）
- **铁律**：回撤结束的位置**必须**在安全边际附近，否则盈亏比不划算——宁愿错过，不要犯错

---

## 七、核心驱动力量的判断（资金一致性）

> 不是简单看涨停，要看资金行为和结构背景。

| 维度 | 判断标准 |
|------|----------|
| 板块效应 | 板块/题材内多只个股同时有结构性买点信号 |
| 涨停质量 | 涨停由特大单/大单主导（非小单、中单） |
| 个股结构 | 底部需要有构造结构（不能是无结构的突然涨停） |
| 外部驱动 | 涨停有明确的政策、行业等外部事件支撑 |

### 实战复盘方法

1. 每日复盘涨停榜
2. 识别板块效应（板块内多只涨停）
3. 检查个股底部结构
4. 纳入「鱼池」跟踪，等待买点信号

---

## 八、34均线的关键应用

| 位置 | 买点类型 | 资金配置 |
|------|----------|----------|
| 34均线**以下** | 只有防守买点 | **不配置资金**（不含底仓） |
| 34均线**以内/以上** | 进攻买点 | 可做滚动和加仓 |

### 铁律

- 任何个股价格回到34均线以下 → **剔除**
- 回到34均线上方后 → 重新考虑

> 34均线是进攻买点和防守买点的分界线。

## 九、卖点简述（本课非重点）

| 卖点类型 | 说明 |
|----------|------|
| **主动卖点** | 预测性卖点，借助前方筹码密集区（历史中枢上轨、170峰值等）判断相对高位 |
| **被动卖点** | 价值投资式长期持有底仓的卖点 |
| **滚动操作** | 不在于判断绝对顶，而在于「市场无法继续前行时」主动减仓 |

## 十、执行与心态

| 原则 | 说明 |
|------|------|
| **宁愿错过，不要犯错** | 回撤买点若离做多区域太远 → 放弃 |
| **拒绝「弱者思维」** | 不要感叹错过牛股，每天复盘，底部启动的个股永远存在 |
| **避免「自我魔障」** | 不要钻牛角尖、过度优化、只追求形态量化——否则学死 |
| **学习路径** | 掌握结构（分组多）→ 理解买点三要素 → 精通一类买点 → 找到自我买点 |

## 核心精炼

```
买点 ≠ 上涨点
买点 ≠ 公式化信号

买点 = 做多区域（安全边际）+ 线段终结 + 核心驱动力量
       结构前提              信号确认        资金验证

分类：选股买点（侧重核心驱动） vs 建仓买点（侧重安全边际+线段终结）
获取安全边际：分→组→多 → 中枢上轨
执行铁律：34均线以下不配资金，回撤远离安全边际则放弃
```
