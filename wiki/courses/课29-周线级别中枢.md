# 课29：二阶段讲十四 — 职业交易之周线级别中枢

## 一句话摘要

**周线级别中枢由三条或以上日线级别线段（或日线级别走势类型）通过170均线重叠构成。** 有两种构造方式：标准定义（独立线段组合）和非同升级（非独立但具备日线中枢）。独立日线级别线段的判断是前提。周线中枢用于转折判断和分类中继，是中小资金的最大有效级别。

| 维度 | 说明 |

### 核心思想

```
周线级别 = 中小资金投资者的最大有效级别
（无需关注月线）

周线级别的三大作用：
① 判断市场环境（牛市/熊市/震荡）
② 判断上涨力度和持仓周期
③ 选股与买卖点级别的判断

方法学习 = 「零件组装」 → 最终形成复利机器
```

> ⚠️ **关键提醒**：周线级别中枢是走势分类组合法则学习前的最后一个知识点，必须先掌握。

## 二、周线级别中枢的定义

### 2.1 标准定义

> **周线级别中枢 = 由至少三条或以上日线级别线段（或日线级别走势类型）通过170均线重叠构成。**

| 要素 | 说明 |
|------|------|
| 次级别 | 日线级别（日线级别线段 或 日线级别走势类型） |
| 数量要求 | 至少三条 |
| 重叠工具 | 170均线（由数学推导得出的量化参数） |
| 类比 | 日线级别中枢由30分钟级别线段通过34均线重叠构成 |

### 2.2 与日线级别中枢的类比

```
日线级别中枢：
    次级别 = 30分钟级别线段
    均线参数 = 34均线
    构成 = 三条30分钟线段 + 34均线重叠

周线级别中枢：
    次级别 = 日线级别线段/走势类型
    均线参数 = 170均线
    构成 = 三条日线级别线段/走势类型 + 170均线重叠
```

> 逻辑完全一致，仅是级别的跃升。

---

## 三、周线级别中枢的两种构造方式

### 方式一：标准定义（多条独立日线级别线段组合）

| 条件 | 说明 |
|------|------|
| 数量 | 至少三条**独立**的日线级别线段 |
| 重叠 | 满足170均线重叠 |
| 独立性要求 | 日线级别线段必须是趋势性的，且能通过「独立」判断分割出来 |

#### 案例：重心菌业

```
走势结构：下跌 → 上涨 → 下跌（上下上）
判断：
    中间的上涨线段是独立的（通过三买或新低判断）
    因此可以将整体走势分割为三条独立的日线级别线段
结论：
    → 标准定义的周线级别下跌中枢 ✅
```

### 方式二：非同/升级方式（非独立但具备日线级别中枢）

| 条件 | 说明 |
|------|------|
| 适用场景 | 某段走势**不是**独立的日线级别线段（无法分割），但内部拥有日线级别走势中枢 |
| 处理方式 | 该段走势作为「日线级别走势类型」参与周线级别中枢的构造 |
| 本质 | 非同概念在周线级别中的应用 |

#### 案例：爱普股份

```
走势结构：从起点到终点，整体是一条日线级别走势（有日线中枢）
判断：
    无法分割成独立的日线级别线段（没有三买/三卖或新低）
    但因为内部有日线级别中枢，它可以视为一个日线级别走势类型
处理：
    结合前、后的其他日线级别线段（或走势类型）
    形成上下上三段 → 构成周线级别转折中枢
结论：
    → 非同处理得到的周线级别中枢 ✅
```

### 两种方式的本质相同

```
标准定义：能独立 → 直接拆成多条日线级别线段
非同升级：不能独立 → 整体视为日线级别走势类型

共同点：次级别都是「日线级别」的东西
区别仅在于「独立性」的判断
```

---

## 四、独立日线级别线段对周线级别的作用

### 核心逻辑

> **要判断一个走势是否构成周线级别中枢，必须先明确日线级别线段的独立与否。**

| 独立的作用 | 说明 |
|------------|------|
| 分割走势 | 将整体走势分割成多条日线级别线段 |
| 数量判定 | 判断是否满足「三条或以上」的要求 |
| 转折判断 | 配合170均线形成看多区域 |
| 周线升级 | 后续在分类组合法则中应用 |

### 案例对比

| 案例 | 日线级别线段数量 | 周线级别中枢 | 结论 |
|------|-----------------|-------------|------|
| **重心菌业** | 三条独立线段 | ✅ 成立 | 标准定义的周线级别中枢 |
| **任子行** | 从头到尾一条线段 | ❌ 不成立 | 仅为日线级别走势 |
| **爱普股份** | 非独立但含日线中枢 → 非同升级 | ✅ 成立 | 非同处理的周线级别转折中枢 |

## 五、周线级别中枢的两种应用场景

### 5.1 周线级别转折中枢

| 维度 | 说明 |
|------|------|
| 用途 | 判断**底部反转** |
| 位置 | 通常出现在下跌末端 |
| 配合 | 配合170均线形成**看多区域** |
| 示例 | 重心菌业的上下上三段构成下跌中枢 → 属于转折中枢（后续有买点可能） |

### 5.2 周线级别分类中枢

| 维度 | 说明 |
|------|------|
| 用途 | 判断**上涨过程中的中继** |
| 位置 | 出现在上涨趋势中 |
| 示例 | 穆高迪案例（未完成走势）→ 构成上涨过程中的周线级别分类中枢 |

> ⚠️ **注意**：转折中枢和分类中枢的上下轨取值可能不同，具体在下节课（课30）讲解。

## 六、非同在周线级别中的理解

### 非同的应用场景对比

| 级别 | 非同使用频率 | 说明 |
|------|-------------|------|
| 日线级别 | 较少 | 多数被本质线段延伸处理掉 |
| **周线级别** | **较多** | 周线中枢构造中非同应用广泛 |

### 核心逻辑

```
一段走势 ≠ 独立的日线级别线段（无法分割）
    但
内部有日线级别走势中枢（已经升级）
    ↓
它可以充当周线级别的「次级别走势类型」

这与日线级别中枢构造中：
    虚线（不成线段）但具有30分钟级别走势 → 可构成日线中枢
的逻辑完全一致。
```

## 十一、总结

### 周线级别中枢核心公式

```
周线级别中枢 = 三条及以上日线级别线段（或走势类型） + 170均线重叠

构造方式：
├── 方式一：标准定义 → 三条独立日线级别线段 + 170均线重叠
└── 方式二：非同升级 → 非独立但含日线中枢的走势类型参与构造

独立日线级别线段判断 = 前提条件

应用场景：
├── 转折中枢 → 判断底部反转 + 看多区域
└── 分类中枢 → 判断上涨中继

使用级别 = 中小资金最大有效级别
```

### 一句话精炼

> **周线级别中枢由三条或以上日线级别线段（或日线级别走势类型）通过170均线重叠构成；有两种构造方式：标准定义（独立线段组合）和非同升级（非独立但具备日线中枢）；独立日线级别线段的判断是前提；周线中枢用于转折判断和分类中继，是中小资金的最大有效级别。**
