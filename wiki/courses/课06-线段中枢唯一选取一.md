# 课6：本质线段中枢之唯一选取（一）

## 一句话摘要

线段中枢的唯一选取依赖于**前后相邻两中枢不重叠三法则**——共用一根连接K、后中枢必须完成起手三式+黑K、肉（肉眼可见）优先于非肉。本课是一阶段线段中枢从静态知识点到动态分析的过渡之讲。

## 二、前后两中枢不重叠三法则

线段中枢的唯一性依赖于相邻两个线段中枢之间必须同时满足以下三个条件，否则后一个中枢会被**无效化**或**延伸处理**：

### 法则一：共用一根连接K（唯一共用K线）

- 相邻两中枢之间，**仅允许一根**K线同时属于前/后两个中枢——即连接K。
- 除此之外，**不允许任何K线的价格区域重叠**。
- 任何超出连接K的K线重叠 → 前中枢延伸，后中枢不成立。

### 法则二：后中枢必须完成「起手三式 + 黑K」

- 后一个中枢必须在起手三式形成（反正两穿/三笔/5k重叠任一种完成）**且**黑K确认之后，才算独立存在。
- 如果在起手三式构造过程中（尚未完全形成中枢），K线就已回到前一个中枢区域内部 → **后中枢无法形成，前中枢延伸**。
- 反之，如果后中枢的起手三式 + 黑K**已经形成**（上轨或下轨已确定），即使后续K线回到前中枢区域 → **两个中枢独立并存**，互不影响。

### 法则三：肉优先于非肉

- 当前一个中枢是**非肉**（五均线无波峰波谷重叠，但起手三式+黑K仍构成中枢），后一个中枢是**肉**（肉眼可见的五均线缠绕），且两者区域有重叠时——以肉眼可见的**肉中枢**为准，非肉中枢被无效化。
- 理由：线段中枢唯一选取的最终目标是找到五均线真正的肉眼可见重叠，以此统一级别、走势分类和买卖点。

---

## 三、连接K的定义与作用

### 3.1 定义

```
连接K = 前一个线段中枢的最后一根K线 = 后一个线段中枢的第一根起始K线
```

连接K是前/后两个线段中枢之间**唯一允许共用的K线**。

### 3.2 如何找到连接K

最简单的方法：离开前一个中枢区域后，**再也没有回到该区域的那根K线**（肉眼可见即可判定）。

### 3.3 连接K与首破坏K的关系

> **Q：连接K能不能是首破坏K？**

**A：可以**。但前提是连接K之后的新中枢完全从该K线开始观察——之前的K线全部忽略（养成「抹掉」习惯：眼里只看连接K之后的走势，前一段的K线统统抹掉）。

---

## 四、后中枢需满足「起手三式 + 黑K」

### 4.1 核心判断逻辑

```
判断流程：
  后中枢起手三式是否已完成？
    ├── 未完成 → 其间任何K线回到前中枢区域 → 前中枢延伸，后中枢不成立
    └── 已完成 + 黑K确认 → 即使后续K线回到前中枢区域 → 两中枢独立并存
```

### 4.2 延伸的判断

如果在一个可能的线段中枢形成过程中，**始终无法满足起手三式 + 黑K**（例如一直未能出现新的中枢线刷新，或一直回到前一个中枢区域），则：
- 不形成新中枢
- 原线段中枢**延伸**覆盖整个K线区间

---

## 五、肉优先于非肉

### 5.1 冲突规则

| 前中枢 | 后中枢 | 有重叠 | 处理 |
|--------|--------|--------|------|
| 肉 | 肉 | 有 | 按不重叠三法则正常处理 |
| 非肉 | 肉 | 有 | **肉中枢保留**，非肉中枢无效化 |
| 肉 | 非肉 | 有 | 按不重叠三法则正常处理 |
| 非肉 | 非肉 | 有 | 按不重叠三法则正常处理 |

### 5.2 核心要义

肉（肉眼可见的五均线波峰波谷重叠）代表市场的真正共识区域，非肉中枢虽然有起手三式 + 黑K的形式构成，但五均线并未真正缠绕——在唯一选取时，肉中枢具有更高的优先级。

> 一分钱的线段中枢也是有效中枢——非肉中枢同样合法，只是在与肉中枢冲突时被降级。

## 六、作业讲评：关键案例

### 6.1 贵州茅台案例

**核心要点**：判断正向一笔的起始K时，必须以**动态跟踪**的思维——先看反向一笔是否已结束，若未结束则起始K会变化。

前后两个中枢不重叠的判断方法：看后一个中枢的起手三式K线是否回到了前一个中枢区域。若在形成过程中已回到，则前中枢延伸。

### 6.2 金益科技案例

**核心要点**：
- 非肉中枢与肉中枢并存时，**肉优先**。
- **一分钱的线段中枢也是有效中枢**——不要因为中枢幅度小就忽视。

### 6.3 江苏吴中 & 小康股份案例

**核心要点**：
- **5k重叠是最快完成起手三式的方式**——可用于判断「三种起手三式中谁最先完成」。
- 肉眼可见的中枢取前两个波峰/波谷；若第二个无效，则只取第一个。

### 6.4 其他补充问题

- **跳空情况下如何取最左连续两根K线的重叠？** 跳空视为中间插入K线，取跳空前那根K线的价位。
- **连接K之后的习惯**：从连接K开始，把之前的所有K线「抹掉」，只看连接K之后的新中枢构造。

## 十、总结：核心法则速查

| 法则 | 要点 |
|------|------|
| **不重叠三法则** | ①共用连接K ②后中枢须完成起手三式+黑K ③肉优先于非肉 |
| **连接K** | 唯一共用K线，离开前中枢后不再回来的那根 |
| **后中枢成立条件** | 起手三式+黑K完成后，与前中枢独立并存 |
| **延伸条件** | 起手三式完成前K线回到前中枢区域 |
| **肉优先** | 非肉中枢在与肉中枢重叠时被无效化 |
| **5k重叠** | 最快完成起手三式的方式 |

> 主讲人强调：线段中枢的唯一性是画出标准线段的基础，后续走势分类、买卖点都依赖于此。务必通过作业和回放彻底掌握，不要贪快。
