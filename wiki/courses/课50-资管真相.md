# 课50：三阶段讲五 — 职业资管的真相

## 一句话摘要

**职业资管的真相：以「零成本」为思维方向，以「单笔亏损2%」为风控底线，以「股数不变」为操作纪律，以「不赚钱不开新仓」为账户配置原则，通过选股、买卖点、做T、滚动的一体化运作，实现账户资金的持续稳健增长。**

## 二、风控的最高原则：时间即财富

> 一切风控、仓位管理、止损止盈的最终目的：**让未来的每一刻时间都成为你的财富，而不是灾难。**

| 维度 | 说明 |
|------|------|
| 本质 | 通过控制回撤、降低成本、滚动操作，使账户资金在未来任何时间点都处于相对安全状态，不会因单次暴跌而毁灭 |
| 投资 vs 投机 | 纯投机：一买一卖赚差价，持有期间不创造价值，时间可能是灾难 |
| 将投机做成「事业」 | 通过滚动、做T、换股，让时间成为朋友，最终实现零成本或账户持续正向 |

### 「跌」为什么是好事？

> 只有跌，你才能以更低成本买回，才能实现零成本或降低成本。如果一直涨不跌，你无法摊薄成本，反而容易在高位重仓被套。

---

## 三、零成本思维（核心指导思想）

| 概念 | 说明 |
|------|------|
| **什么是零成本** | 以某价格买入，上涨后卖出部分或全部，等跌回原位再买回相同数量，此时持仓成本为0（或很低），后续涨跌都是纯利润 |
| **中小资金现实** | 几乎不可能在单只个股上做到真正零成本（需要大级别上涨+回撤+精准操作），但思维方式不能丢——每一笔交易、做T、滚动，都应朝零成本方向努力 |
| **替代方案** | 通过换股实现账户资金的「零成本」——账户资金曲线不断创新高，回撤极小，相当于投入的本金早已收回，剩下的全是利润在滚动 |

> **中小资金的资管核心** = **控制单笔亏损 + 保持股数不变（仓位不变）+ 通过滚动/换股让账户总资金曲线正向增长。**

## 四、个股仓位管理（单只股票）

### 4.1 总账户持仓数量

| 条件 | 推荐持仓数 |
|------|-----------|
| 资金100万以下 | 2-3只 |
| 资金100万以上 | 3-5只 |
| **上限** | **最多同时持有 3-5只**（超过则盯盘困难，操作质量下降） |

### 4.2 单只个股最大持仓仓位

| 边界 | 仓位比例 |
|------|---------|
| **上限** | 总账户的 **40%**（一只股票最多占你总资金的40%） |
| **下限** | 总账户的 **8%**（当安全边际较远时，用最小仓位） |

> 40%上限由 **单笔2%亏损原则** 推算而来（见4.4）。

### 4.3 建仓仓位由「安全边际距离」决定

> **安全边际**：你的建仓价格到「被动卖点」（结构破坏点）之间的空间。

| 止损空间 | 仓位 | 说明 |
|----------|------|------|
| **5%以内** | 可满仓该个股上限（40%）| 优秀买点 |
| **25%左右** | 仅用最小仓位（8%） | 安全边际远 |
| **超过25%** | 不建议参与 | 风控困难 |

### 4.4 单笔亏损原则：总账户的 2%

> 任何一只个股，从建仓到触发被动卖点（止损），你**计划亏损的金额不超过总账户资金的2%**。
>
> 例如：100万账户 → 单笔最大亏损 2万元。

| 仓位推算公式 | 最大仓位 = 2% ÷ 止损空间% |
|-------------|-------------------------|
| 止损空间 5% | 2% ÷ 5% = **40%** |
| 止损空间 25% | 2% ÷ 25% = **8%** |

> **目的**：单笔小亏，长期积累，避免一次大亏毁灭账户。

### 4.5 仓位不变原则（核心铁律）

> 一旦确定某只个股的初始股数（例如2万股），在整个持有期间（直到主动卖点或被动卖点清仓），**股数保持不变**。

| 操作 | 规则 |
|------|------|
| 做T | 当天买卖保持股数不变 |
| 滚动 | 主动卖点后减半仓，剩余半仓股数仍不变 |
| **禁止加仓** | 中小资金做波段（30分钟级别），加仓会提高成本，回撤时亏损扩大；加仓后股数变多，被动卖点亏损超2% |

### 4.6 机动仓位（做T资金）

| 维度 | 说明 |
|------|------|
| 预留比例 | 总账户中预留 **20%** 作为机动资金 |
| 建仓:机动 | 约 **4:1** |
| 操作方向 | **朝零成本方向**——所有操作目的是降低成本 |
| **铁律：不反T** | 做T卖飞了（卖出后没接回）→ 放弃，**绝不追高买回**，更不逆向加仓。错过就等下一次买点 |

## 五、账户资金配置（多只股票）

### 5.1 不赚钱，不开新仓（重要原则）

> 当你持有第一只股票，该股票处于**浮亏状态**时，**绝不开第二只股票的新仓**。
>
> **原因**：防止一错再错，防止在行情不好时盲目扩大亏损。只有第一只股票赚钱（浮盈）后，才考虑建仓第二只。
>
> **本质**：让你的账户进入「良性循环」——赚钱 → 加新仓 → 继续赚钱。

### 5.2 满仓方式

- 不是一开始就满仓，而是通过 **一只只股票盈利，逐步把仓位加满**（持仓3-5只）。
- 满仓 = 账户总资金几乎全部投入，但每只股票仓位不超过40%，且每只都在盈利中。

### 5.3 行业配置

| 原则 | 说明 |
|------|------|
| **避免同板块** | 不买同一板块或同一概念的股票（容易同涨同跌，失去对冲） |
| **行业分散** | 应配置不同行业的个股：如一只科技、一只消费、一只医药 |
| **攻防搭配** | 初始买点（防守型）和进三买点（进攻型）合理搭配，**不要全仓进三**（波动大） |

### 5.4 34均线原则

> **永远不配置34均线下方的股票**（那些处于下跌趋势的个股）。你的资产应该是良性资产（处于上涨结构中的个股）。

## 六、不同资金规模的操作差异

| 资金规模 | 核心策略 |
|----------|----------|
| **小资金** | 发挥「灵活换股」的优势——快进快出，换股频繁 |
| **大资金** | 注重「零成本」和「时间财富」的长线思维——通过滚动降低成本，靠时间创造复利 |

> 核心差别：**资金越小，越要灵活换股；资金越大，越要注重零成本和长线思维。**

---

## 七、与之前课程的关系（买卖点/做T/滚动）

| 概念 | 定义 | 仓位影响 |
|------|------|----------|
| **主动卖点（背驰卖点）** | 出现时减仓 **50%**（股数减半，剩余仓位继续持有） | 股数变化（减半），但剩余仓位股数不变 |
| **被动卖点（结构破坏）** | **清仓**（只留100股观察） | 全部清仓 |
| **做T** | 只能在结构还在、主动卖点未出现的情况下做；目的是降低成本，保持股数不变 | 股数不变 |
| **滚动** | 主动卖点之后的新买点（如进三），重新建仓（仓位可与初始相同或减半），是另一段交易的开始 | 新一轮开始 |

> 所有操作都要服务于「股数不变」和「单笔亏损2%」的原则。

## 八、常见错误与纠正

| 常见错误 | 正确做法 |
|----------|----------|
| 追高加仓（越涨越加） | 股数不变，通过做T降成本 |
| 死扛不卖（被动卖点已触发） | 结构破坏立即清仓，留100股观察 |
| 满仓一只股票 | 单只≤40%，同时持有3-5只分散 |
| 亏损时开新仓 | 不赚钱不开新仓，先处理好持仓 |
| 卖飞后追高买回 | 放弃，等下一次买点 |
| 全仓做进三 | 攻防搭配，进三波动大需控制比例 |

## 九、资金管理背后的认知要点

| 认知要点 | 说明 |
|----------|------|
| **市场无标准** | 任何量化指标都是参考，你的唯一标准是自我标准（级别、仓位、亏损容忍度） |
| **心理层面** | 卖飞、止损、踏空都是常态；只要按自己的标准执行，就应坦然接受。纠结于单次得失，永远走不出「心魔」 |
| **职业 vs 散户** | 职业玩家注重账户曲线的稳健增长，而不是某只股票的暴利；散户往往相反 |
| **时间换财富** | 只要你的账户不回撤过大，时间会站在你这边。**复利的力量依赖「不回撤」** |

## 十、每日复盘与「资金承接」

### 10.1 复盘目的

> **发现新的资金承接（新旧题材切换），提前潜伏下一波主流。**

### 10.2 如何判断新资金承接

| 条件 | 说明 |
|------|------|
| 板块资金龙头 | 固有的大资金个股出现涨停或大阳线 |
| 底部个股涨停 | 初始买点构造中的底部个股也出现涨停 |
| 资金一致性 | 板块内多个个股涨停 |
| 外部驱动 | 存在外部消息或行业变化（非必需，但常见） |

> 上述条件满足，且个股正处于构造转折结构中（买点未出现），则 **纳入跟踪**。

### 10.3 当前大盘状态参考

- 30分钟震荡，没有新的核心驱动出现
- 老题材（传媒、汽车零部件等）未结束
- 策略：短线为主，不重仓

---

## 十一、总结：职业资管的核心公式

> **稳健持续盈利 = （选股能力 + 买卖点规则） × 仓位管理 × 不赚钱不开新仓 × 单笔亏损2% × 股数不变**

| 要素 | 作用 |
|------|------|
| 选股与买卖点 | 决定胜率 |
| 仓位管理 | 决定盈亏幅度 |
| 中小资金优势 | 灵活换股——不要与大资金死磕一只股票 |

> **最高级的风控不是技术指标，而是让时间成为你的财富——即任何时刻你的账户都有能力应对下跌，并最终创新高。**
