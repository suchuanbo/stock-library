## 一、课程三阶段架构

### 第一阶段：本质线段中枢（1~9课，约1~1.5个月）

**目标**：掌握线段中枢的静态画法，建立标准线段。

| 课次 | 主题 | 核心知识点 |
|------|------|------------|
| 课1 | 资本入门礼 | 炒股本质、散户定位、学习要求 |
| 课2 | 本质线段之中枢线 | k0、k 的定义与配对关系 |
| 课3 | 线段中枢之起手三式 | 反正两穿、三笔、5k重叠 |
| 课4 | 线段中枢之形成确认 | 起手三式 + 黑K = 中枢成立 |
| 课5 | 线段中枢之中枢取值 | 肉/非肉判断、上下轨取值规则 |
| 课6 | 线段中枢之唯一选取（一） | 前后两中枢不重叠法则、连接K |
| 课7 | 线段中枢之唯一选取（二） | k0/k 刷新法则、中枢延伸与重组 |
| 课8 | 线段中枢之唯一选取（三） | 形成当下的临界处理、独立K 刷新 |
| 课9 | 线段中枢综合融会贯通 | 整合全部静态知识，以两个核心（中枢线、是否触发重组）为框架 |
| 课10 | 线段顶底职业交易四法则 | 3k组合、独立两k、34均线两侧、结构完美与转折k最近、转折k量化定义 |

### 第二阶段：买点体系（10~45课，约4个月）

**目标**：建立完整的买点体系，理解安全边际与核心驱动。

**核心结论（一句话）**：

```
买点 = 大级别看涨 + 小级别下跌结束 + 安全边际
```

**买点分类**：

| 买点类型 | 定位 | 安全边际依托 |
|----------|------|-------------|
| 防守类（防一、防二） | 大级别底部，安全性高 | 170托中枢 / 中枢托170 |
| 进一类 | 独立转折后的首次回撤 | 中枢上轨附近 |
| 进二类 | 走势转折后回撤 | 中枢内部 |
| 进三类 | 34均线上方，强者恒强 | 34均线托底 |

**核心驱动力量**（选股关键）：

```
核心驱动 = 资金趋同（协同）+ 外部/内部驱动 + 结构支撑
```

识别条件：
1. 板块内多个个股涨停（资金一致性）
2. 板块资金龙头也涨停/大涨
3. 涨停由特大单/大单主导
4. 底部个股正在构造转折结构

## 二、技术体系总览（第一阶段核心）

### 2.1 概念层级关系

```
线段（最小级别走势）
├── 中枢线（由 k0 + k 配对形成，实体K线与五均线的关系）
├── 起手三式（构成线段中枢的三种方式）
│   ├── 反正两穿（先反穿后正穿，不下才有上）
│   ├── 三笔（价格持续新高/新低，与方向无关）
│   └── 5k重叠（首k与破坏k之间有5根K线）
├── 黑K（非正向跳空五均线，保障线段级别）
└── 线段中枢（上下轨之间的区域）
    ├── 肉（肉眼可见）：五均线出现反向再形成正向一笔
    └── 非肉（非肉眼可见）：五均线未出现反向但仍构成中枢
```

### 2.2 线段中枢唯一选取法则

```
前后两中枢是否重叠？
├─ 不重叠 → 各自独立
└─ 重叠
    ├─ 后中枢形成中已回到前中枢区域 → 前中枢延伸
    └─ 前非肉、后肉 → 重组，非肉无效，以肉为准
```

### 2.3 关键量化指标

| 概念 | 量化法则 |
|------|----------|
| k0 | 实体在五均线正侧，且独立于线段中枢之外 |
| k（中枢线确定K） | 实体反突破五均线，与k0唯一配对 |
| 黑K | 非正向跳空五均线（向下跳空/粘合/同价） |
| 肉（肉眼可见） | 正向一笔形成过程中，五均线已出现反向 |
| 波峰 | 母均线值比前一天高、比后一天高（同价也算） |
| 波谷 | 母均线值比前一天低、比后一天低（同价也算） |

### 2.4 中枢取值规则

| 线段方向 | 固定轨 | 取值方法 |
|----------|--------|----------|
| 下跌线段 | 中枢线 = 上轨（固定） | 肉：取前两个波谷中较高者；非肉：找最左连续两根K线重叠区下沿 |
| 上涨线段 | 中枢线 = 下轨（固定） | 肉：取前两个波峰中较低者；非肉：找最左连续两根K线重叠区上沿 |

## 三、交易系统总览

### 3.1 核心公式

```
稳健持续盈利 = (选股能力 + 买卖点规则) × 仓位管理 × 不赚钱不开新仓 × 单笔亏损2% × 股数不变
```

### 3.2 卖点体系

| 类型 | 定义 | 触发动作 |
|------|------|----------|
| 主动卖点（背驰卖点） | 某个级别上涨出现背驰（30分钟/5分钟） | 30分钟背驰→清仓；5分钟背驰→减仓50% |
| 被动卖点（结构破坏） | 跌破持仓线/做多成本线/中枢上轨/170线 | 清仓（只留100股观察） |
| 资金卖点 | 涨幅达40%~50%且触及重要压力位 | 可考虑全仓主动止盈 |

### 3.3 仓位配置（基于大盘级别）

| 日线级别 | 30分钟级别 | 建议仓位 |
|----------|-----------|---------|
| 上涨 | 上涨 | 8~10成 |
| 上涨 | 盘整 | 6~8成 |
| 盘整 | 上涨 | 6~8成 |
| 盘整 | 盘整 | 4~6成 |
| 盘整 | 下跌 | 2~4成 |
| 下跌 | 任意 | 0~2成 |

### 3.4 复盘流程

```
每日涨停复盘（条件筛选）
  → 鱼池（核心驱动板块中所有涨停股）
    → 龙门（已出现或正在构造转折结构的个股）
      → 化龙/自选（最有可能形成买点的3~5只）
        → 建仓后进入操作表格每日跟踪
```

## 四、学习要求与心态

- **时间要求**：零基础至少坚持 9~12 个月，前1.5个月打基础
- **核心原则**："非黑即白"——所有判断必须有明确、量化的法则依据，不能模糊、不能靠感觉
- **思维转变**：从"寻找标准答案"→"建立自我标准"，接受市场不确定性
- **关键警示**：选股问题不会让你大亏，**操作问题才是亏损主因**（死扛、不止盈、不止损）
