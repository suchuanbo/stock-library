# 课38：二阶段（下）讲四 — 职业交易之线段终结（上）

## 一句话摘要

**没有回撤，就没有买点。** 小级别（30分钟线段以下）回撤的终结判断是职业交易者的核心技能。非职业交易者只需理解「回撤」和「做多区域」两个概念即可建仓。回撤的两种级别——小级别（未破34均线）对应进二/进一/部分进三买点，30分钟走势类型（破34均线）对应防守买点。回撤的确认，从一根K线的最高价开始，到跌破五均线、跌破前日收盘价，形成最小趋势。

## 二、行情与环境简评（国庆前）

- 国庆前市场下跌，属于**利润兑换**性质，无系统性风险
- 进三买点较多，进二也有机会
- ⚠️ **强调卖点的重要性**：利润回吐超过20%说明卖点没学明白
- 大盘：30分钟级别震荡，仓位建议50%以下

## 三、小级别线段终结的核心概念

### 3.1 回撤的两种级别

| 级别 | 特征 | 对应买点 |
|------|------|----------|
| **小级别回撤** | 30分钟线段以下，**未破34均线** | 进二、进一、部分进三 |
| **30分钟走势类型回撤** | **跌破34均线** | 防守买点 |

> 区分两种回撤级别的关键分界线：**34均线**。

### 3.2 小级别终结需要的四个知识点

| 序号 | 知识点 | 重要性 | 说明 |
|------|--------|--------|------|
| ① | **小级别结构画法** | ★★★ 核心 | 基于日线K线的开盘价/收盘价/最高价/最低价画折线 |
| ② | **做多区域（安全边际）** | ★★★ 必须 | 必须存在，否则买点无效 |
| ③ | **背驰** | ★★☆ 次要 | 辅助工具，非核心，用于潜伏买入 |
| ④ | **回撤的定义与确认** | ★★★ 必须 | 起始条件 + 跌破条件 + 五均线条件 |

---

## 四、小级别结构的画法（日线图上的折线）

### 4.1 基本定义

| 概念 | 阳线（收盘 > 开盘） | 阴线（收盘 < 开盘） |
|------|---------------------|---------------------|
| **起始**（同侧最值） | 最低价 | 最高价 |
| **终结** | 收盘价 | 收盘价 |

**连接规则**：前一根K线的「终结」与后一根K线的「起始」相连，形成折线。

### 4.2 方向确认与最小趋势

```
连续两根K线同向 → 确定最小级别趋势（上涨或下跌）

示例：
  Day1 阴线：起始=最高价，终结=收盘价
  Day2 开盘价跌破 Day1 收盘价 → 形成向下折线 → 向下最小趋势确认
```

> 回撤开始后才需要判断方向。方向确认只需两根K线。

### 4.3 回撤的确认条件（三项同时满足）

| 条件 | 说明 |
|------|------|
| **① 起始为最高价** | 回撤开始的那根K线以最高价为起始 |
| **② 跌破前日阴K收盘价** | 第二天跌破昨日阴K的收盘价（可间隔时间） |
| **③ K线价格跌破五均线** | 否则属于五均线内部震荡，不构成进二级别回撤 |

> ⚠️ 回撤不一定立刻形成趋势，也可能只是盘整中枢。

## 五、小级别回撤的两种结构类型

| 结构类型 | 特征 | 买点信号 |
|----------|------|----------|
| **盘整（下跌只有中枢无趋势）** | 回撤不形成持续下跌趋势，在中枢内震荡 | 需等方向确认（突破折线高点） |
| **趋势下跌** | 回撤形成明确的向下趋势结构 | 背驰 + 方向反转信号 |

### 背驰的使用原则

| 有背驰 | 无背驰 |
|--------|--------|
| 可做**潜伏买入**（提前介入） | 必须等**方向确认**（突破折线高点）后再买入 |

> 背驰不是核心判断工具，只是次要辅助。

## 六、小级别终结在实战中的应用

### 6.1 非职业交易者

```
建仓条件（三者同时满足）：
  结构正确 + 做多区域（安全边际） + 有回撤

不需要精确判断回撤是否终结，可接受时间成本（稍早或稍晚买入）。
```

### 6.2 职业交易者

```
精准流程：
  回撤起始K线 → 趋势延续 → 背驰或方向反转 → 买入

核心能力：两根K线之间看到机会，精确捕捉回撤终结点。
```

### 6.3 买点三前提（任何买点不可缺失）

| 前提 | 说明 |
|------|------|
| **结构到位** | 独立转折、做多中枢（走势分类完成） |
| **做多区域** | 安全边际附近（中枢上轨或关键价位） |
| **核心驱动力量** | 涨停/大阳线 + 主力资金介入 + 板块一致性 |

## 七、选股与板块建议

### 选股原则

| 规则 | 说明 |
|------|------|
| **优先34均线以上** | 进三、进二买点为主，强者恒强 |
| **避免34均线以下** | 除非做防守买点，否则不配置资金 |
| **核心驱动识别** | 涨停 + 主力资金（大单流入）+ 板块内多只个股同步 |

### 关注板块（基于当时行情）

- 造纸、环保、电力等有资金一致性的板块
- 通过复盘涨停榜 → 识别板块效应 → 检查个股底部结构 → 纳入跟踪

## 八、交易心态与常见错误

| 常见问题 | 纠正 |
|----------|------|
| 忽视卖点 | 利润回吐超20%说明卖点体系未掌握 |
| 无回撤买入 | 「没有回撤，就没有买点」——这是铁律 |
| 忽略做多区域 | 无安全边际的买点盈亏比极差 |
| 死磕34均线以下个股 | 34均线以下只做防守，不配资金（或极低仓位） |

## 核心精炼

```
回撤两种级别：
  小级别（未破34均线）→ 进二/进一/部分进三
  30分钟走势类型（破34均线）→ 防守买点

小级别终结四知识：
  结构画法（日线K线折线）→ 做多区域 → 背驰（次要）→ 回撤定义

回撤确认三条件：
  起始=最高价 + 跌破前日阴K收盘价 + 跌破五均线

买点三前提（缺一不可）：
  结构到位 + 做多区域 + 核心驱动力量

灵魂法则：没有回撤，就没有买点。
```
