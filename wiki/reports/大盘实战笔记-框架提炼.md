# 大盘实战笔记 · 框架提炼

> 从 77 次实战分析中提取的**重复出现的分析维度、概念和操作模式**。这不是某一课的具体判断，而是老师面对任何市场时都在使用的那套「心智模型」。

---

## 一、分析框架全景图

```
           ┌──────────────────────────────────┐
           │         大盘分析三步走             │
           │                                  │
           │  ① 定级别 → 判断当前处于什么阶段   │
           │  ② 看结构 → 涨/跌/盘？中枢在哪？   │
           │  ③ 找点位 → 哪些价位是关键防守线   │
           │                                  │
           │  然后推导：                        │
           │  → 仓位该多少？                    │
           │  → 能不能开新仓？                  │
           │  → 选什么方向？                    │
           └──────────────────────────────────┘
```

老师几乎每次大盘分析都遵循这个顺序。以下展开每个维度。

---

## 二、级别判断（定操作节奏的锚）

老师从不含糊地谈「大盘涨了」「大盘跌了」，而是精确到**哪个级别**。

### 2.1 三大级别及其作用

| 级别 | 作用 | 常见表述 |
|------|------|---------|
| **日线级别** | 定**系统性风险**——日线上涨=系统性机会，日线盘整=无系统风险按节奏走，日线下跌=系统性风险严控仓位 | 「日线级别上涨未结束」「日线级别盘整」 |
| **30分钟级别** | 定**操作节奏**——这是散户的核心操作级别，决定开仓/持仓/减仓的节奏 | 「30分钟级别上涨过程中」「30分钟震荡」「30分钟下跌」 |
| **5分钟级别** | **辅助做T和预警**——用于判断当日操作方向、预警个股小级别卖点。散户可忽略，只做好30分钟一买一卖 | 「5分钟级别回撤属正常调整」「5分钟反抽未确认企稳」 |

### 2.2 级别判断的典型模式

```
日线级别：上涨 / 盘整 / 下跌
    ↓
30分钟级别：上涨 / 盘整 / 下跌 （重点关注是否出现「转折确认」）
    ↓
5分钟级别：涨 / 跌 （可选，辅助判断日内方向）
```

**常见表述模式**（贯穿全部课程）：
- 「当前大盘处于日线盘整 + 30分钟震荡」
- 「30分钟级别上涨过程中，5分钟回撤属正常调整」
- 「30分钟级别下跌过程中的反抽，未确认下跌结束」
- 「未出现30分钟级别进攻买点 / 被动卖点」

### 2.3 级别确认的判断依据

- 30分钟上涨确认：站上关键点位（如3496）+ 结构完成
- 30分钟下跌确认：跌破关键点位（如3480）+ 结构破坏
- 企稳确认：需要结构和买点双重验证，不能只看反弹

## 三、结构分析（市场状态的定性）

### 3.1 三种基本状态

老师对市场的定性始终围绕三种状态：

| 状态 | 特征 | 操作含义 |
|------|------|---------|
| **上涨** | 30分钟级别上涨结构中，关键点位未破 | 可积极操作，持仓为主 |
| **下跌** | 30分钟级别下跌过程中，未确认结束 | 谨慎，不开新仓，利用下跌选股 |
| **震荡/盘整** | 无明确方向，在区间内反复 | 控制仓位（4-6成），以个股为主 |

### 3.2 结构分析的核心概念

| 概念 | 定义 | 实战作用 |
|------|------|---------|
| **中枢** | 价格波动的核心区间，多空平衡区 | 判断当下处于中枢上/中/下，决定攻防策略 |
| **中枢震荡** | 在中枢区间内反复波动 | 说明方向未选择，不宜重仓 |
| **中枢上轨** | 中枢区间的上边界 | 突破可能转为上涨，也是安全边际的参考 |
| **结构成本位** | 当前上涨或下跌结构的「成本」位置 | 跌破 = 结构可能被破坏 |
| **看多结构** | 走势终完美后的做多区域 | 买点的前提——先有看多结构，后有买点 |
| **进攻买点** | 30分钟级别的转折买点 | 确认上涨启动的信号 |

### 3.3 结构判断的典型模式

```
「当前处于30分钟级别[上涨/下跌/盘整]过程中」
「[是否]出现[买入/卖出]信号」
「若[跌破/突破][关键点位]，则可能转为[新状态]」
```

例如：
- 「仍处于30分钟上涨结构中，未出现上涨结束信号」
- 「30分钟级别下跌过程中的第二个中枢震荡」
- 「若3480跌破，可能引发30分钟级别下跌」

## 四、关键点位体系（市场的「防线」）

老师在53课的分析中反复使用以下关键点位。这些点位本质上都是**结构的关键位置**。

### 4.1 核心点位清单

| 点位 | 性质 | 出现频率 | 含义 |
|------|------|---------|------|
| **3480** | 30分钟结构成本位 | ⭐⭐⭐⭐⭐ 最高频 | 30分钟上涨结构的成本线，破 = 上涨可能结束 |
| **170均线** | 中长期趋势线 | ⭐⭐⭐⭐ | 个股强弱分水岭，「170线上方选股」是铁律 |
| **中枢上轨** | 震荡区间的上边界 | ⭐⭐⭐⭐ | 突破则可能转上涨；也是防守买点的安全边际 |
| **3347** | 前期低点防守位 | ⭐⭐⭐ | 课6中明确提到「3347未破不看空」 |
| **3437** | 短线防守位 | ⭐⭐ | 课8中若跌破需减仓 |
| **3441** | 中枢下轨参考 | ⭐⭐ | 课2中震荡区间下沿 |
| **3496** | 上涨确认位 | ⭐⭐ | 课9中站上3496确认上涨 |
| **3498** | 170均线位置 | ⭐⭐ | 课29中明确提到，跌破延续下跌 |
| **3572** | 后期防守位 | ⭐⭐ | 课44-45中出现，跌破可能进一步走弱 |
| **3300** | 日线级别成本位 | ⭐⭐ | 课24中提到，跌破则日线宽幅震荡 |

### 4.2 点位判断的典型逻辑

```
「若[点位A]不破，则仍属[当前状态]」
「若[点位A]跌破，可能转为[新状态]」
```

这种「若……则……」的条件判断模式贯穿全部77条分析——老师从不预测，而是明确**触发条件**。

### 4.3 关键点位的本质

关键点位不是随意画的一条线，而是**结构的「成本位」**：
- 结构成本位 = 当前走势的「资金成本区」
- 中枢上轨 = 多空平衡区的边界
- 170均线 = 中长期趋势的生命线

## 五、仓位原则（大盘分析的最终落脚点）

老师反复强调：**大盘分析的核心输出是仓位指导，不是个股买卖。**

### 5.1 仓位与级别的对应关系（课51核心框架）

| 日线状态 | 30分钟状态 | 建议仓位 | 策略定性 |
|---------|-----------|---------|---------|
| 日线上涨 | 30分钟上涨 | 8-10成 | 积极进攻，重仓操作 |
| 日线上涨 | 30分钟盘整 | 6-8成 | 正常操作，注意节奏 |
| 日线盘整 | 30分钟上涨 | 6成 | 偏积极，半仓附近 |
| 日线盘整 | 30分钟盘整 | **4-6成** | 中性仓位，精选个股 |
| 日线盘整 | 30分钟下跌 | 2-4成 | 谨慎操作，等待安全边际 |
| 日线下+30分反弹 | — | ≤2成 | 仅做反弹，快进快出 |
| 日线下+30分盘/下 | — | 0-2成 | 系统性风险，轻仓/空仓 |

### 5.2 仓位管理的反复强调点

贯穿77条分析的高频仓位原则：

| 原则 | 出现场景 | 频率 |
|------|---------|------|
| **4-6成** | 30分钟盘整/震荡时的标准仓位 | ⭐⭐⭐⭐⭐ |
| **50%以下** | 国庆前后、不确定阶段的保守仓位 | ⭐⭐⭐⭐ |
| **不超过6成** | 震荡市防守仓位上限 | ⭐⭐⭐⭐ |
| **≤2成** | 30分钟下跌中仅做反弹 | ⭐⭐ |
| **40%本金不动** | 始终保留安全垫 | ⭐⭐ |

### 5.3 仓位调整的触发条件

- **30分钟状态变化**（如上变下）→ 即使个股无卖点也要整体减仓
- **关键点位被破** → 提前减仓防范
- **日线级别转入下跌** → 系统性风险，大幅降仓

## 六、操作建议模式（新学员 vs 老学员）

老师对不同类型的学员有**分层操作建议**，这在77条分析中反复出现。

### 6.1 分层模式

| 学员类型 | 典型建议 | 适用场景 |
|---------|---------|---------|
| **新学员** | 「不急于开新仓」「静心学习」「先学好买点再实践」「等待确定性的日线级别买点」 | 几乎每次下跌/震荡分析中都出现 |
| **老学员** | 「可小仓位参与」「做好滚动减仓」「不要追高」「可参与防守性买点」 | 有一定确定性但非最佳时机时 |
| **所有人** | 「个股无卖点就持有」「不因大盘波动错杀个股」「严格按买卖点规则操作」 | 普遍适用 |

### 6.2 两种赚钱效应的区分（贯穿全课程的核心理念）

| 类型 | 含义 | 出现场景 |
|------|------|---------|
| **持仓赚钱效应** | 前期已布局、个股无卖点的持股者正在赚钱 | 上涨初期、震荡市 |
| **开新仓赚钱效应** | 新开仓也能马上赚钱 | 30分钟上涨确认后的强势行情 |

老师反复提醒：**当前是「持仓赚钱」还是「开新仓赚钱」，决定你能不能进场。** 大部分时候只有持仓赚钱效应，不适合盲目开新仓。

## 七、板块分析方法

### 7.1 板块分析的两个核心维度

老师对板块的判断始终围绕两个概念：

| 维度 | 含义 | 判断标准 |
|------|------|---------|
| **资金一致性** | 不同派系的大资金同时做多同一方向 | 板块内多股涨停 + 资金龙头（涨幅最大趋势最强的个股）涨停 + 底部个股协同上涨 |
| **核心驱动** | 有外部逻辑支撑的资金一致性 | 政策驱动/涨价驱动/技术突破等 + 资金一致性 → 持续性更好 |

### 7.2 板块分类模式

贯穿77条分析中的板块分类：

| 类型 | 特征 | 操作策略 | 示例 |
|------|------|---------|------|
| **新核心驱动** | 刚出现的资金一致性+外部驱动，底部构造结构 | 纳入鱼池跟踪，等待买点 | 数字货币（课51）、汽车整车（课52） |
| **旧核心驱动（尾声）** | 前期有过核心驱动，仍在延续但接近结束 | 只能做短线，非初始买点 | 传媒（课47/50）、汽车零部件（课51/52） |
| **有资金一致性无核心驱动** | 有板块效应但缺乏外部逻辑支撑 | 仅适合短线进三，不适合中长线 | 医美（课40/41） |
| **无资金一致性的异动** | 个别股票涨但板块无协同 | 不宜追高 | 煤炭银行（课43）、电力（课44） |
| **高位抱团（机构被动配置）** | 高位板块，机构因配置需求持有 | 不适合开新仓 | 锂电/煤炭（课35/37） |

### 7.3 板块分析的核心认知（课34、课53总结）

> 「个股涨跌是**因**，板块/题材是**果**——计算机软件自动归类的结果。」
> 「真正的主线 = 一致性资金驱动的**个股结构**，不是题材名称。」

## 八、哲学层面的反复强调

### 8.1 核心信念体系

从课2到课53反复出现的底层信念：

| 信念 | 含义 | 来源 |
|------|------|------|
| **不测而测** | 不做预测，只明确「当下触发哪个信号」，准备好应对方案 | 贯穿全部 |
| **市场无形，唯我有形** | 市场无固定形状，你的框架（结构+核心驱动+买卖规则+操作表格）是你的「有形工具」 | 课53 |
| **非黑即白** | 技术层+认知层都有明确秩序，拒绝灰色地带 | 课1/2 |
| **结构是唯一可量化的落脚点** | 市场的核心变化=大资金驱动，唯一可量化的是结构 | 课26 |
| **大盘只是温度计** | 大盘指导仓位和风险，不直接决定个股买卖 | 课51 |
| **买卖点比大盘重要** | 个股按自身结构操作，大盘只在仓位层面起作用 | 课16/23 |

### 8.2 反复出现的操作铁律

| 铁律 | 出现频率 |
|------|---------|
| 「个股无卖点就持有，不要因指数波动错杀」 | ⭐⭐⭐⭐⭐ |
| 「新学员不急于交易，静心学习」 | ⭐⭐⭐⭐⭐ |
| 「不追高」 | ⭐⭐⭐⭐ |
| 「利用下跌/震荡过程选股」 | ⭐⭐⭐⭐ |
| 「严格按买卖点规则操作」 | ⭐⭐⭐⭐ |
| 「仓位以防守思维为主」 | ⭐⭐⭐ |

## 九、框架的核心公式

如果把老师77次分析提炼成一套可操作的流程：

```
每日大盘分析 = 
  ① 判定日线级别状态（定系统性风险）
  → ② 判定30分钟级别状态（定操作节奏）  
  → ③ 识别当前结构（涨/跌/盘）+ 中枢位置
  → ④ 标记关键点位（结构成本位、170均线、中枢边界）
  → ⑤ 确定对应仓位区间（日线×30分钟 → 仓位百分比）
  → ⑥ 判断赚钱效应类型（持仓赚钱 vs 开新仓赚钱）
  → ⑦ 结合核心驱动判断选股方向（新驱动跟踪 / 旧驱动短线）
```

这七步就是老师面对任何市场时，脑中运行的分析程序。
