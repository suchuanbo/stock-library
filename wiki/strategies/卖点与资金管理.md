## 一、卖点的本质

**市场无标准**——如果存在固定的赚钱标准，那么所有人学会后都会赚钱，谁亏钱？投资者必须建立自我的标准。

> 市场如水，没有固定形状。你能赚多少钱取决于你用什么"容器"（自我标准）去盛水。

---

## 二、卖点分类

### 主动卖点（背驰卖点）

| 项目 | 内容 |
|------|------|
| 定义 | 在某个级别上涨出现背驰时卖出 |
| 本质 | 你主动选择卖出，锁定利润，避免后续回撤 |
| 级别选择 | 30分钟背驰适合波段持仓；5分钟背驰适合短线、需盯盘 |

#### 组合建议（散户模式）

```
第一步：出现5分钟级别背驰 → 减仓50%（主动部分）
第二步：剩余仓位等待30分钟级别背驰或结构破坏 → 再清仓或被动卖出
第三步：若下跌后结构未坏，可用日T降低成本；若结构破坏，只留100股观察
```

### 被动卖点（结构破坏）

| 项目 | 内容 |
|------|------|
| 定义 | 当上涨结构被破坏（如跌破持仓线、做多成本线）时卖出 |
| 本质 | 右侧确认的卖点，滞后但确定性强 |

常见参考：
- 五均线跌破34均线
- MACD死叉
- 跌破中枢上轨或170线
- 跌破自己的成本线（安全边际）

> 作用：防止更大的亏损或利润回吐，是最后的防守线。

### 资金卖点

- 当个股上涨幅度达到 **40%~50%**，且触及前期重要压力位（中枢上轨、170线等），可考虑全仓主动止盈
- 来源：家庭资产配置中高风险部分的预期收益，扣除通胀（约6~8%）后，单笔交易需要40~50%的涨幅

## 三、持仓线、成本线、做多成本

| 概念 | 说明 |
|------|------|
| 持仓线 | 决定何时离场、何为"高点"。不是一条均线，而是你的操作级别（如30分钟级别） |
| 成本线 | 结构的破坏线 + 做多成本的参考（如中枢、170线、34线等） |
| 做多成本 | 告诉你何时该止损（结构破坏），何时可做T（结构未坏） |

> 注意：不要用34均线作为做多成本（它只是牛熊参考）。做多成本应使用中枢或170线。

## 四、资金管理（职业资管）

### 风控的最高原则：时间即财富

一切风控、仓位管理、止损止盈的最终目的：**让未来的每一刻时间都成为你的财富，而不是灾难。**

### 零成本思维

- 以某价格买入，上涨后卖出部分或全部，等跌回原位再买回相同数量，持仓成本为0
- 中小资金的替代方案：通过换股实现账户资金的"零成本"——账户资金曲线不断创新高，回撤极小

### 单笔亏损原则：总账户的2%

- 任何一只个股，从建仓到触发被动卖点（止损），计划亏损的金额不超过总账户资金的**2%**
- 例如：100万账户，单笔最大亏损2万元
- 据此反推：
  - 若止损空间5%，则最大仓位 = 2% ÷ 5% = **40%**
  - 若止损空间25%，则仓位 = 2% ÷ 25% = **8%**

### 仓位由"安全边际距离"决定

| 止损空间 | 可用仓位 |
|----------|---------|
| 5%以内 | 可以满该个股仓位的40%（优秀买点） |
| 25%左右 | 最小仓位8%，甚至放弃该股票 |
| 超过25% | 不建议参与 |

### 股数不变原则（核心）

- 一旦确定某只个股的初始股数，在整个持有期间（直到主动/被动卖点清仓），**股数保持不变**
- 做T、滚动操作都不能改变底仓股数
- 目的：严格控制单笔亏损不超过2%

### 仓位配置（基于大盘级别）

| 日线级别 | 30分钟级别 | 建议仓位 |
|----------|-----------|---------|
| 上涨 | 上涨 | 8~10成 |
| 上涨 | 盘整 | 6~8成 |
| 盘整 | 上涨 | 6~8成 |
| 盘整 | 盘整 | 4~6成 |
| 盘整 | 下跌 | 2~4成 |
| 下跌 | 任意 | 0~2成 |

## 五、"不赚钱，不开新仓"原则

当你持有第一只股票，该股票处于浮亏状态时，**绝不开第二只股票的新仓**。

- 防止一错再错，防止在行情不好时盲目扩大亏损
- 只有当你第一只股票赚钱（浮盈）后，才考虑建仓第二只
- 本质：让你的账户进入"良性循环"

## 六、操作表格（操作日志）

### 建仓初始记录

| 内容 | 说明 |
|------|------|
| 日期、个股名称 | 基础信息 |
| 建仓股数 | 风控核心是控制股数不变 |
| 建仓成本 | 买入价+手续费 |

### 建仓后的跟踪流程

```
第一步：建仓成本是否被跌破？
  ├─ 是 → 立即"反踢"（先卖后买），记录做多成本
  └─ 否 → 继续持有，可做正踢（先买后卖）

第二步：做多成本是否被跌破？
  ├─ 是 → 触发被动卖点，清仓（留100股观察）
  └─ 否 → 持股待涨

第三步：主动卖点 vs 被动卖点（两者并行，谁先触发谁执行）
  ├─ 主动卖点（级别背驰）→ 卖半仓
  └─ 被动卖点（结构破坏）→ 清仓
```

## 七、核心公式

```
稳健持续盈利 = (选股能力 + 买卖点规则) × 仓位管理 × 不赚钱不开新仓 × 单笔亏损2% × 股数不变
```
