## 一、买点的本质（核心结论）

```
买点 = 大级别看涨 + 小级别下跌结束 + 大级别提供的安全边际
```

### 买点的五个共同要素

无论防守、进一、进二、进三，任何有效的买点都满足：

| 要素 | 说明 |
|------|------|
| 大级别看涨 | 确认上涨趋势（独立转折、走势转折、分类中枢） |
| 小级别下跌结束 | 回撤结束，结构确认 |
| 安全边际清晰 | 中枢上轨/170均线/34均线的支撑关系明确 |

> 缺少任何一个要素，都不是可靠的买点。

### 减法思维

- 不要死记买点名字（防一、防二、进一、进二、进三）
- 当你理解了本质，任何符合"大级别看涨+小级别回撤结束+安全边际"的位置，都是买点

### 看多 ≠ 做多

- 看多（上涨趋势确认）不一定马上做多，因为可能震荡很久，时间成本高
- 做多需要等待"起始点"或"加速点"——即安全边际附近的小级别回撤结束

## 二、买点分类

| 买点类型 | 定位 | 安全边际依托 | 回撤级别 |
|----------|------|-------------|----------|
| 防守类（防一、防二） | 大级别底部，安全性高 | 170托中枢 / 中枢托170 | 30分钟走势 |
| 进一类 | 独立转折后的首次回撤 | 中枢上轨附近 | 30分钟线段/走势 |
| 进二类 | 走势转折后回撤 | 中枢内部 | 30分钟线段/五均线 |
| 进三类 | 34均线上方，强者恒强 | 34均线托底 | 短线 |

### 进三类补充说明

- 属于**强者恒强**买点，适合震荡市或牛市
- 下跌市中进三失败概率高（大盘环境差）
- 30分钟下跌之后做进三，需要30分钟转折结构，且本身已有核心驱动

## 三、核心驱动力量（选股关键）

### 定义

```
核心驱动 = 资金的趋同（协同）+ 外部/内部驱动 + 结构支撑
```

### 资金趋同的识别条件

1. **板块内多个个股涨停**（资金一致性）
   - 板块成分股越多，涨停数量越多越好，至少2只以上
2. **板块的资金趋势龙头也涨停或大涨**（最强信号）
   - 在行业板块中涨幅最大、趋势最强的个股即为资金龙头
3. **涨停由特大单/大单主导**（不是小单、中单）
   - 说明大资金抢筹，怕拿不到筹码
4. **底部构造结构的个股也在上涨**
   - 形成"老大涨，小弟也涨"的协同

### 外部 vs 内部驱动

| 类型 | 说明 |
|------|------|
| 外部驱动 | 政策、行业消息等导火索，反映在涨停分析、热点事件中 |
| 内部驱动 | 板块内部资金自己的动作（如下跌后的回抽），没有外部协同，难成主流 |

## 四、选股操作流程（每日复盘）

```
每日涨停复盘
  → 鱼池（核心驱动板块中所有涨停股）
    → 龙门（已出现或正在构造转折结构的个股）
      → 化龙/自选（最有可能形成买点的3~5只）
        → 建仓
```

### 每日复盘涨停股的具体方法

| 条件 | 内容 |
|------|------|
| 条件① | 涨停（或10%以上）的大阳线，且正在构造转折结构（即使尚未完成转折也要纳入跟踪） |
| 条件② | 检查该涨停股所属的行业板块，是否满足资金一致性（多个涨停）？ |
| 条件③ | 该板块的资金趋势龙头是否也涨停或大涨？ |
| 条件④ | 涨停是否由特大单/大单主导？ |
| 条件⑤ | 是否有外部事件驱动（涨停分析/热点题材）？ |

> 选股不是为明天立即买入，从核心驱动到出现买点，通常需要3~5天（运气好）到数周甚至1~2个月。

## 五、安全边际的重要性

### 作用

- 提供**止损参考**（破了安全边际，结构可能破坏）
- 区分"该做T降低成本"还是"该止损离场"
- 是"进可攻、退可守"的关键位置

### 安全边际的构成

- 中枢上轨 + 170均线/34均线的关联关系（谁托着谁）
- 两者距离越近，安全边际越好（止损清晰）

### 实操原则

- 若结构未破、仅小幅回撤 → 可做T降低成本
- 若安全边际被有效跌破 → 应止损离场，留下100股跟踪，等待更大级别买点接回
